# 🌍 Jarvis 远程访问指南 (Anywhere Access)

如果您希望在**任何地方**（使用 4G/5G 或公司 Wi-Fi）都能访问家里的 Jarvis，而不仅仅是在同一 Wi-Fi 下，我们需要搭建一条安全的“虚拟专线”。

最推荐的方案是使用 **Tailscale**。它免费、安全、且无需复杂的路由器设置。

## 🛠️ 核心原理
Tailscale 会在您的手机和电脑之间建立一个加密的虚拟局域网。无论您身在何处，只要开启它，手机就仿佛还在家里一样。

---

## 🚀 第一步：电脑端安装 (PC)
1.  **下载**: 访问 [Tailscale 官网](https://tailscale.com/download/windows) 下载 Windows 安装包。
2.  **安装**: 运行安装程序。
3.  **登录**: 安装完成后，任务栏右下角会出现一个“网状”图标。点击它，选择 `Log in`。
    *   使用 Google 或 Microsoft 账号登录（建议使用微软账号，国内连接较稳）。
4.  **获取 IP**:
    *   登录成功后，右键点击任务栏的 Tailscale 图标。
    *   查看顶部的 **IP 地址**（通常以 `100.x.x.x` 开头）。
    *   **记下这个 IP**，这就是您电脑的“全球唯一内网 IP”。

## 📱 第二步：手机端安装 (Mobile)
1.  **下载 App**:
    *   **iPhone**: 在 App Store 搜索 `Tailscale` 并下载。
    *   **Android**: 在 Google Play 或 APKPure 搜索 `Tailscale` 下载。
2.  **登录**: 打开 App，使用**和电脑端完全相同**的账号登录。
3.  **连接**: 点击屏幕中间的开关，允许添加 VPN 配置。
    *   当状态显示为 `Active` (绿色) 时，表示“虚拟专线”已打通。

## 🔗 第三步：远程对话
现在，无论您是用 5G 还是在星巴克：

1.  **关键**: 打开手机上的 Tailscale App，点击左上角的开关，确保它变成了 **Active (绿色)** 状态。
2.  打开手机浏览器。
3.  输入地址：
    > **http://100.101.227.116:8501**
    
    *(这是您电脑的专属 Tailscale IP)*

4.  您现在可以像在家一样与 Jarvis 对话了！

---

## ⚠️ 备选方案 (如果不方便装 App)
如果您不想在手机上装 App，或者希望生成一个公开链接发给朋友（**不推荐，有安全风险**），可以使用 **Cpolar** 或 **Ngrok**。

但考虑到 Jarvis 包含您的私人数据，**强烈建议使用 Tailscale**，因为它只有您自己能访问。
