# 0720 来思考思考

* 逃避车祸后的收尾工作
* 老婆意识到我们花钱多挣钱少的现实

先说逃避吧

1. 昨天把车开回来了，也把出院手续办了。剩下的就是让家属一起去签了协商好的协议，再去交警队拿到责任认定书，然后交警把案销了。再去保险公司报账！大概就这么个流程。家属，平安保险，交警队，五菱车4S店，车管所。五个地方需要去跑，其实也是很大的工作量。
2. 今天早上的时候就特别不想去触碰这些事儿，说到底还是怕被拒绝，怕达不到预期，带来的失败感。其实为啥不能去享受那个过程呢，沟通时不好意思提要求，不敢说明情况，怕家属失望 ，怕交警那边不配合，怕保险公司那边会有很多手续。  为什么我心中那么多怕呢，那么多脆弱的东西，把别人的拒绝当做常态，然后找到一种特殊的方式吸引起拒绝方或抵触方的认可度，愿意帮忙，那才是提升的地方。  别怕提要求，你不说出你需要的，真正想帮你的人就不知道怎么帮你。那些拖拉拒绝的人，你不跟紧点让他觉得自己很在意的话，对方会更不重视。因此摆正心态去跟进就是了，比如交警队那边对方确实很拖的话，就慢慢跟进。上次等到了他要去吃饭，我其实应该再等到他吃完饭，再把证件拿了的，无非就是等嘛，等的时候养神思考，或者看看书，不都是很好的吗。
3. 还有，要家属那边配合的时候，我总是觉得很为难，不好意思开口。如果这一步是非走不可的流程，就只有麻烦家属了，如果会引起家属的不开心感的话，我也就只有损失家属对我的满意感了。而且，有个前提是我先把钱付给了家属了，正常情况下是，要等保险公司那边签字画押了，交警队那边销案了才付款的我提前把款付了，所以需要家属出面，我是有理有据的。我却不好意思开口，可能还是我太在意自己的羽毛了，内心一直对方认为“小王这人还是不错的，多负责任的”，为了维持这种我认为对方会认为的形象而不敢提出要求配合，其实语言说到位了，有礼貌，有理有据的话，对方会出面的，也不会增加对我的负面印象，因为我确实没有提无理要求。如果对方真的有对我那种好印象，我提出需要配合的要求应该不会被拒绝的才对，一切都是我想多了。
4. 因此，我要准备这样发信息给梁姐：梁姐你好，有两个地方需要出面一下哦。我这两天咨询到的信息是这样的，保险公司已经同意了6500的赔付标准了，需要我们双方去签字确认。然后确认后，需要拿到交警队，交警队会出具责任认定书，目前我都是按照我的全责来负责任的，所以交警队那边也不会需要再调解 <u>了</u>，我承担全部责任就是。只需要我们双方确认已经达成协议了，就可以销案了 。所以保险公司和交警队那边还要麻烦你出面去一趟，基本上就没什么事了，剩下的就是我去保险公司报账，报账时需要两个东西，现在我还没有，一是医院检查的那个CT片，二是医院病历，不过病历需要本人去医院复印或者我拿上谢哥的身份证去复印。说了这么多有点乱，我总结一下嘛，就是需要你去两个地方，再给我两个手续。                           

             两个地方是保险公司和交警二大队，我们下周一去如何？  两个手续是医院检查的CT,和谢哥的病历。
（复印病历时本人不能去的话就需要我拿谢哥身份证去复印。）  真的有点让你们麻烦了，理解一下哦

* 关于不自信
	* 今天去金利美询问门市的事情，我一进门就问销售员“上次我来过打算买门市的，结果钱不够，这次来想问问租门市的情况。。。。。。。” 老婆在旁边听得遭不住了，直接说，我们想租门市来了解一下。  我瞬间觉得自己真的有点过于啰嗦了，我干嘛非要解释那么多，是想让人更多尊重我吗？为啥非要解释我以前来过呢，为啥不能直奔主题了。
	* 我回忆了一下，刚进门的时候，感觉没人搭理我们，销售员都没怎么招呼我们，我就有点尴尬的那种感觉，因为没人搭理嘛，可能就更希望被尊重。因此这个要强调自己之前是打算买门市的，这样来提升自己的身价。其实，销售员可能也没想那么多吧！就是简单的有点懒散，没及时来接待，我脑中没有直奔我想要的信息，而是去要别人的尊重去了。
	* 接下来，销售员是自己不是负责出租的只负责销售所以不太知情，我就打算回撤了。老婆，接着问，你们就算分工了，也不可能一点都不知道吧，毕竟是同一个公司。那人只好说了些他知道的信息。所以，我这方面还是不执着，面子薄。就记着自己想要的目标，索取信息。我们毕竟得到了些优惠政策的信息，还有大致的市场规划，就是安置内部品牌为主，外租其次。   波哥多思考目标，集中在想要表达的东西上，不去想多的。
