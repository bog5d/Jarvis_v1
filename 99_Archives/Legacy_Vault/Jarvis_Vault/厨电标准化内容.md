# 厨电标准化内容

各美的厨卫代理商  
紧张刺激的卫浴标准化刚刚过去，我们又迎来了更紧张刺激的厨电标准化。不过还好，厨电标准化还是要比卫浴标准化稍微
简单些，文件特别复杂冗长，我就给大家提炼一下： 从11月20日开始，所有专卖店和8米以上专柜，必须出样美的厨房电器13年新款。
必须出样型号：AJ9013、EJ9016、TJ8055、DJ106A、DT23A，美的消毒柜MXV-ZLT-Q1036-SD、美的灶具MQ7635-GL。
厨电出样8米以上专卖店，必须出样嵌入系列。美的嵌入式微波炉MG720DKI-S（常规价1399）、美的嵌入式微波炉TG823QLV-NS（常规价1999）、美的嵌入式烤箱ET0856LV-61SE（常规价1698）
这就是出样要求，也就是说，总共有10款新品，其中有三款是嵌入式产品可以选择性出样（也就是可以选择不上样），另外七款新品烟灶消是必须出样的。
既然有要求，就有奖惩措施.如果新品出样率达到60%则计为合格，不奖励也不惩罚。如果新品出样率低于60%则计为不合格，公司将扣除不合格代理商2013年的厨电返利（注意哦只是厨电返利 ，卫浴的还在）。
如果新品出样率高于百分之70%，则公司将奖励该代理商一台TJ8055透明演示机。
什么叫新品合格率，其实很简单，一共有十款新品，如果你的店内出样了五款新品，则合格率就是50%。 考察方法很简单直接，希望各代理商切勿以身试法啊，快年底了，别把厨电返利整没了。
为了，配合美的厨房电器的标准化，我们会坚持新品上样8折优惠。   检查时间是12月5日至12月10日之间，个代理商需在12月5日之前完成新品上样。
