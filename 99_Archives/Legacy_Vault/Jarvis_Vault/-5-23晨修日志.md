|     |     |     |
| --- | --- | --- |
| **昨日交流之人** | **观点** | **素材** |
| 万维钢 | \- 我们应修炼上游思维，而非被动地为琐事做反应；<br>\- 增加余闲，可能是修炼上游思维最可操作的事情 | \- 故事：下游总漂孩子，两人日救不暇；一人喊出，你来救，我去上游看看谁在扔孩子。<br>\- 底层的人民每天疲于生活的人民，你让他们去看书吗？去学习吗？去思考一下，怎么改变人生？ 这太难了，方法只有一个：增加余闲。<br>\- 什么叫增加余闲：1，个人思考一些更务虚的想法，比如如何让生活更美好？怎样提高效率？  2. 让经济更余闲，别总被房贷、各种欠款压得喘不过气，那永远没有机会思考理财。 3. 企业同样要开一些务虚会，先有随意讨论讨论的时间和空间，说不定就出现了一两个棒点子。 |
| Bog | \- 来上海两年，无收入的两年像是脱产上学2年，只是我学的是军工行业，资源整合，金融资本的实战知识，这些都是nevis全程在带。<br>\- 创业的心态比上班者，为结果负责的意识更多一些，包括从合作大局出发也更多些，而上班者可能迎合的人更多是老板，创业者迎合的更多是合作方。<br>\- 越多的人希望你成事儿，那你就更易成事儿，那为什么其他人希望你成功呢？   要么你的成功对他有利，要么你帮助过他人成功。 <br>\- 透支他人对你的预期，容易变成焦虑，一旦交付压力来临，会缺乏耐心去耕耘或布局。<br>\- 中国地产时代已过，下一个时代是资产证券的时代。<br><br>\- 不想碰BP这件事，原因是就算做出来了，也没觉得自己多牛逼。  何况现在做得很痛苦。  应对之策： 认识到这件事其实牛逼的，换别人也不能轻易做出来 |     |
| Bog | \- 表格方式整理学习心得，会有一种技术活的感觉，把不稳定不可控的创意工作，变成稳定可控的指标工作。 <br>\- 切换感性思维到理性思维，最大的好处可以实现稳定批量化的生产作品 |     |
