# 请描述一次你打破常规并付诸实践的经历 *

请描述一次你打破常规并付诸实践的经历 \*

推动部门变革，把市场部门从被动的甲方招商模式变成主动的系统性招商模式，实现连续4年完成率第一的全国标杆市场。

13年时，我是部门最年轻的渠道主管，负责市场开发。
彼时，渠道主管开发新客户方式为：新客户找上门，主管筛选有实力的客户。 （这些客户会打电话到总部，说想加盟，总部把信息派到对应分部）。
我因为资历最浅，拿到了基础最差的市场，于是下沉到市场，挨家陌生拜访，半年开发30家专卖店，110家专柜店。 完成了计划任务量的7倍，总结了一套地毯式开发模式。
分部决定推广我的市场开发模式，其他市场主管不太认同，认为这是特殊案例，会有量无质，遂抵制。
市场总监于是升我为渠道副总监，让我牵头新建扁平化的市场团队。
我采用了共建业务模式，并让全渠道大换血，4年内区域市场规模从8千万到破10亿，连续4年成为全国标杆市场。

请用一两句话说说，除了创建这家公司之外，你做过最了不起的事是什么？
大学期间做校园创业项目，从只知道拉朋友合作，到学会组建团队协作，并以此实现在校期间的财务自由。

————————

大学时和死党一起到寝室推销游泳裤赚了一笔钱，后来就屯了大批货，准备和死党再来一波大的。
结果死党开始沉迷游戏，不愿意去推销了。 我自己一个人又搞不定这么一大批货。
无奈之下，我决定发布招聘兼职的信息，结果前来应聘的人数超过100人，然后选用10个同学，并成立了“皮皮校园服务队”。 结果那次依靠团队，在我自己一次没去推销的情况下，将货卖空了。
后来每年招聘一次，依靠这只服务队接了不少业务。 实现了校园级别的财务自由。
