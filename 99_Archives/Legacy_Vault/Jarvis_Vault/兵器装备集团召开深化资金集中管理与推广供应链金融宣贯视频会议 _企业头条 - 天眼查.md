# 兵器装备集团召开深化资金集中管理与推广供应链金融宣贯视频会议 _企业头条 - 天眼查

兵器装备集团召开深化资金集中管理与推广供应链金融宣贯视频会议
来源：[搜狐新闻](http://www.sohu.com/a/394994255_822642)发布时间：2020年05月13日
[金融](https://news.tianyancha.com/tag/52t4352.html)[供应链金融](https://news.tianyancha.com/tag/15808tb0b8.html)[供应链](https://news.tianyancha.com/tag/1638tb343.html)[供应商](https://news.tianyancha.com/tag/28220tce76.html)

原标题：[兵器装备集团](https://www.tianyancha.com/company/948125969)召开深化资金集中管理与推广供应链金融宣贯视频会议

4月20日，[中国兵器装备集团有限公司](https://www.tianyancha.com/company/948125969)召开2020年深化资金集中管理与推广供应链金融宣贯视频会，集团公司总会计师、党组成员张登洲对前期以财务公司+保理公司“双轮驱动”运营供应链金融业务试点取得的成效给予了肯定，并明确了集团公司将在“保理融资+票据贴现+中兵保兑单”三位一体金融产品谱系基础上，大力推广供应链金融业务发展，不断促进供应链管理提升，持续推动产融结合向更深发展。

会上，集团公司要求各成员单位要高度重视开展供应链金融业务的重要意义，要通过加强组织领导、主动宣传推荐、引导供应商注册供应链金融综合服务平台、提高使用财务公司商票和保理公司中兵保兑单支付结算比例等方式多措并举开展供应链金融推广工作，同时，要确保保理、商票、中兵保兑单的到期兑付率100%，维护好成员单位和集团公司的金融市场信誉。

208厂率先实现供应链金融服务平台全级次注册率100%

![044bc7ffcea64bb195b87ceea3d60f5f.jpeg](./_resources/兵器装备集团召开深化资金集中管理与推广供应链金融宣贯视频会议__企业头条_-_天眼查.resources/044bc7ffcea64bb195b87ceea3d60f5f.jpeg)

[成都光明光电股份有限公司](https://www.tianyancha.com/company/24259498)（208厂）为深入贯彻落实集团公司重要部署，有效推进供应链金融业务开展，专门成立供应链金融推进领导小组，由党委书记、董事长[李小春](https://www.tianyancha.com/human/1983084078-c24259498)和总经理[刘晓东](https://www.tianyancha.com/human/1812562865-c24259498)集团共同担任组长。领导小组下设供应链金融推进领导小组办公室，由总会计师陈卫星担任组长，物资供应部部长罗伦、财务部副部长谢俊杰等任副组长。为保证与供应商的对接及时性，提高供应链金融服务响应效率，推进小组办公室就设在208厂的物资供应部，由物资供应部、财务部、光玻销售部、功能玻璃部等部门及各分子公司协同联动，为供应商提供供应链金融的沟通解释和服务对接工作，进而起到“有统一指挥、有过程管理、有效果跟踪”的统筹推进效果。

经过有序组织和深入宣传，保理公司全方位对接保障服务效率，208厂及分子公司已率先实现全级次11家单位上线“[中国兵装](https://www.tianyancha.com/company/6520818)供应链金融服务平台”，实现供应链金融平台注册率100%。同时，208厂已将中兵保兑单等供应链金融产品向其一级供应商进行了全面的宣传推广，结合中兵保兑单“免上门、免开户、灵活拆分、快捷融资”等诸多优势向供应商做了深入介绍，部分供应商已形成开通意向拟上线供应链金融平台。

后续，208厂将借助中兵保兑单等供应链金融产品的多元优势，不断发挥其对于各级供应商的融资服务实效，将其作为缓解供应商融资压力、增加供应商资金流动性和降低供应商融资成本的有效抓手，持续推进产业链融合、互助与协同发展。返回搜狐，查看更多
