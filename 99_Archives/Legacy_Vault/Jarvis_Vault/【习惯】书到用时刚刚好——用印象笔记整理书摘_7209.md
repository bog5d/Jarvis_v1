# 【习惯】书到用时刚刚好——用印象笔记整理书摘

【习惯】书到用时刚刚好——用印象笔记整理书摘( http://www.douban.com/group/topic/49350358/ ) 分享自@豆瓣小组 Android 客户端
<图片7>
“书到用时方恨少”，有时候可以指读书太少，提笔无言，但是有没有想过，如果我们只是单纯的读书，而不对阅读后的内容进行整理，我们也许就对不起“书籍是人类进步的阶梯”这句话了，剩下的也许仅仅是阅读当时的快乐，而这种快乐，随着时间的流逝，也就被忘记了。
如果你希望读过书后，在用的时候能够被用到，让读过的书变成自己的无形资产，那还真是需要下一番功夫，今天就来说说我关于用印象笔记做做书摘的体会。
★关于纸质书和电子书
这个问题被无数人提及，我们在这里可以忽略不计，对我而言，只要是书，管他是纸质的，电子的，扫描的，甚至是拍照的，只要对自己有用，都可以拿来读，一般我读书的三个类型:
纸质书：纸质书拿在手里最实在，最有质感。读一本书，我比较喜欢写写画画，标出我觉得有意义，有价值，有用或者印象深刻的句子和段落，有时候还会写下自己的体会，随着看，随着画，随着写。我不是那种对书特别在意的人，一般书看完了，上面有很多读过的痕迹。如果对某些书籍实在爱不释手，我也会再买几本，留着或送朋友。
kindle读电子书：手里有一个kindle会随时带，遇到喜欢的新书会在亚马逊书店上买电子书，价格比纸质书便宜很多，小说一类的一般会在网上下载，随时都可以看，不会很伤眼睛，而且kindle随身携带非常方便，不得不说，买了kindle以后，我的阅读量大了很多。
IPad读电子书：有一些书我也会用IPad来读，尤其是一些PDF的书，一般epub我读的倒是不多，因为有了kindle，更多读mobi的，网上可以找到的资源差不多，PDF可能会找到一些有意思的资源，例如海外杂志的扫描版等等。用IPAD的阅读体验非常好，只是读书的时间不宜过长，否则眼睛会比较累了。
★电子书的保存
一般我会把电子书保存在百度云里，我的百度云里有两个文件夹，一个是”待读书籍“，另一个是”电子书仓库“，现在百度云的电子书资源很多，从网上找到后，可以直接转存到自己的网盘里，非常方便，尤其是epub和PDF格式的，可以在想看的时候直接下载到IPad上，看不过来的也不用占本地的硬盘资源，存储就好了。如果是mobi格式的，想看的时候再下载到本地。
另外，我和三人行读书会的朋友会分享自己的电子书，每次读书会结束，我们会把提到的一些电子书放到网盘上，想看的时候直接下载到IPad或者kindle上，有好资源大家共享，十分方便。纸质书看过的，我也会带到读书会，谁需要就送给谁，让书流通起来，彼此交流。
当然，大家还是要多多支持正版图书，一般新书我会毫不犹豫购买的，一些时间比较久远或者绝版的图书，电子版还是很不错的选择。至少在获取知识方面，这并不矛盾。
★用印象笔记做书摘
所谓书摘，就是书中精华的部分，一本书有时候甚至不需要完全读完，精华的东西，对你有价值的内容，把他存储下来，以后可以随时翻看，这也就是书摘的作用了。
做书摘的方法，其实很多。纸质书和电子书，略有不同。
如果是看纸质书，可以把书中的精华，一般都是我做笔记的地方，我会用手机拍照下来，然后存放到印象笔记中，一本书中，最精华的部分保存下来，然后放在印象笔记里永久保存，我想即使书送人了也没有关系了，因为重要的东西你都存下来了。
当然你可以用裁纸机和扫描仪把书变成电子版，也可以用扫描全能王这样的软件进行OCR文字识别，但是我想直接用手机拍照也许是最简单的方法了吧，没有任何技术含量。拍完照，用百度云自动进行备份，然后用电脑处理一下存放在印象笔记就可以了。虽然对于印象笔记高级账户来说，上传容量足够大，但是让图片变小一些，也会增加同步速度，当然这一步也可以忽略。
★为每一条书摘建立一个笔记
在存储的时候，我建议一个图片(一般是书的一页)建立一个笔记，而不是把很多页面放入到一个笔记里，这样的好处是方便使用和查询，最好每个笔记能起一个标题，方便辨识。做书摘放入印象笔记的目的就是能够在用的时候把书中的某一个部分快速调取出来，如果我们找到的是一本书，而不是一本书的某一个部分的话，那做书摘的意义也就没了。
电子书的书摘其实和纸质书一样，如果看kindle，我会随时标注有用的部分，书看完后，将kindle连上电脑，把标注的部分拷贝到印象笔记里，并且做相关的注解。kindle的书摘很容易整理，也比较容易编辑。
如果是PDF文档，我比较喜欢用IPAD里面的 Neu.Annotate 这个软件，其实看PDF的软件很多，包括百度云的ipad版下载后就可以直接读取，只是这个软件中有画笔功能，可以直接在PDF上进行标注，标注后的电子版，直接用截屏功能把屏幕保存下来，就算做了书摘了，这个方法我一直觉得很好，不用再刻意的整理书摘了，截图会上传至百度云，然后按照纸质书的方法，放入印象笔记，这样就一劳永逸了。

★书摘的加工
对于读过的书的书摘来说，未来可能有两种用途
1、想再看看的时候回顾一下
2、在其他领域（例如文案、写书、培训、写报告或者其他）需要的时候，能被检索出来。
除了主动去回顾外，无论你是否还记得曾经阅读过这个部分，在用到的时候，书中提到的内容会自动被搜索出来，这无疑是一件非常爽的事情。其实只要将书摘进行加工，并且养成良好的整理习惯，你的印象笔记就好像是你的资料库一样，随时取用，思维永不枯竭。
其实也很简单，只需要关注印象笔记的分类和标签就可以啦
我的印象笔记里，有一个读书电影旅行的笔记本，存放和读书电影和旅行有关的笔记。
关于读书的分类只有三个：
<图片2>
我的书库：用来存放所有的书摘，只要是我自己做的书摘，都会放在这里
推荐书单：你会在某些博客或者微博微信上看过很多人的推荐书单或者图书吧，把这些信息放在这里，就不怕没有书可以读了
读书笔记：我自己写的读书笔记和别人写的读书笔记都放在这个文件夹里，读书笔记和书摘不同，书摘是摘录，原封不动的摘录图书的信息，而读书笔记的形式确实五花八门，写作的角度也不同。网络上看到的好的读书笔记，我都会保存下来。
你可能会问，那么多书，那么多书摘，都混在一起，岂不是会混乱，当然了，别着急，这不是还有标签么。
标签相对与分类最大的好处是可以为一个笔记建立多个维度。因为一个笔记只能属于一个分类，但是我们可以为一个笔记建立多个标签，这样一条笔记的作用就无限被放大了。
★建立一个以书名命名的标签
例如 z整理的艺术 q清单革命 ，什么样的笔记可以打上这个标签呢
自己做的书摘当然可以打上这个标签，这样即使所有的书摘都放在”我的书库“这个笔记本里，也不会混乱。
当然，这个标签的作用绝对不仅仅如此，当你看到任何一个有关这本书的资讯，你都可以保存下来，可以是某人的分享，或者读书笔记，甚至是某篇晨间日记里带有这本书的内容，都可以打上这个标签。
这样，当你查看这个标签的时候，得到的绝对不仅仅是书摘，还可以得到任何与这本书有关的内容
