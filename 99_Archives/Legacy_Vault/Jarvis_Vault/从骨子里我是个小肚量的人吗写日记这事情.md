# 1003 从骨子里我是个小肚量的人吗写日记这事情

写日记这事情我还是怕没有有瘾，容易中断，那种一生都在写日记的人，其实是自己在享受日记的过程，我现在好像更多还是在处于一种责任在写，所以不够有境界呀！
不过我也能发现记录日记的好

* 比如说

1. 如果自己心中真的有挣扎的思想斗争，那样会停不下来的，在写的时候，而且往往我都愿意很直白的把内心书写出来，比如前天记录对王斌的厌恶时，我自己又知道也许是自己太好面子，可能是在刻意美化自己的在思考一些问题。 所以我的日记里面上一段文字还在记录对他的厌恶，下一段已经是自己在怀疑自己的情绪是出于自私的欲望，比如掌控别人的欲望。。。。，总之就是日记能促进自己更客观一些的评估当下。
2. 日记里面有思考过的事情，在第二天表述给别人的时候，一般情况下，效果会好上至少一倍的。比如上次描述对王海梦的恶劣态度时，我很淡定地说出原因，是把她感动到了的，不过这些真的是因为有提前思考权衡的过程。 日记促进思考，会给出更好的沟通方式和促成更好的沟通效果。
3. 思考的问题真的很容易增多呀。我之前写过一篇文章，我说人之所以成长，在于思考。不过记得是写在纸上的，已经找不到了，。。。。（我找找看印象）没找到。  白天把要思考的东西，稍微放脑子里面一下，到了晚上，只需要提出思考的问题，让思路自己自由驰骋就行的。大脑就是这样，抛出问题，它就开始运作起来的，只要不受打断，基本上能得出比之前更多信息的一个结果，比如通过哪个渠道解决呀，或者值不值得呀，会有什么影响呀，都会至少有一个结果的。  而且晚上有思考习惯的人，几乎在任何一个领域都不会差到哪儿去的。 如果再加上冥想打坐，和瑜伽之类的话，就是一个不庸俗的人了，在很多有层次的人眼里这种人都是清冷不俗之人的。有档次！可以设置为目标哟，培养成这些习惯，作为人生挑战目标。哇哈，人生有事做了。
4. 白天思考一下，今晚日记消化什么问题，应该很能促进日记的质量。波哥，加油哦。
