# 2017年1月3日

* 今天上午办公室开会，只有王海梦。 我感觉到让下属说的重要性，自己能闭嘴，就尽量闭嘴。 然后，梦姐喜欢表达，就让她表达，我也可以多听听，说不定能得到更多信息。 而且更不容易，凭感觉定出错误决策。
* 下午，去门市上开会，确定了几件事。 
	1. 门市上的货，有多少发多少。 别在用所谓的“等到货到了一起发”，当作借口，然后延迟发货到乡镇。 因为这样会引起无限拖延。
	2. 钱账分离，刘妈记账，但绝不摸钱。王斌所保管现金，受刘妈记账的管控。 
	3. 预付款，现款，欠款。不同客户，不同待遇。 打预付款的客户，享受0.5个点预付款奖励。 现款交易的客户，享受0.1个点。   同时预付款出货，享受1.5个点的提成，现款出货1个点，欠款客户0.5个点。  此制度现在先内部试运行， 先用在计算工资上面。到3月份后，制度成熟了，再执行到客户预付款奖励上。（此制度我希望能执行下去，用来降低现在居高不下的欠账） 
	4. 对了，我需要做17年的工资表模板了，业务薪资按照不同出货类型来核算。 刘妈和冷瑞的工资需要，加入每月门市净利润的4%。
	5. 要货计划，最好能公开透明，换了一个人也不会断线。

* 明显感觉到王斌态度好了许多，我猜测是我挑明说了，
	* 要核查现金，
	* 还有要货计划能够共享信息，不能一人独揽信息。这样不会出现，王斌一走，工作即瘫痪的情况。 
* 这样他就没那么多的其它猜想了。
