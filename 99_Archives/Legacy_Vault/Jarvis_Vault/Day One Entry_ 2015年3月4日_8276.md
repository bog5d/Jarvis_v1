# Day One Entry: 2015年3月4日

* 这次的迷茫\_
* 准确的说，是对个人行事风格的迷茫。我觉得老马说我们的思路都没用心，对工作规划都太浅，我真的蛮难受的，因为我其实是用心思考了的，但我真的思考不出有新意的什么点子或规划出来了，我感觉自己真的没有什么方向性的思考得，好无力呀，真的是电影的江郎才尽，再也做不出大型亮点举动来了，真的好惭愧，好无力呀。 我很想有抱负性的大动作，能让我实现些人生意义或者成就感，但真的想不出来。
* 目前，唯一能做的就是，把基础工作做得更匝实。 更务实。
* 哪些基础工作呢？
* 政策传达，电话或者信息发放到位。二，数据核算给他们弄好指引，方便他们计算盈利。
* 三，帮他们开盘开会，做好谅解工作，这里就要倒逼我，真正的去学习产品知识，到底哪些型号值得推荐？ 到底被推荐型号好在哪里？
* 额外的就是，我自己对常规价已经跟不熟悉了，现在我要熟悉常规价了，保持对价格的敏锐。
* 这些事情，是我喜欢的吗？ 嗯，做ppt和上台讲东西是我喜欢的，自信源于充分的准备，肯定是这么回事，只要准备够充分，在台上表达清楚，我自己就已经很爽了，所以做ppt讲解政策，是我乐意干的事情，好生发扬。
* 核算数据呢，我也愿意，因为可以顺带把我大竹门市也给核算了的，很好。
* 传递信息呢，同样是我愿意干的，我觉得每次传递了政策信息，成就感满满的呢。
* 好吧，就干这些。对于大竹，我迷茫吗？比起去年还是要清晰些，还债，建立速达财务系统，售后录单➕增收计划，转门市，等等。我去大竹，到底应该干啥子，我觉得是整理数据和尽量和王斌冷瑞愉快相处，说白了尽可能带点快乐给到他们，而不是老是一副下去挑毛病的心态，让自己也只想逃离，也不要把自己的作用想得太大了，我能做的就是去发现问题，然后提出解决或者督促解决。 抱着这个心态下去，不会那么痛苦了
