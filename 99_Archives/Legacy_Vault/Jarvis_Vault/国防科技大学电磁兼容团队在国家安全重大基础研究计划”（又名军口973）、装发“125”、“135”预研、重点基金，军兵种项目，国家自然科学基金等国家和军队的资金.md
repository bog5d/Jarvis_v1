# 国防科技大学电磁兼容团队在国家安全重大基础研究计划”（又名军口973）、装发“125”、“135”预研、重点基金，军兵种项目，国家自然科学基金等国家和军队的资金

       国防科技大学电磁兼容团队在国家安全重大基础研究计划”（又名军口973）、装发“125”、“135”预研、重点基金，军兵种项目，国家自然科学基金等国家和军队的资金支持下，通过十二五和十三五10年的潜心研制，在全世界独创性提出”基于能量选择机理”的强电磁防护方法，是国内唯一可以有效解决信息系统前门电磁安全防护问题的方案。

       国防科技大学在电磁安全行业的该研究成果，经西安21研究所、绵阳核9院、石家庄强电磁环境模拟国防科技重点实验室、武汉电磁兼容性国防科技重点实验室以及国防科技大学等单位进行了强电磁实验，结果表明所研制的强电磁防护装置可有效防护不低于50 kV/m的强场攻击，防护门限500-5000V/m可调，涵盖超短波、短波、Ｐ、L、S、C、X波段，响应时间小于10 ns ，并且S波段以下插入损耗小于0.5dB。
