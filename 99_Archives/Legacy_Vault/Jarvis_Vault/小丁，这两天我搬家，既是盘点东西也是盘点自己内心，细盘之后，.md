# 小丁，这两天我搬家，既是盘点东西也是盘点自己内心，细盘之后，

小丁，这两天我搬家，既是盘点东西也是盘点自己内心，细盘之后，我想给我们的关系按个暂停。
原因主要是两个：
一、前任的影响并没根除。
去年7-15日，我为了来上海创业，选择离开了尚在热恋期的重庆女友。 我希望她来上海，她最后选择留重庆，于是分手了。 当时，我说过等她一年到21年7月15号，如果改变主意了想来上海，我随时ok。
结果，4月份你出现了，我本想放下这个桎梏，开始新人生。
但内心还是有个声音，觉得自己背弃承诺了。
我还是想等到今年7月15日后，再心无旁骛地开始新恋情。

二、当下我的状态可能不适合恋爱。
其实，我这段时间一直都很down，只有跟你一起的时候，我会很开心。 比如约会的时候，和跟你聊天的时候，除外的时间还是很down的。
我跟你说过，你就像止痛药，有你的时候就不痛。 不过，我生活的难处依然都在，离开了止痛药，我还得回去面对那些生活和事业的难题。
我现在创业期，无收入，是靠老本和家里一些支持在生活。 我给自己定的每天消费不超过100，不然我的库存可能会提前耗完，支持不到成功。
加上，我们还没融到资，现在几个合伙人需要自筹资金90万，摊到我身上是二十几万，也是蛮头大的。 这让我更没太多心思去照顾到你的恋爱体验。 一个每天焦虑的男朋友，应该不会让你有太好的体验。

犹豫了好久要不要说出来，不过这些就是我最真实的情况。 感觉自己像个渣男，给了你期望又给摁住。

如果你愿意的话，我想过段时间，等我感情心态调正和困难缓一缓后我们再深入尝试。
