# 中国式私募股权投资

![Evernote Snapshot 20220127 172038.jpg](./_resources/中国式私募股权投资.resources/Evernote Snapshot 20220127 172038.jpg)
中国有说完你的居民储蓄，但是这些储蓄资金是厌恶风险的，另外在股市房市里面也有大量的资金，但这部分资金呢属于套利的资本，也就是说中国不缺资金，但是缺长期资本，这是中国的特色问题。

![Evernote Snapshot 20220127 173207.jpg](./_resources/中国式私募股权投资.resources/Evernote Snapshot 20220127 173207.jpg)![Evernote Snapshot 20220127 173208.jpg](./_resources/中国式私募股权投资.resources/Evernote Snapshot 20220127 173208.jpg)
私募股权市场是一个缺乏效率的无非有效市场，那么这个事就是因为他有各种不确定性，比如投资效率有不确定性，信息也有不对称性资产的流动性也相对较差，那这个事儿他看起来是他的特色，不过他也有一个好处，就是这里面有极大的离散性，他的说法叫私募股权基金收益的离散性特别高，就意味着说做的好的收益极高，而这里面情善和智慧能够给私募股权的投资者带来丰厚的回报，这几个好处就是说什么呢，你在这个行业里面你足够勤奋和智慧，他是能够换来高额回报的，但是比如说像在在证券市场上那么很多投资经理，他的收益其实是跟市场水平的收益密切相关的，他的个人情分和个人社会很难在平均的行业从业者里面拉开差距。
