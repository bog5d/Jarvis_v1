# 造人起源

神为了让生灵在这个世界上流放就创造了人，但人总不能像神一样永生吧，所以神创造的人是会枯竭的身体，如果要保证身体不枯竭必须不断的探索生存所需之物，比如食物，水。这些都是现在人存在的主要活动内容，然而也有智慧之人，寻求了足够保证不枯竭之物后，开始思索自己存在于世的价值。
我想宗教就是这么出现的吧！

我想怎样创造人呢，也许就是把不听话的神派到人这个世界，做短暂的停留抑或是惩罚，更或者是历练。如果在人世表现得好，待到身体枯竭以后，再重新进入循环之道，而人世这个灵魂锻炼场所，像是越来越庞大了。
人生这么短，肯定有些永存的东西。我多么希望是灵魂或者是其他的我无法命名的神圣事物。

但神给人这个世界却布下了一些不变的规律，比如如何去过去成功，怎样会自取灭亡，也就是宗教所说的善恶，如果人世这个场所不能充分把善恶因果规律发挥出来，待到人枯竭之后终会由神来进行矫正，比如恶人在世未收到恶报，这是神所设定规律的失效，那待到恶人枯竭后其灵魂再度受罚！
为了让不同的灵魂得到不同的历练，所以会安排不同的人生来这个世界上。
所以总结一句话，神为了历练那些不够合格的灵魂，安排了这个世界，既然来了就好好锻炼灵魂吧。
