# 如何在日常生活中培养阅读习惯：象亲@苏豆芽 现场分享实录

![unknown_filename.2.jpeg](./_resources/如何在日常生活中培养阅读习惯：象亲@苏豆芽_现场分享实录.resources/unknown_filename.2.jpeg)
有一支只有三分钟的TED视频，主人公[马特·卡茨（Matt Cutts)](http://www.ted.com/talks/matt_cutts_try_something_new_for_30_days)来自印象笔记的故乡硅谷，他建议大家用30天尝试一件新鲜事，或者养成一个好习惯，并全程记录真实的自己，感受每天的改变，我们把他叫做“马特焕新”。

上周日，印象笔记与近百位象亲，相聚北京独立书店单向街，一起分享自己的马特焕新故事，思考未来30天可以做些什么。

现场，我们还请到了一位资深象亲[@苏豆芽](http://www.douban.com/people/sudouya/) ，她从2012年开始坚持每周阅读两本书，热爱用印象笔记随时多地点写读书笔记并分享。豆芽独立运维了一个拥有8万人粉丝的豆瓣小组——“学习型友邻大集合”，曾发起过“每周一本书，有空一起读”的号召。她说“如果没有读一本新的书，就只能给这周的生活质量打两星。”很多朋友告诉豆芽：“很想读书，却读不完，读不进，每天太忙没有时间……” 

那么，如何在日常生活中培养阅读习惯，让阅读成为一种生活方式，下面就和大家分享@苏豆芽 的现场实录。
![unknown_filename.4.jpeg](./_resources/如何在日常生活中培养阅读习惯：象亲@苏豆芽_现场分享实录.resources/unknown_filename.4.jpeg)

完整版演讲稿，现场我没说这么多，因为我忘了带小抄o(╯□╰)o 

首先，非常的感谢印象笔记邀请我来参加这次的分享活动。先自我介绍一下吧，我是苏豆芽，你可能在豆瓣看到过我写的书评，其实我觉得那不算书评，更像是对书内容的总结和分享。这也是我一直在做的事情，读一本书，会把收获和感悟记录下来，最初发表在豆瓣上，后来是在微信公众账号。

刚刚听这边的朋友们介绍自己，都是已经在工作的人，我想我们的共同点很多，就是虽然想读书，但是杂事越来越多，时间越来越少。读书变成了一件需要自我约束的事情。

2012年夏天，我下定决心重拾阅读，并希望阅读能够成为自己生活的一个固定部分。这使我拥有了很多平静又美好的时间。

希望我的分享，能够带你重拾阅读，养成一个好的习惯，让读书成为生活的一部分吧。

# **读书是找时间做回自己**

我也是上班族，工作压力就不说了，大家都有所体会。

我是个已婚人士，我老公是个文艺处女座,此处省略1000字。（老公是处女座就是好啊，一语胜千言）

我是个独生女，我爸妈比较粘我。

人越长大，身上的角色就越多。以后我还要成为一个“儿子想要吃什么就马上变出什么的小叮当一样的妈”

夸张地说，阅读让我克服了恐婚，也就是在2012年夏天的时候。就此要求自己养成这个习惯：

每天给自己一小时的时间阅读，这一小时，抛开所有的角色，就只是做自己。

# **书就像捕梦网一样**

现在的人们工作压力都很大，我自己抗压性就比较差，白天工作上的琐事和情绪，很容易带回家里，甚至带到睡眠里。

每天晚上睡前读一段文字，好的文字带来的想象，就像捕梦网，隔断了现实的纷扰，带来好的睡眠。也算是锻炼自己抗压性的一个办法。不一定是晚上，当我觉得自己在一个现实问题里钻牛角尖的时候，我愿意看一本书换换脑子，发现别处的世界。

# **读书会让时间变慢**

也许是因为年纪大了吧，我已经步入了28岁，感觉生活节奏变快，琐事的增加，似乎时间过得飞快，会感到恐慌。

如果周末按照平常一样早起，看一本书，我阅读的速度还算快，一本200多页的书，大概4个小时看完。看书的时候，就觉得时间过得很慢了，书里发生了那么的事情，现实里却还不到12点，就好像还有一天的时间，好像还有很多事可以做。

## 那么，具体我是如何在日常生活中来培养自己的阅读习惯呢？

## **一、碎片时间：平淡生活中，我们更需要一点想象力**

经常听人提碎片化阅读，当然因为人们现在的时间是碎片化的。我觉得，这不是一个好事儿。专注是一个非常宝贵的习惯。但有时候，这又是没有办法的事情。如果你的时间比较零碎，我有很多好故事推荐，比如：一些文学作家的短篇小说集，每个故事大概需要十五分钟就可以看完。这些故事对于我就是换脑子的作用。
![unknown_filename.jpeg](./_resources/如何在日常生活中培养阅读习惯：象亲@苏豆芽_现场分享实录.resources/unknown_filename.jpeg)
**推荐：**

* 国内作者的：中国文学短经典系列、共和国文库
* 国外作者的：外国文学短经典 系列 特别一提的是 布考斯基的《苦水音乐》

## **二、假期的意义：时间上的奢侈**

假期里的沉浸式阅读。当我有了一整块儿的时间，总是让我亢奋不已。读过上面那些短篇的故事之后，总能根据自己的口味发现喜爱的作者。想要读他们的长篇作品，想要读完他们的所有作品。假期里读这样平时没有办法完全沉浸其中的书，是假期的意义：时间上的奢侈！

## **三、我今天读完了一本书哦！**

除了短篇故事，我还有一些适合培养阅读习惯的书来推荐，他们的共同特点是：不超过200页。

单纯的追求阅读数量，是件犯不着费嘴皮子去批评的一件事儿。如果你觉得“读完一本”才有成就感，才能继续读下去，就从这些比较薄的书入手。成就感越强烈，坚持越久，慢慢的，长的也就没问题了。

**推荐：**

* 那些字数少，又好看的书：《晚安玫瑰》《一个人的好天气》等等待列出。

## **四、那些看了就想做点什么的好书**

有哪些书可以为生活增加灵感呢？

生活日复一日，总有疲倦的时候。觉得生活很无趣的时候就去看这些好玩的书：然后按照书里的方法开始玩。比如说手账，我一开始也是从书上看到的。《断舍离》，看完就开始整理房间，折腾东西。《吃早餐，彻底改变了我》看完第一反应不是做早餐，是买托盘和餐具，看着就太美好！但后来也真的开始实践早餐了。

我们的这次主题就是，30天的时间尝试一件什么事儿？如果你不知道自己做些什么好，又觉得一些事情开始较难，就去书里找。类似的图书还有：《简单就好，生活可以很德国》、《晨间日记的奇迹》、《一个人去跑步》等等。

# **五、一年读多少本书？什么时间读书？**

* 早上读，好文字，好笔法，好故事的，自己脑袋清醒精力充沛的时候读最好。适合于那种自己非常偏爱的作者的，又是第一次读的文字。
	
* 晚上累了，读自己喜欢的人的书。每次觉得无书可读，就拿起我最爱的迟子建的作品，都不会失望。
	
* 周末读，两天里拿一天出来看书，一天做家务。这可能会是很形式化的，周末一定要拿出来一天读书。即便是形式化，时间久了也会产生意义或者价值。

我的速度是一周2本，一本适合平常碎片化读的，一本适合周末大把时间读的。这样一年就是100本左右。我觉得这样足够了，经常看到豆瓣上一些牛人晒自己的读书数量，有的甚至一天一本。对于我可能是很难的一件事情。

# **怎么发现好书的？**

张佳玮在上海书展的分享会上说过一个方法：找你喜欢的作者的随笔，去看他师从谁，钦佩谁，你就去看这些人的文字。一抓一个准儿。类似的办法还有，你喜欢的一本书里提到了什么书，你就都找来看。我的习惯是，听说了一本书就要拿来看一看。不好读再扔下。不那么好就大概浏览完，好的话，藏着慢慢读。但这就像人与人之间的缘分，人与书之间也是讲究缘分的，遇见了珍惜，错过了也不要强求。顺其自然。

再一个，有个爱读书的朋友很好。读书这件事，在喜欢读书的人眼里，不值得一提，读就是了，哪儿那么多废话。在不读书的人眼里，读书的人就是文艺青年了，好不个性！

# **读书怎样记笔记？**

大家都说要养成独立思考的能力，怎么养成呢？我的办法就是记笔记。

对于文学类的图书，我很少点评，总觉得自己还缺少些什么。对于生活的，自我管理的，我都要记笔记。先是读的时候，就在引起思考的地方贴上便利贴，写上自己的想法和思考。回过头来，再看一遍这些地方，把自己的整体想法，有自己的判断和决定的，写成一篇分享文字。

**用印象笔记的好处在于：我随时都可以拿起手机来就写，已经养成了这个习惯。更方便的是，直接用手机把喜欢的段落拍照存下来。自然就在印象笔记里形成了一个读书笔记。**因为有过这样的经历：有那么一段特别有感触的文字，翻好多遍书都没找到，后来，书也找不到了！

# **读过的书怎么处理：**

我一般会保存下来，因为它已经不仅仅是一本书，而是凝聚了我度过的美好时间的书籍。

这两年里面的，充满回忆的书：（2012~2014）

150进20，凝聚了美好回忆的书：

① 《战胜拖延症》大半夜看完的，建议：不要晚上看，看完太激动了，容易失眠。幻想着各种的改变而不能自已。

② 《都市速写薄》偷得浮生半日闲，坐在家里的沙发上看的，每一篇文字，每一张图细细品味，不舍得看完。③ 《断舍离》看的很飞速，因为着急看完之后去扔东西。目前仍在在每周扔东西，扔的时候都想着这本书呀！④ 《我的100件时尚单品》女人味的时间，在逐渐的堆起自己的100件凝注了岁月和故事的风格单品。

⑤ 《深夜食堂》系列，1~8一起购买，连着好几个晚上看完，因为看得太快而看到这些书的时候就懊悔。好书像好菜，慢点吃。

⑥ 《简单就好，生活可以很德国》如遇知己，非常符合自己的性格的书。每看一次，燃起一次改善生活的冲动！

⑦ 《我们家》（段逸兴一家）假期里的沉浸阅读，好故事，好文字，好时光。

⑧ 《我的男友》青山七惠的书都看了，不是说这书最好，只是喜欢她的心情都凝聚于此。

⑨ 《幸福的建筑》阿兰德波顿陪我度过很多美好的时间，敏感于周遭的环境，不时想到这本书的观点。

⑩ 《一匹马两个人》如果不知道读什么，看自己最喜欢的迟子建就行了。

⑪　《孤独的池塘》年纪大了之后才理解萨冈。

⑫　《到黑夜想你没办法》周日一整白天什么都没做就看它了。太震撼。

⑬　《目送》这本书读得很控制自己（不忍太快），印象深刻。

⑭　《工具》用来调整自己的情绪，时常提醒自己这个问题。

⑮　《自控力》“自控力是有限的，我要策略的使用它。”这个观点渗透了我的整个生活方式。

⑯　《天下卵》喜欢有才的男人，尤其喜欢他们得瑟自己才华的时候。

⑰　《万物有灵且美》对乡村生活的“叶公好龙式”爱意时常被它唤起。

⑱　《没有人给他写信的上校》膜拜马尔克斯我就读这一本。

⑲　《苦水音乐》每一个小说的开头都似曾相识，大师一直被模仿，我却才读到。

⑳　《命令我沉默》诗集，虽然觉得我的触动跟它是诗的形式没有关系，但又觉得必须得是诗！ 

# **30天读书计划**

我们今天的主题是30天做一个改变，如果是读书的话，我给建议一般是做计划不要目标定得太高，如果你以后的目标是一周两本书，那么这30天里的目标是一周一本。

我看书都会讲究不同类型都有，因此可以建议大家这30天里，读一本心理学类的知识读物，比如《战胜拖延症》，一本好故事的书，比如《我这个普通人的生活》，一本能促进你做出一些改变的书，比如这本《吃早餐，彻底改变了我》。那么还有一本？我个人的习惯是计划一定要留有空白，这本书就留着你在这30天里发现新的。

# **怎样才算读懂一本书？只说知识类的。**

如果推荐大家马上去读一本书，我用人品担保这本《战胜拖延症》，这是我最近看的一本特别好的书。它符合我之前说的那些培养读书习惯的阅读选择：

1. 看这本《战胜拖延症》，它中文四万多字，英文两万多字。短，又是非常好的一本书。一定要边做笔记边读。
	
2. 从上面那本书里学来到方法，给自己想做的事情设立一个信号：比如中午吃完饭之后马上读上半个小时 “吃完午饭”是个信号。预先估计会有别的什么事儿会影响你读书？比如同事拉着你聊天，你就先预设好对策，说：等我看完的，看完我给你讲一下。
	
3. 看完了给别人分享，跟身边的人谈论这个话题，能跟人说明白了，你才真的读明白了，也就是这书没白读。
	

当然，这类型的书最重要还是使用。书里有很多方法，帮你来坚持做出真正的改变。

**最后：**

我也是普通的上班族，我每天要为老公做饭，当然还要花时间臭美，我也追韩剧《来自星星的你》。再每天都找一点时间看书。

怎样做到呢？我不想说有什么高效率的方式，最现实的是：其实，我天天就这么几个事儿。我告诉你我做了什么，但不告诉你我都拒绝过什么。不断地做减法，给自己真正喜欢的事情留出来时间。这是我一直坚守的方式。

谢谢大家！
