# 150327 王斌撞人

今天王斌在高穴，倒车时，刮倒了一个85岁老太爷。     于是乎，就开始了各种折腾，先是在高穴医院检查，结果没问题，家属不干，又要到县城检查，检查完也没问题，协商后要去庙坝交警大队，取回行驶证，取回后呢，那边家属接了电话，突然变卦，要求赔偿一万元，我们只接受一千五。 紧接着又是各种消耗内力，孙家权也牵扯进来了， 然后到了晚上七点，交警也要下班，就开始打圆场，让我们交一千元资压金，再去把本来扣了的车开走，再接着就偷偷把车给放了， 王斌和王海梦就开车回大竹了。 回来的路上，对方家属还来电，说要再去住院，让王斌送病历过去，我们商量了下，这个真不能过去了，一过去就走不脱，又会被耗起，而且这次不会轻易放王斌走了。

不去管他，让家属们折腾。

不过有一点，我觉得王海梦很不错，就是，胆子大，这种场合下还是特别稳得住，比如王斌先离开了一会儿，她就一个人留下来，态度强硬坚决不给钱，而且我打电话时，她心情还多悠闲的，点赞。  中途交警也来电话，希望王斌他们回去，王海梦说的就比较懂行情，大哥，你处理了这么多事情，肯定懂得起，喊我们回去是要什么，你也很明白。如果那老年人真有啥子事，我们肯定不得跑，何况还有保险公司在那儿，请你放心好啦。    交警也拿她没法，我估计也想不出这么合理的推脱词了，两个字呵呵。
