# 投资人说辞1.0 2021年03月02日17:44:35

* 幕布链接 <https://share.mubu.com/doc/4pLKnZVBec>
* 背景
	* 航空航天项目经历，发现问题：产业的强大往往是产业供应链的强大
		* 8年航空航天经历，负责项目管理时，和设计总师，物资沟通时发现外协沟通的方式低效，传统。 比如设计师需要就某个模块,沟通参数指标时，得提需求给物资部，物资部再电话供应商询有无。 每天在基本信息沟通层面费时耗力。
		* 需要供应商配合，缩短交付周期，多备库存时，供应商也会面露难色，因为交付后正常账期在6个月以上，甚至12个月的都不少。 供应商的资金流动性不足，导致备货不足，进而交付效率不高。  
		* 在多次复盘竞标案例时，发现企业间的竞争到最后往往是各自供应链之间的竞争。
	* 身边资源刚好能解决这个问题
		* 传统市场领域的连续创业者
		* 深耕航天航空中电中船领域的兄弟伙伴
		* 在BAT的互联网伙伴
		* 金融端的伙伴
	* 再研究后发现我们想要解决的问题正是产业互联网方面的事儿，所以有了这个项目。
	* 几经迭代后，形成现在的商业模式「2.解决方案是什么？」「1. 客户是谁？」
		* 先给研究所级的核心企业建立SaaS平台【物资数字化平台】，帮助核心企业强化对供应链的管理，第二步提供应收帐款解决方案，解决供应商的资金流动性难题，用供应链金融赋能供应商客户。
		* 即我们的客户是研究所和供应商。
	* 验证商业模式 「7. 到底能做多大？」
		* 为了验证商业模式，我们做了很多实地调研，跟您简单汇报些关键数据，军工行业一年应收帐款规模1.28万亿，核心企业上千家，供应商企业上万家。 
		* 另外供应商集中程度较高，以航空某所为例，该所年采购规模60亿，对接3000家供应商。 这意味着，推动一家核心企业采用我们的数字化平台，即可覆盖3000家供应商客户。
	* 盈利主要来自几方面：「 3. 怎么赚钱？ 」
		按这种商业模式，能产生多大收益呢？
		* SaaS收入
			* 前期集中1~2家核心企业，SaaS收入140~280万
		* 供应链金融收入
			* 以目前的12家为例，做模拟测算。
				* 现合作供应商12家，平均每家持有应收帐款5000万，合计可融资规模6亿； 综合收入按0.3%～6%计，依托现有客户群可实现180万～3600万营收。 综合预估900万。
				* 融资后可实现合作供应商600家，平均每家持有应收帐款按1500万计，合计可融资规模为90亿； 综合利率按0.3%～6%计，可实现2700万～5.4亿营收。 综合预估1.35亿。
		* 流量广告类收入
* 我们的团队「为什么我们能做？，为什么只有我们能做？ 」
	这里又有另外一个问题，就是为什么这个钱我们能赚
	* 我们团队的核心班底来自哈工大校友系，大多分布在军工领域和互联网领域。 对这个项目直接的帮助在于，军工行业特有的规则和红线能够有效应对，比如CMO王鑫在航天领域负责项目多年，深知行业里的坑洼在哪儿，在互联网科技的伙伴，有极深的技术沉淀，比如CTO是蚂蚁金服的张勇勇。
	* 当然，这只能说明我们能做这件事，不代表只有我们才能做，要是类似于咱们这结构的其他团队来做的话，光凭军工行业和互联网行业的团队优势，我们算不得独特。 
	* 让我们真正有底气的是两点：  
		* 1\. 行业核心人脉资源积累。 军工行业有其特殊性，比如这行业的市场透明度远低于其他行业，这意味着，关系资源有其不可替代的前期优势，在这方面我们有多年的积累。 同时，我们用战略合伙人制度，把重要的资源干系人，变为了利益相关方，而且这套制度可以方便地与更多的战略合伙人合作共赢。 这为项目早期的冷启动，奠定了基础。  
		* 2\. 团队跟金融口资金方的对接能力。我们的商业模式里，一开始就把供应链金融当做重要的盈利点，是因为团队中有京东前高级副总裁李大学，中兴战投的郭枫，以及原建设银行原董事长和原中信银行，微众银行行长，一众大拿给了指导和支持。 这些支持让我们对接到了足够多协作界面良好的供应链金融资金方，这让我们一开始要对供应商做金融赋能时，底气十足。 
* 我们近期的计划是，
	* 签订第一单核心企业的SaaS订单，跑通针对供应商的商票融资额业务，实现第一阶段6亿融资额。
		* 目前完成核心企业合作意向，已立项，现推动合作协议签订；
		* 有12家供应商客户明确意向，应收/商票融资合作协议陆续签订中， 平台上线后首批融资规模有望达到6亿；
	* 商票业务跑通后，我们的目标是将合作供应商由12家拓展至600家。完成开发并上线SaaS平台，闭环天使订单。
* 我们测算后认为实现这些目标，我们需要融资500万。  
	* 其中供应商客户拓展和研发团队投入占比较多，预计需投入250万左右
	* 利益相关方的关系维护和外协开发，预计156万左右
	* 整体运营费用预计94万左右
