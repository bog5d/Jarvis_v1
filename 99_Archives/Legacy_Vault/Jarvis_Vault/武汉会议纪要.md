# 20231127武汉会议纪要

# 一、**会议主题：**

1. 关于业务1【视频IP】和业务2【私董会】的产品价值及方向
2. 业务3【股权基金】的路径及行动规划

# 二、**会议时间：**

		2023-11-27

# 三、**参会人：**

		LuLu，Bing，Bo

# 四、**会议共识：**

1. 团队长效沟通机制：每周一下午两点半连线沟通，若有临时安排提前沟通。
2. 业务1【视频IP】和业务2【私董会】的产品要素及价值：
	* 培训指导。 授课并帮助专精特新民营企业成为民参军企业；【具体形式：军工大咖培训+帮助企业军工资质获取】
	* 商业访谈。  以契合企业甲方客户和资本关注点的方式进行访谈展示；核心是增加市场曝光（相对间接），资本投资机构的曝光。
	* 股权投融资。  跟企业有访谈链接后，帮他解决融资问题，基金直投或融资FA里赚到钱，因为有采访，可以签独家。同时我们能解决大部分FA只做传话筒而没有深度赋能的角色。  另加入并购等业务， 若早期的企业，可以收部分费用和股份并行的形式合作。
	* 供应链金融的服务。 大部分企业都有账期，供应链金融协助供应链融资。
3. 关于参加私董会的粉丝能得到的价值梳理：
	1. 创始人专访增加企业市场曝光，
	2. 跟100家上市公司创始人或高管链接交流
	3. 能链接到500家以上投资机构
	4. 提供高效/千万级的供应链金融解决方案；
4. 关于基金业务的行动路径：
	1. 盘点手里可访谈对接的企业：
		* 第一类：能证明我们投资能力的，已经上市的公司。 【链接该类公司的路径可以是带上券商研究所去采访老板，若采访不到老板，我们讲投资的逻辑，或是剪辑方式展示】
			* 盟升可从配套华为卫通角度，有可能；
			* 亿华通，氢能第一股；
			* 腾盾现在也一百多亿了，可以找个所长来对谈，这种视频出来业方便募资啊；央视视频也可以剪进去的；
			* 星际荣耀，回收火箭；
		* 第二类，影响力足够大的，能为我们背书的公司。
		* 第三类，正在募资，需吸引LP进来的公司。
	2. 确定步调，一个月2-4家访谈。
		* 访了一个企业后，多维度出视频 ，【比如切片】；
		* IP和口播要输出内容，观点，【可用数字人】；
		* 内容形式：相当于把程前朋友圈+今日思考的内容放在一个IP上；
		* 配合种评论，自动匹配投资需求群体，吸引潜在LP群体。
	3. 基金组建
		* 组织去看企业，建立去企业的信任，以及对我们的信任；
		* 关于平台的选择和基金注册等运作；【现可选RX和经协】
5. 其他要点：
	* 爆点前置，才是视频获得流量的核心。   比如3年翻10倍的企业怎么投出来的； 如果被黑也不一定是坏事，可能会增加讨论度 只要项目是真实的，就ok。
	* 关于选RX作为基金合作平台，RX的优势是在圈内项目质量高，受认可度高；  如果ta很重要，也可以牺牲一部分利益去维护关系，即也可以让渡多一点比例给到RX；【若投RX已投过的项目则不宜选其做平台】。
	* 关于募资：
		* 2B募资应该会比2C募资效率高一些，我们开始阶段多推进2B募资方，分利润出去，提高效率；
		* 2C的LP资源，黑马可能没那么匹配，长江VC班和沙丘匹配度可能要高一些，但需花时间经营。

# 五、最近一周**行动项**

1. 商议敲定并落实合作协议【LuLu，Bing，Bo】；
2. 梳理出2B和2C的募资渠道【Bing，LuLu】；
3. 提前约2B和2C意向出资方【LuLu，Bing，Bo】。
