# 录音

* 砍价  客户肯定是会砍价的
* 销售技巧匮乏
* 坐着挖指甲的导购 肯定不会有强大的意愿去接待客户。
* 必须讲环保，讲到环保必须讲到“死”方面，讲到孩子，讲到健康。很多人讲得没感觉，
* 很多人端水时用一只手，销售做的是服务。
* 销售人员报的意向客户一般会多报。

* 吊顶的颜色只能浅于墙面的颜色 ，销售不是卖而是帮助客户买。
* 边缘知识很重要。
* 相思木案例。
* 不会微笑的不要开店，赞美，讲故事。
* 一个家没有厨房是宾馆，厨房文化就是爱的文化。
* 客户都有表现的欲望“这个可以有”，“这个真没有”。

* 哎呀，我的烟机装在你家里比我门市上好看多了。
* 不傍大款
* 留了电话，要短信跟踪。短信肯定不算骚扰，一条短信的力量

[2014年6月4日 AM11-44-30.amr](./_resources/录音.resources/2014年6月4日 AM11-44-30.amr)

* 短信应该怎么发：作为个人我不想失去您这个朋友，作为这个品牌不想失去您这个客户

[2014年6月4日 AM11-49-28.amr](./_resources/录音.resources/2014年6月4日 AM11-49-28.amr)

1. 先介绍品牌，再介绍自己，分两条信息发
2. 不是因为销售服务，而是有了服务才有销售。
3. “你肯定是第一次看”，一次看，二次看，我都说好看，客户肯定会觉得好看。
4. 赞美不停，鼓励不断，销售员必备。接受赞美时，智商刷刷刷往下降。
