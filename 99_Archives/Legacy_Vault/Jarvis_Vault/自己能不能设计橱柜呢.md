# 自己能不能设计橱柜呢

我在想乔布斯能设计出简约风格的完美产品，尤其是带员工去看法国的玻璃艺术品时说”既然我们都是做设计的干嘛不把产品也设计完美点”
  如果我只能在销售环节，不能主宰我的产品，那橱柜这一块是不是可以设计完美点呢？
