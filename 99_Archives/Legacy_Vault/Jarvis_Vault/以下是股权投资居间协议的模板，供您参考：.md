# 以下是股权投资居间协议的模板，供您参考：

以下是股权投资居间协议的模板，供您参考：

\*\*股权投资居间协议\*\*

甲方：（投资机构名称）

乙方：（项目公司名称）

鉴于：

1. 甲方是一家在（所在地）注册成立的投资机构，拥有一定的资金实力和投资经验；

2. 乙方是一家在（所在地）注册成立的公司，具有（项目概述）的经营计划和运营能力；

3. 甲方愿意向乙方提供资金支持，以获取乙方（项目名称）的股权；

4. 乙方同意接受甲方的资金支持，并将其用于（项目名称）的开发和运营；

经双方协商一致，达成如下协议：

第一条 投资金额

1.1 甲方同意向乙方提供（投资金额）的资金支持，用于乙方（项目名称）的开发和运营。

1.2 投资金额将分期支付，具体支付时间和方式由双方协商确定。

第二条 股权转让

2.1 乙方同意将其（项目名称）的股权转让给甲方，转让比例为（股权比例）%。

2.2 股权转让的具体时间和方式由双方协商确定。

2.3 甲方支付全部投资金额后，乙方应立即将股权转让给甲方。

第三条 投资收益分配

3.1 在乙方（项目名称）运营期间，甲方享有（股权比例）%的股权收益。

3.2 在乙方（项目名称）退出时，甲方享有（股权比例）%的退出收益。

3.3 退出收益的计算方式由双方协商确定。

第四条 保密条款

4.1 双方应对本协议的内容和执行情况保密，未经对方书面同意，不得向第三方披露。

4.2 因一方违反保密条款给对方造成损失的，应承担赔偿责任。

第五条 争议解决

5.1 本协议的履行与解释均适用（所在地）法律。

5.2 因本协议引起的任何争议，双方应友好协商解决；协商不成的，任何一方均可向（所在地）仲裁委员会申请仲裁。
