# 想通过提高驾驶技术和减少驾车坏习惯减少油耗，从哪些方面入手？

题主！如果你不说说自己的**具体情况**，这种问题会很难回答啊！比如：你开的是手动挡还是自动挡？车的排量有多大？平时用车的频率以及常走的路况？你生活的城市拥堵情况怎样？既然没有给出以上信息，我只能尝试给出一些**普适的建议**。
首先，网上有很多**省油技巧**，题主应该先学习一下这些**理论基础**。可以在知乎搜索**“油耗”、“省油”**或者百度**“省油技巧”**等等……这些理论都是大同小异的，总结一下，基本就是以下这些：
1.确保发动机工况、胎压正常；
2.平稳换挡，温和加油，避免频繁刹车及换道；
3.避免长时间怠速、长时间低速行驶、短途行驶；
4.减轻负载，减少用电，合理使用空调。
我重点要说的不是以上这些，而是想提醒题主注意，**。我相信题主也不可能为了省油不顾行车安全与舒适性。比如频繁刹车费油，开车的时候真的就不顾安全少踩刹车了吗？减轻负载省油，难道题主参加完聚会就不顺道送女神回家了吗？少用空调省油，夏天就一直开窗了吗？其实在高速开车，开窗由于增大风阻，还不如开空调省油。我个人以为，开车，。下面我就重点谈谈上面那些理论应该如何科学地运用。
首先我想说的是，最最最省油的技巧其实被很多人忽视了，那就是——**。出门前查好地图做好合理规划，最好给爱车配个导航，绝对是最有效的省油手段。本来10km的路程出错一个出口跑成15km是常有的事，找不到目的地慢速沿路找寻既危险又费油……总而言之，**如果不能保证少走冤枉路，一切省油都是空谈**。

再然后就是推荐题主使用一些手机APP，包括**地图软件**（高德地图、谷歌地图等）和**油耗计算软件**（小熊油耗等），当然好用的软件还包括停车软件和违章查询软件等。活用手机地图一方面可以做好出行规划，另一方面可以**避开拥堵的路段**。宁可多绕几公里不能堵死在路上，拥堵时段绕点路反而会省油。另外建议题主养成**错峰出行**的习惯，高峰时段出行，即便选择最好走的路，油耗猛增到畅通时的**2倍以上**也是很常见的。路畅通了省时省油又省心，何乐而不为呢？其实，，做好这3项绝对能帮你省**一半以上**的油。至于那些驾驶技巧，做得再好也就是节约个**一成**。油耗计算软件可以作为行车电脑的辅助，你可以看到季节、气温对于油耗的影响，更重要的是可以真切的感受到自己通过改善驾驶习惯让油耗越来越低，这种**成就感**是很重要的。
最后我想说说。其实与省油相关的驾驶技巧，别的地方也多次提到，核心就是少踩刹车。原理很好理解，本来加速到80km，如果不踩刹车能在不耗油的情况下滑个1km，结果踩了刹车就只走了500m，距离减少了，百公里油耗自然增加。但**行车首要是安全**，怎么可能少踩刹车呢？其实我个人的理解是，。前方500m有红绿灯就别再猛给油了，**提前做好预判**。其实驾驶技巧看多了，说来说去就是一条：**。
