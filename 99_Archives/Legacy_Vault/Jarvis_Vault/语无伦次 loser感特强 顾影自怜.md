# 0623 语无伦次 loser感特强 顾影自怜

1. 车祸记录
	1. 2014.6.19日晚10点左右，开面包车到南外想去接开会完毕的导购，结果在金属回收站附近的一个人行横道上撞到了一个48岁叫谢波的男人。我脚穿拖鞋，没有开远光灯。那条路没有路灯，我没有看见人行道上有人，直到撞到人才反应过来，然后我大叫起来，双腿无力，大概五秒后才踩刹车了，那人被推行了约10米，我才开倒档回了一点路程。
	2. 而后紧张的我先打了120急救，然后电话小马哥说我撞了人啦，马让我打保险公司电话。我联系保险公司，报了发动机号码。交警让我交了驾照，并拿走了临时牌照，而后把车也拖走了。紧接着，交警要求我去医院把钱付了，我等到小马哥来了后再一起去的达县中医院。
	3. 小马哥帮忙付了2500元，然后去检查，CT显示大脑内部正常，然后手指骨折了一根，身体多处擦伤。他家属朋友来后要求转医院。后来就转到了，三里坪领域广场附近的达县人民医院。不过谢波的熟人在这儿当主任，就只交了100元钱。我们凌晨一点才回来。
	4. 第二天，我们要准备装家军团的活动会场， 所以忙了一上午，下午本来和保险公司一起去的，结果保险公司放水了。等到晚一点的时候，梁姐催我了，我才去的。去交了3000元钱，然后买了一箱六个核桃过去。就回来了。
	5. 第三天，也就是前天，装家军团的活动开始了，我一上午还是专心搞活动去了，下午梁姐电话催我去把手机给买了，我说确实要晚一点。然后就让抓了点钱交给李大育，王海梦也在想办法拿内部价手机。我就连夜下了大竹。
	6. 第四天，也就是昨天，李大育把手机送过去了，同时还把之前转院的钱结了。
	7. 第五天，也就是今天，梁姐给我打电话说还是要让保险公司去一趟，万一有啥闪失，我就约了保险公司明天去看看。我也给柏警官打了电话，咨询一下要取车要走哪些流程，他说明天去交警二大队三楼找他就是。车子在西外什么地方停起的哦，没问清楚。明天就去交警二大队看看能走什么流程。。。。
