# 想知道你为什么会衰老，会得心脏病，高血压，糖尿病吗？

想知道你为什么会衰老，会得心脏病，高血压，糖尿病吗？

那是因为吃进去的食物产生了胰岛素抗性，要想知道胰岛素抗性。

需要首先了解胰岛素。

胰岛素是胰腺产生的主要作用是降血糖
（血糖就是人体吃进去的米饭，馒头，面条分解后的产物）

当我们吃了一顿饱饭，血液中的胰岛素就会升高，胰岛素能帮助细胞利用血糖当作能源供人体活动。

胰岛素抗性又指什么呢？
我们把胰岛素想像成一名交警，血糖就是刚刚发生的交通事故。我们都希望出警勤快的交警，但每次事故都叫交警，时间长了，交警出勤的积极性就不高了，这就是胰岛素抗性。

那怎么样减少胰岛素抗性呢？
就要多吃复杂碳水，就是粗粮。
所以有条件把白米饭换成杂粮饭，白面条换成荞麦面或意大利面。
