# EverMemo

兄弟，跟你同步一下进展，本轮融资我们对预期和节奏有了新的调整，估值和回购等不变，有调整的地方主要是两点：
1\. 本轮统一不做对赌等约定； 2. 融资节奏我们考虑放缓，24年内完成融资即可，老大意见是泽天还是专注做业务，感觉这两月为了融资大家有点偏离初心；

近日鼎信那边跟咱们的考虑也一样，认为业绩不及预期，也想加入对赌条款，我们想要单独给某家机构对赌看来并不可行。坦白讲我们并不担心未来业绩，但会担心企业因为资本期待，失去战略定力；

昨夜老大召集各高管深度交流，形成结论是1.回归初心，我们当初开启融资是为了发展提速，而非要有融资才能生存，若暂时没太相互匹配的情况下，我们跑慢一点就行，不为了融资成功而融资；
2.对资本我们开放合作，但保持战略耐心，若有缘分能达成深度共识，携手同体，我们定不负所托；

所以老大给我的融资工作安排，由之前的3月份完成延迟到24年内有结果就行，让我也别太着急，咱们天惠这边大家保持沟通，在合适的时机大家共同推进🤝🤝🤝
