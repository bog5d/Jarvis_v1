# 帮核心企业（山东潍坊某城投）融资案例

[EverRec 20210820_213729_Realtime.mp3](./_resources/帮核心企业（山东潍坊某城投）融资案例.resources/EverRec 20210820_213729_Realtime.mp3)

山东潍坊的一个城投，有几百供应商，
城投当时有近一亿现金缺口，需要融资。
我们通过帮助其银票变现，实现三千万杠杆融资1亿的资金支持。
具体如下：

1. 该核心企业银行里面存三千万，用三千万做个保证金之后拿到了银行一个亿的银行承兑额度
2. 接下来这一个亿票开出来之后，就用我们的资金，帮他把一个亿的银行承兑变成现金，
3. 通过合规的贸易背景给反馈到了山东潍坊城投。
4. 以此帮助潍坊城投的融资部老大实现用三千万撬动一个亿融资的杠杆效应。

这是我们帮助核心企业融资的一个案例。
