# 各位领导及合作伙伴好，我是泽天智航的投融资负责人XX，很荣幸跟大家汇报我们泽天的企业情况。

各位领导及合作伙伴好，我是泽天智航的投融资负责人XX，很荣幸跟大家汇报我们泽天的企业情况。 
记得第一次市里面某领导来视察看到这幅图，有领导打趣地说“呀，航母图片都放出来啦，你们家造航母的呀？”，航母肯定不是我们造的，但航母上的军用电子有我们的。 
因为我们泽天的方向是做国防装备信息化 智能化，自主可控这方面； 从大行业来讲，属于军工电子，同时，我们在该行业的两个细分领域属于国内民营企业的龙头和引领者，一个是信息化装备的核心处理单元，一个是算力单元。 这个我们后面会再展开细讲；我们的核心处理单元和算力这些年海军和空军应用相对较多。 所以这幅图我们想要表达的是我们产品的重要应用场景

国防装备自主可控 信息化 智能化一定是大势所趋，从2个维度的来看，一个是国家层面，2022年，习总书记在求是杂志发文说”推动关键技术自主可控，事实证明我国在这方面大有可为“，其实是着重认可了这几年我们国家走自主可控取得的进展，并且未来一定要坚定地走自主可控； 
二是资本市场层面，中航证券也公布了军工电子行业的深度研究报告，从资本角度重点看好军工信息话 智能化，认为未来是军工信息化和智能化的大时代。 因此我们泽天也时常感觉自己很幸运，我们一直深耕在这个大势所趋的行业里面。

图3
公司的情况18年正式运营，注册资金1000万； 经过这6年的发展在行业里做到了一个唯一和一个唯二；
首先这个唯一呢，是国防装备信息化领域唯一打通核心处理单元到核心分系统在到整机以及底层驱动核心代码和底层操作系统的民营企业；军工装备的产业生态从原材料开始，往上是元器件，再往上核心处理单元，然后是分系统，最终交付品是整机。 而我们除了原材料和元器件不涉及，其余都有而且底层核心代码和底层核心操作系统，我们都有打通。  在军工装备民营企业里，我们是唯一一家； 
其次是这个唯二， 军用舰船公共算力单元领域我们和上市公司国家队浪潮电子，是唯二能做的。 而且我们在交付时浪潮的启动时间四十几秒，而我们的不到十秒，就是因为这里面的底层驱动核心源代码，是我们根据这个场景自己写的。后来浪潮还想找我们买底层驱动的核心代码。 

核心产品有8个，涉及海 空 陆 战支等多军种。 关于产品后面会详细展开，目前我们在国防装备信息化 智能化核心处理单元和算力单元领域属于民营企业头部。

关于发展历程
运营至今已有6年，前三年拿资质和海军为主算是站稳脚跟（用移动互联网的说法，我们前三年养号➕业务储备），近三年开始布局市场和产品的生态；
其中有三件事，我认为对我们企业有着里程碑意义的作用。 一个是20年海j火控和指控系统核心处理单元和舰船公共算力单元大获成功，这算是把我们这家创业公司在舰船装备核心处理单元领域奠定地位的两场胜仗，全公司都很振奋；
第二是21年，海J鱼雷🥚载核心处理单元，空J无人机任务管理机的正式交付，正式进入🥚载市场和无人机市场，从俄乌战争大家都知道，🥚和无人机本质上都是消耗品，未来的量很大，能进这两个市场我们业务上限一下拉高了一个数量级；
第三件事，就是今年，唐总下定决心，我们泽天一定要开发自主可控的嵌入式实时操作系统，这个意义很重大。 首先我们军工需要这种高可靠性系统，该系统特点后面讲到时再展开，更重要的是，这其实是基于要建设整个军工电子行业基础设施的角度出发的， 同时这也意味着我们真正做到了打通核心处理单元到核心处理系统到整机再到底层驱动核心代码以及底层操作系统的生态型企业，这在军工民营企业里目前是唯一一家；
