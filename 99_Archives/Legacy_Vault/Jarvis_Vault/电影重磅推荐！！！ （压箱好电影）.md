# 电影重磅推荐！！！ （压箱好电影）

1、《恐怖游轮》又称《迷失三角洲》极度牛X的片子！令人绝望的轮回……完美的衔接……在相册刚刚建的是就有人推荐这部电影，真的超级后悔自己现在才看~！重磅推荐！！！

2、《万能钥匙》看过以后回味无穷！~真正做到让人后怕的电影！

- [ ] 3、《致命魔术》重磅推荐！！！这部片子无论是从制作到编剧都堪称经典，它给你的惊喜是一直贯穿于整个电影之中，不仅仅是结局！！而且男主角长得很有贵族气质~

- [ ] 4、《绝命调查》又叫《命悬一线》凯特温斯莱特和凯文·斯帕西影帝影后联手打造，及其非常绝对好看！结局已经不能用意外来形容！不但剧情经典，最难能可贵的是这部电影的节奏掌握的极好！紧凑而且环环深入……爱死推荐这部电影的朋友了！！

5、《盗梦空间》真是千呼万唤始出来！~~带着观众游走于梦境与现实之间，被百度百科定义为“发生在意识结构内的当代动作科幻片”~~

- [ ] 6、《心慌方》系列又称《异次元杀阵》，有点老的系列悬疑片，还是一个记者姐姐介绍我看的，不过编剧真的很赞，让人绝望的编排！

- [ ] 7、《生死停留》看一遍不太好懂的电影，关于灵魂的电影！~看懂了就会有想流泪的冲动~

- [ ] 8、《孤儿怨》让人心颤的结局，让心惊叹的演技！！看完你一定不后悔！！

- [ ] 9、《小岛惊魂》很好很经典，个人觉得是妮可基德曼最巅峰的片子了！！

10、《禁闭岛》经典之极……莱昂纳多演技大爆发！
11、《寂静岭》很经典的恐怖片，根据游戏改编~看完之后，那警报声在脑海里久久不能消散……

- [ ] 12、《如月疑云》很别致的一部悬疑片，整部片子两个小时完全是由五个男人在一间屋子里的对话推进的！抽丝剥茧，层层揭秘，个人觉得唯一美中不足的是结尾的部分稍显拖沓

13、《杀人回忆》结局很让人压抑…环境也很压抑，是导演故意不让凶手出现吗？韩国电影的集体回忆！让人窒息！
14、《迷雾》这部电影最大的亮点就在于结局，看惯了英雄主义的大团圆，这样的独树一帜的反英雄主义的结局才是真正的现实主义！否则这部电影只会沦为一部二流的科幻片……
15、《八面埋伏》又名《死亡特训》，情节紧凑，气氛营造比较成功，有点“死神来了”的feel，不过跟过程比起来结局不算是亮点~~

- [ ] 16、《火柴人》大爱尼古拉斯凯奇~~结局确实没猜到，不过个人感觉前面的铺陈实在是长了一点~~
- [ ] 17、《神秘窗》影片再次证明史蒂芬.金是永远不会让人失望的！~而德普显然是锦上添花！

- [ ] 18、《不请自来》，没看过简介你永远猜不到结局，结局让人心中一紧！根据韩国电影原著“蔷花红莲”改编，据说韩国的更加可怕……

- [ ] 19、《死寂》恐怖片中的经典巅峰！！bewarethestareofmaryshaw双目瞪人玛丽肖shehadnochildrenonlydolls傀儡为子常怀抱andifyouseeherinyourdreams汝儿小辈需切记besureyounever,everscream梦中见她莫尖叫这段翻译让整部片子恐惧中带有诗意！

- [ ] 20、《致命游戏》又名《心理游戏》极度感谢推荐这部电影的朋友！！伟大的结局，伟大的史蒂芬金……在看过众多此类型片之后结局依然把我惊呆了……重磅推荐！！！！

21、《搏击俱乐部》伟大的编剧，伟大的电影，不多说！
22、《猫鼠游戏》又叫《逍遥法外》莱昂纳多与汤姆汉克斯的斗智斗勇，大爱汉克斯！！！重磅推荐！~
23、《完美陌生人》我承认这部电影我确实没有猜到结局~~！建议大家看第二遍，否则会错过很多精彩的细节！~！
24、《水果硬糖》绝对的心理折磨，心理暗示！！~
25、《终极面试》禁闭密室悬疑片，还不错哦~
26、《假面》镜头剪辑相当精彩炫目！结局意外之极~~爱死这部电影了！！重磅推荐！！！
27、《魔术师》虽然猜到了结局，但是还是一部制作精良的电影，总的来说悬念设计还是不错的！~诺顿依然那么帅！
28、《老男孩》韩国经典剧情片，看到结束才能领悟深意！~
29、《逃之夭夭》很精彩的一部片子！~~最后才恍然大悟！法国的此类影片确实和美国的风格大不相同，如果可以的话，我想说这部电影结局意外得“很有气质”！PS：苏菲玛索美呆了……
30、《十二宫杀手》大卫.芬奇继《七宗罪》《致命游戏》《搏击俱乐部》之后的又一犯罪悬疑片，根据真实案件改编，不过个人认为这部电影的成就并没有超过前三部~片子2个小时40分钟，略嫌拖沓，最后依然有很多的谜题没有解开，看到结尾有一种被骗的感觉……
31、《天才雷普利》马特达蒙的表演令人叫绝！人物的复杂多层次性格把握的恰到好处！还3有“惊艳”的裘德洛！希腊雕像般俊美的脸庞简直迷倒万千影迷！~~
32、《黑暗侵袭》个人觉得太过血腥了，但是全女性视角还是比较独特的~结局也比较意外！~
33、《天使与魔鬼》《达芬奇密码》的前传，汉克斯虽然已有斑斑白发，但依然魅力无穷！！~
34、《七宗罪》经典是犯罪心理电影！！结尾很完美！~~
35、《捉迷藏》结局差点让我窒息！！如此经典的恐怖片~~小范宁是这部片子的全部！
36、《赛末点》伍迪艾伦的电影总有着他独特的风格，电影中一如既往的有他的御用演员斯嘉丽约翰逊~
37、《达芬奇密码》汉克斯，你为什么演什么就像什么呢？
38、《1408》，虽然没有吓到我，但是史蒂芬.金的电影还是要顶一下哦！~还有牛B的约翰库萨克！！！
39、《灵异第六感》一直是恐怖片的代表之作，经典之作！
40、《七磅》开头所有人都看不懂，但是请坚持到最后，你会深深地感动！
41、《人骨拼图》丹泽尔.华盛顿，朱莉早期作品，常常与《七宗罪》相提并论，但是个人认为和七宗罪相比还是有一段距离的，但是演技依然没话说~~
42、《黑洞频率》出人意料的结局，体现父爱的伟大，结尾处让人紧张的流冷汗……重磅推荐！！
43、《死亡录像》最为绝望的恐怖片！~
44、《风声》中国电影的骄傲，原来中国的编剧也可以写出这么出色的悬疑片，中国电影在崛起！
45、《沉默的羔羊》心理电影的经典之作！膜拜……
46、《闪灵》被推荐看的，结局真的很朦胧！~
47、《乱战》前80分钟是普通的猫捉老鼠，后13分钟悬疑出现了……中间就猜到了结局，感觉就是一部动作版的《非常嫌疑犯》……不过也算得是警匪片中的佼佼者了！
48、《罗生门》50年代的片子，黑白、粗糙、纯朴但是直指人性！黑泽明的表现手法独特有力，深刻的揭示人性中赤裸裸的懦弱和谎言，如今“罗生门”这个词已经可以用于直接表达一种社会现象~可见这部年代久远的片子对后世的影响！
49、《电锯惊魂》很牛的结局……
50、《非常嫌疑犯》这样的片子谁能不喜欢~
51、《魔鬼代言人》基努里维斯的经典之作！！~
52、《记忆的碎片》独特的倒叙表现手法，一遍永远不够……
53、《肖申克的救赎》无需多言的经典！
54、《一级恐惧》结尾虽然很出人意料，但是略显草率~
55、《伤城》突然间想到这种类型的片子国产的不止《风声》一部，还有一部更早的，我高中毕业的时候看的这部《伤城》刘伟强的片子一贯很商业，但是却商业的很有内涵！~ps:金城武帅得很犀利！！
56、《通缉令》看得出导演在尽力创造一个令人惊叹的结尾，但是却让人感觉到一丝牵强，不过总的来看，还是一部不错的动作电影！
57、《死神来了》系列，个人受不大了里面的各种血腥~~不过不得不承认编剧要想出各种令人咋舌的死法真的挺不容易的……结局真的很绝望……
58、《精神病患者》天才希区柯克的经典代表作，惊悚文艺类影片的鼻祖，惊悚片的开山之作！！喜欢剧情片的朋友一定不能错过……看过之后，你会发现之后美国很多恐怖片和剧情片对结局的处理都是模仿这部片子的！
59、《我是传奇》在音像店里和人偶说话的片段看得让人心痛，那是多么深刻的孤独啊！！唯一不足的是结尾稍显草率~
60、《香水》极度的浪漫主义！极度的震撼……
61、《空房间》一部奇特的电影~男女主角没有一句台词！展现了极度寂寞的心灵~结局意味深长……没有台词的表现显示出导演的镜头语言的叙事功力不浅！
62、《返老还童》很新奇的构思！~特别是结局，男主角退化成婴儿死在垂垂老矣的爱人怀中……顿时无言！
63、《训练日》很反映现实的一部电影~~第一次看到丹泽华盛顿演反派，依然传神~~
64、《血钻》极度真实地表现了非洲土地上地狱般的生活，钻石是美丽的，但在非洲大陆，任何的美丽为了生存都将被血色所浸染……
65、《守法公民》~又是一部高智商犯罪的电影，看这样的电影总是有一种酣畅淋漓的感觉……
66、《怒火救援》丹泽尔.华盛顿真的是太帅了！！范宁的那双大眼睛真的让人感动，怪不得一个男人要放弃自己的生命去救她！~
67、《迫在眉睫》很棒的电影！！伟大的父亲……大爱丹泽尔华盛顿~！
68、《撞车》让你看到泪崩……！
69、《闻香识女人》不同的视角诠释人生……探戈舞那段堪称经典！！
70、《秋日传奇》生命是场漂泊的漫旅……皮特一出场就让人眼前一亮！~
71、《冷山》大一看过的片子……战争太残忍了！~依然记得那段：“Ifyouarefighting,stopfighting,ifyouaremarching,stopmarching,comebacktome,it’smyr
equest,comebacktome!”……
72、《幸福终点站》凯瑟琳泽塔琼斯真漂亮…真的爱死汉克斯了！！最伟大的演员~~
73、《死亡诗社》及时行乐！！
74、《心灵捕手》又叫《骄阳似我》很好看的电影~~看完心中好温暖！印象最为深刻的还是马特达蒙那几段大段大段的独白！！台词功力真的是太好了！
75、《可可西里》伟大！~~中国第一部这正意义上的户外电影，陆川不但塑造了经典，还开创了先河！
76、《美丽人生》这是最热闹的集中营，也是最冷酷的集中营，但是因为圭多的存在，人们用笑容而不是泪水送别他，送别那个几十把人性呀摘到转角还能传来歌声的时代！
77、《蓝宇》中国经典同性恋题材电影，刘烨因此获得金马奖一炮而红！~不得不说在这部电影中他真的演得很好！眼神中都透着浓浓的爱意……这部电影我连看三遍，依然觉得余味无穷《情书》日本经典爱情……依旧体现了日本电影的含蓄内敛~
78、《天浴》唯一一部禁片，李小璐的处女座，也是让17岁的她得到金马影后的一部作品！看了很心痛……
79、《弱点》很好的励志片，根据真实故事改编，看完以后陷入了深深地感慨之中，原来一个人的人生是可以如此不同！女主角真的很伟大….个人感觉比《当幸福来敲门》好看！但是名气却没有它大~重磅推荐……
80、《拯救大兵瑞恩》这是一个关于拯救和被拯救的故事，斯皮尔伯格说：“不管你拍摄的二战电影好坏与否，其主旨都是一致的，战争是地狱，我们无需提醒，而《拯救大兵瑞恩》吸引我们的另一个地方是在战争的炼狱中，你如何找到自己的尊严？”
81、《夜访吸血鬼》个人认为是吸血鬼题材电影中的最经典的作品！哥特式电影就应该这样！~
82、《绿里奇迹》让人流泪的经典！！依然是我最爱的汤姆汉克斯~
83、《荒岛余生》汤姆汉克斯拍的最辛苦的一部片子……
84、《后天》让人反思的灾难片！~
85、《百万宝贝》经典，大爱！看到结尾依然不能接受现实的残忍……
86、《忠犬八公》真实的故事改编，真的很感人！养宠物的人看一定会哭的稀里哗啦的~！
87、《天使之城》大爱的凄美爱情片，无论是表现手法还是主题立意都极度浪漫！天使为爱坠入人间，爱人却永远离去……尼古拉斯凯奇忧郁的眼神，让人无法抗拒~
88、《赎罪》很美很经典~凯拉·奈特利的那套绿色礼服真的是美到家了！！
89、《傲慢与偏见》名著剧情基础，淡淡的浪漫基调……
90、《当幸福来敲门》激励了一批人的电影……同时这部片子的英文对白很简单，可以不看字幕就听得懂，是所有英语老师都推荐的片子！
91、《生死朗读》汉娜对自己是文盲的羞耻感告诉我们一个普通的德国人是如何变成刽子手的：看守是工作，维持秩序是准则——哪怕为此生灵涂炭！而那羞耻感，又是她人性不灭的微光，但是在整个战争中，在无数个集中营里，那点人性是不足以阻止骨子里的服从与规训……《阿甘正传》此片的经典无需多言，但是对我却有特殊意义，这是我第一次看汤姆汉克斯的片子~对他的印象永远定格在憨憨的阿甘~！\\\\
92、《李米的猜想》周迅的演技没话说的……她跟在邓超后面一边哭一边背着邓超写给她的信，而邓超却牵着另一个女人大步走着！这个画面太震撼了，眼泪不自觉地就留下来了！《海上钢琴师》这部电影有太多太多的经典片段了！斗琴的那段一辈子都不会忘！！
93、《西西里的美丽传说》和《天堂电影院》都是多纳托雷的经典之作，两部片子从音乐到拍摄手法都感觉很像~大爱……看完之后你会知道人性有多丑陋！
94、《美丽心灵》很经典的老片，很激励的结局！！大爱……
95、《辛德勒名单》永远的震撼！！永远忘不了辛德勒对阿蒙说，“在这个集中营里，真正的权利不是杀戮而是饶恕……”因为杀不值得杀的人，并不能显示力量！
