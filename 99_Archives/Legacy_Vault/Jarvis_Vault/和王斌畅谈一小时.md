# 0902 和王斌畅谈一小时

![mmexport1409674024771.jpeg](./_resources/0902_和王斌畅谈一小时.resources/mmexport1409674024771.jpeg)
- [ ] 王斌心沟通，

* 最近脾气特别差，多半是因为钱不够花了，睡眠也很差，不想去面对繁琐的事了。
* 其实也可能是很久没有真诚的反省过了，越活越浮躁。今天，从莲花湖回来，接到老蒋电话，说尺寸没有测量好，我顿时火冒三丈。  虽然他有工作不细致，我却把这个小问题放得很大。王海梦也觉得我有点过分了，想来惭愧。  今天白天窝在家里看成人网站，行尸走肉地看电脑，处理起很多事情都不想动脑壳，所以遇到问题了立马冒火，以回避需要解决的问题。然后王斌打电话来关心，也怀疑他是在查岗，就没给他好脸色，于是激起了王斌这段时间心里的所有委屈。 
* 晚上十点半，王海梦着急地打来电话，说王斌可能在哭，觉得工作没做好，十分愧疚，同时今天就是想打电话关心一哈，我却在怀疑他查岗，完全把他的好意给误解了。  （最近我自己真的很小人之心了，可能是跟王海梦觉得有点亲密，心中有鬼吧，自己首先行了小人之事，就会以小人之心看待事物，真的是哈麻批呀王波）。
* 然后，我洗了个澡就给王斌打电话，结果一聊就是一个多小时，主要说了两个问题：一是我们感情为何疏远了，二是门市的下一步怎么办。我也没找借口，直白地告诉他，我确实因为对他很失望进而不想面对他，故越来越疏远。他告诉我，现在我也不安排门市动作了，他们完全不知道该做什么。是迷茫的。（这其实的确是我的失职。）
* 我告诉王斌，乡镇应该拿点血性出来了，直接干掉烂尾客户，不产量又还嫌弃美的的那种，更关键是开个会也不来，态度都不端正，这种客户就应该拿来练手，干掉再找。  
* 零售方面，直接高薪招勇夫，2800一个月找有能力跑零售的人，然后试用两个月，期间随时可以双向选择，转正后底薪1800，三万以内的部分2个点，超出三万的部分4个点。 这样至少有个专人在管理嘛，如果搞不定可以及时干掉，只要我们的招聘渠道是疏通了的，就不怕人才流失。
* 王斌就负责乡镇和督导，财务做账。 

1. 乡镇开始砍掉烂尾客户，加速货源周转。
2. 督导，督促刘姐售后单据和老客户亲情短信，督促导购跑小区，跟犹豫客户信息，督促冷瑞兑现售后配件和费用。只需要知道他们要干啥子，自己提醒和监督流ok。
3. 财务做账，把批发出货和零售销售都录入进去，我只需要查系统信息流ok了。

* 大致做到以上三点，门市风气会大变的，活动也很久没有搞过了，压根就没人知道美的厨卫是怎么回事儿。也该干一场活动来振兴一下了。
