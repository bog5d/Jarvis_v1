# 有点想法 宣汉回来

我还是有比较水的一面哟。大致要说几点

* 比较喜欢公然演说
* 冲动装大，伤害周阳
* 看书划线，去肥捻瘦
* 跟老婆电话时一定要想到“让她开心”

可能还有几点想说的，不过忘了，就先说说公开演说方面的吧。
这段时间经历了大竹的开盘和今天宣汉的乡镇开盘，我还是找到了点开乡镇会议的感觉，哦，不，准确说是开会在上面讲ppt的感觉。
`大竹开盘那会儿，还是准备太仓促了，上去讲的时候，不太顺畅，而且一直在担心，会不会出现那种接不下下一句的情况，也就是思维中断的情况，而且`关注点也放  在了，如何把ppt的备注拿在手上，能让自己看到。
  然后导致在讲ppt时，会有过多顾虑，进而使自己更加不流畅。

1. 过多关注备注信息，使自己无法集中，是一个问题。
2. 第`二个问题是，总想在ppt里面放入太多东西，这也给自己在讲ppt时带来了困扰。我并没有做太多ppt画面，却增加了大量的备注内容，想把自己知道的美的信息全部表达出来，而备注里面的有些信息压根就跟ppt画面不怎么搭边了。却因为我特别想表达，把内容加在备注中，而演讲时看到ppt中乱如麻的备注，就无法聚焦讲东西了。而且会增加自己的恐惧感，`
3. 还有就是， 充分的准备才是自信的保障，这是余世维对女儿说的。他女儿要做一个课题，但是有点紧张，他就告诉女儿，把注意力集中在想表达的内容上，然后并告诉了女儿多做点材料搜集，其结果是，本来紧张怯场的女儿在汇报讲演时，成为了第一个上台的人。为什么呢？就是充分的准备的成果，是很希望有人赏识的，那就成为讲演时候的自信 了。  我在做ppt时就已经不自信了，因为当成一次完成任务，又想象着收款压力，就没有把ppt当成一次作品来准备。所有的准备都是根据最终能收多少钱来做的。 算不上作品，加上赶工，完成任务的心态特别重。
4. 听众的不太投入，也让我受影响不小，我用能感觉的听众其实并不够热情，而且没有多大的互动感。我估计是我们的投影效果也太简陋了，影响了听众的投入感。我们的投影每次都是卷着的，下次不能这么弄了，必须用一个kt板垫底，或者就直接投影在kt板上，再是那样款起，效果真的是太简陋了。

所以以后我做ppt时就像在周阳那儿这会儿，先思考来的听众希望听到什么？并在目录也直言不讳的说   “这些是我认为你们希望听到的东西”，“我打算讲这几个问题”
再然后，做ppt内容时，先把备注写好，再根据备注做ppt画面，如果备注内容过多，描述的点太多，要么分成几页来分别描述点，要么删减成一个点，然后做简洁的ppt画面。ppt画面能够让我一看到就能想起，我要描述的点是什么，那就好了。那能使我真正脱离备注页，走到观众中去。
好了就说到这里了。

其他我想说的，就是在周阳那儿，放政策时，有点冲动装大，觉得要给客户一种我很大气的感觉，所以就给了双重政策，其实对周阳的利润来说不太有利，就只有几个点。
还好周阳算一个比较忍让的人，不过我也知道从大账来算的话，会对市场特别有利的，我也不是特别特别愧疚。 以后遏制自己盲目装大的毛病，`既要大气，也要利息`。

看书，以后要把看书两边定为规矩，第一遍看书时不用太刻意在有共鸣的地方一直感叹，生怕以后就看不到了，而是随手把当时的心情写在旁，更重要的是把重要有营养的句子给划线，第二遍看书时，就只需要看划线的地方，那样就特别快，而且都是高营养的内容
