# 老孙

老孙，本来前天就该找你谈谈了。不过琐事占据大脑，不是很清醒。  就给你发消息吧，这样思路清楚些。
我们这是第二年的经营了，今年呢毛利估算约在十七八万，扣开门市租金大概就能剩个十万左右的纯利。 利润率不高最大的因素还是在大难题--零售上面，而我们乡镇的增长空间也快见底了，加之各个网点整体销量下滑，依靠批发想要提高利润，已经是疲乏无力，在这种情况下，我们的房租成本是在逐年攀升的，纵观概况就是，开源困难，节流乏力。
最近，在节流方面遇到个转机，就是金利美建材市场里面新修了四栋铺面起来，加上那条东柳道路从中通过，地势区位还是蛮不错的，当然关键就是价格。我们一共去谈了三次了，门市租金价格基本上确定在，600元每年每个平方，而我们需要大概80平方的门市，这种成本就在四万八一年，当然也会递增10％，按照这种算法，也没比现有门市便宜太多。   而且如果位置不优越的话，就更不具有性价比了。
现在，特别吸引我的是，有一个位置特别好的门市，是正门口进来的广场旁，以后手扶电梯旁的门市，也就是人流量更聚集的一个位置。这个铺面相当有吸引力。
不过由于，建材市场要先安置内部经销商，等他们选完了后再由我们这些外来商户选，因此拿到那个位置的概率很低。为了，拿到那个位置，我已经私下贿赂了那个招商经理了，并约定拿到那个位子后封1200的红包。  这就增大了拿到好位置的几率。
接下来，就是我要谈的核心问题，关于租金的问题。租金有两种支付方式，第一种就是年付，每年递增10％ ，600元每平米。 80平方的门市约在5万左右。 两三年后就是现在的水平。
第二种，一次性支付五年房租，一个四十平方的门市五年支付12万，到期后退回七万，实际成本在于一个门市一年一万，我们需要两个四十平方的门市，就需要24万，到期后退回我们14万，实际门市成本成了2万每年。  这是开发商的一种融资手段，也是我们的机会。我对这种一次性付5年的方式，满怀期待，我们门市三年的房租基本上就是24万了，而建材市场的24万里面还会返回14万，相当有性价比。此外，在建材市场里面的商户，更容易实现资源共享，可以加入一些建材联盟圈子，享受点集群福利。我们周边所有门市均已关门，是所有门市。这也加剧了，我们转向的动力。
说到这里，有一个现实问题就摆在面前了，我们的钱够不够拿下这个门市，当然是不够那么多，但是门市挤12万还是问题不大，收回欠款的话基本能搞定，我想的是门市上凑12万，我们三个人再追加个12万，把这个门市拿下，接下来的五年就轻装上阵的挣点钱。  你意下如何？
而且，这次调整会使我们近期内相当艰难，比如周转资金困难，会像去年一样提货可能都没钱，你愿不愿意再承受这样的压力？
目前的门市已经交了三万房租了，还有4万5左右的没交，我的计划是先保持起，明年新门市经营上正轨后，考虑转租，如果转出去了就是我们的福气，如果转不出去我们经营到明年底。
反正走这一步，我们会艰难一段时间，就跟刚创业时一样，我想去干这件事，就不知道你是否愿意再折腾这一次。
我是这么想的，到时候如果那个招商经理把事情办妥了，把那个位置给我们拿下了，就毫不犹豫的干。 我自个儿再追加5万，你和王斌一共追加七万，这事就能成。
这是我的初步想法，你考虑清楚了，给我说说你的意见吧，明天中午我要回温江一趟，最好是中午前给我消息。
