# 航空工业无线电电子研究所(615)篇

**企业背景**: 地址： 上海市闵行区紫月路，中国航空无线电电子研究所（以下简称上电所）始建于1957年，隶属于中国航空工业集团公司
**企业主要产品**:主要从事航空电子系统总体与综合，航空电子核心处理与综合应用技术以及航空无线电通讯导航技术三大领域的研究和相关产品的研制和生产。 上电所作为中航工业旗下承担飞机“大脑”和“中枢神经系统”研制的重点科研单位， 在航空电子系统综合、座舱显示控制、核心处理、无人机一体化控制、无线电通信导航、工程与制造等专业技术领域已达到了国内领先水平，并正在不断缩短与国际先进水平之间差距。
**企业背后的故事:**
**1.**2007年10月1日，中共中央总书记、国家主席、中央军委主席胡锦涛同志来所视察时，高度评价了我所为军、民用航空电子系统研发做出的突出贡献。
**补充故事**：xi大大来615所视察两次，任上海市委书记和国家副主席是各一次，预计还会来一次，这是比较罕见的。（xi大大的生日跟所里有渊源）。

2.由我所主持研制的联合式航电综合系统和综合式航电系统预研等国家级科研项目相继取得成功，并在歼十和运八等数十型现役飞机上得到应用，为我国军、民用飞机的升级换代做出了杰出的贡献,荣获了中国一航授于的“航空报国突出贡献奖”。
3. 近五年来，全所荣获省部级科技成果奖100多项，2007年荣获了[国家科学技术进步奖特等奖](https://baike.so.com/doc/8487371-8807568.html)一项、二等奖一项。
4： 连续五届十年荣获"上海市文明单位"称号
