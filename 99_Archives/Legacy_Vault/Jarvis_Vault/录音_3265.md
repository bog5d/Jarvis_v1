# 录音

10月24日日记。
说衣服对王斌的感受吧，我现在给王斌沟通最大的感觉就是，我不敢很直接的跟他谈事情，他的气场就给人一种，很是痛苦，很是压抑的感觉。
然后就会让我感觉到就跟他提要求或者问题做什么事情的时候，就不好意思开口又没法开口，他总是在向我传递一种我现在很痛苦，我现在很不高兴，或者，你会让我，再仔细的想，为，我们是不是不高兴我呀？或者他现在是不是心情特别难受啊？我现在跟他说这些好不好？
然后很多时候话题就没办法继续下去了。
你说这到底是我王波的问题，还是文明的问题，先从我自己身上找一下原因吧，我可能就是想，打孔一个人或者把空相守，如果，他表现出来不太配合不太愿意让我把空的时候，我可能会不自在吧！
哎哟，我发现，说普通话还是会出一些错别字。
到底该怎么办？那以后跟王斌的沟通该怎么沟通呢？是无视他的那种感受呢，还是多照顾她的感受呢！

当然我自己也有一些问题啊，比如我在部队，别人很严厉的时候，还是有一点没有考虑别人的感受，就像我这几天一直对王海梦特别严厉，那王海梦就很不舒服呢，特别不爽了，可是他不说还好，他作为一个女生，可以直接说出来，直接对我撒娇，再加上他本身又是一个，特别别主动沟通的人，所以这个问题，还不是很严重。但是遇到了像王斌这样的性格的人，就算是我不对把他伤害了她不舒服，他也没有办法特别正确的表达出来。然后就阴郁得像一个雨天一样，真的是让人很痛苦。尤其是让种真正关心她想她，和谐聊天的人，特别的为难。

怎么说呢，这是对于网民最终跟他聊天的时候被一种特别压抑气氛包围的感觉，我还是不知道怎么去解决？解救自己？
这个人总的来说还是太阴郁了一点，没办法就是没办法救了，也许他也不是需要我救，是不是我自己想太多了呢，我也不知道，反正这事情还是挺困扰我的至少跟他，在沟通的时候没办法达到一种，达到一种双赢的开心的结果。
我也清楚，我在处理跟下属关系的时候也不是特别完美的，肯定也会有带情绪化的一面，会不会是我平时的几句话把他伤害了，让他抬不起头了，让他没有自信呢！

现在再来反省一下你小叔的相处吧，啊，我觉得，第一个事情要八个小时的事情安排好。
第一个失球是要把小叔的事情安排好，所以，我就一直在，关注自己有没有？把安排的事情让他们，很好的完成。但是在这种情况下我就回，有的时候自己根本没有什么事情做？真的能做的事情很少，然后我就会陷入一种被电脑控制多种的，比如今天，我就一直把电脑开着，然后自己想做的事情都没做还感觉自己没有精神，没有精力。
如果这时候自己去看一点书啊，打印小猪养一下神，你会更好，狗狗好。
是不是我自己的状态还不好了，我自己做事的状态不好了。

其次，如果遇上发现自己真的没有太多事情做的时候，可能就是近段时间公司部没有太多的事情，那是不是就该放松一下自己的员工呢！ 这也是一个需要思考的问题，所以今天下午我就带他们去看一看电影哪让他们高兴一点啦！
让员工还是感觉？我不仅仅只是在安排他们的工作也在，让他们觉得有福利
