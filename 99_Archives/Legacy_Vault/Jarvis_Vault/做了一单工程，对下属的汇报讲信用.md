# 0603 做了一单工程，对下属的汇报讲信用

1. 培养好习惯
	1. 现在找写日记的感觉呢，虽然用了分段式，却不是严格按照分段来这东西的，只是这段话要说啥。怎么说呢，面对内心的来讲，我觉得利用现在的工作来培养自己的工作习惯应该是很好的。
	2. 可能是寄希望于姚姐明年的帮助，我内心做好了明年要转行的心理准备，于是乎就对当下的工作有了轻视，我自己是能感觉出来的。不过我有了一个理由来支撑自己，就是把当下的工作当成一份答卷来做，给下一任合作者看的答卷，因此需要把当下的工作做出色。
	3. 更重要的是，利用现在的工作环境和职权来，好好的培养一下自己的工作习惯，在以后的场面上能更有作为。比如现在给我配了两个人。我可以用他们来好好的锻炼自己，当一个好管理者的本事。学会分工，明确的委派，事后的监督，确定精准的KPI。委派和监督是最重要的管理习惯。
	4. 波哥真的还可以有很多提升空间的，比如扩大制度的监督力度，我定下的春风行动啦，冷瑞配件制度啦，让冷瑞每天汇报今天有多少安装维修单，收到多少张下面报上来的，欠哪些乡镇

![手写.png](./_resources/0603_做了一单工程，对下属的汇报讲信用.resources/手写.png)

* 大致就这样，不过能以最简单的方式做，是最好的。定一个制度，就加强一个制度，现在愿意好好的发工作计划的，就只有张政 李大育了，王海梦也是三天打鱼两天晒网的，冷瑞也是不提醒压根就不当回事，看来需要重新制定合理的汇报方式了，因为新制定了一些售后制度，比如七天制的配件发放，比如当天单据录入
	* 我又想到个问题，他们汇报的计划我其实并没有认真审阅，真的是应该每天拿出一点时间来专门看一下下属今天安排的是啥子，安排得好的一定要明确告诉，安排得有点图完成任务的就打个电话指出来，然后在下班前再去核对一下完成情况。我自己真的没当回事，下面的汇报我还是要态度端正点去查阅，比如设定每日查阅十分钟，每日检核十分钟。初步就定两人的嘛，李大育和冷瑞。  把他们的每日计划研究透彻，培养自己的信用管理习惯。对下属的汇报讲信用，因为我认真看了汇报，他们才会认真发汇报。

1. 做工程还是 利益分配为主，导购信息
	1. 这两天连续有工程客户来面谈，我觉得原因不仅仅是最近运气好，而是给了导购一个合适的利益分配制度，使得他们对团购工程信息更加敏感。加上，他们认为他们的权限谈不妥交给我这个权限更大的人来谈，会增加信心。因此他们更主动的去跟进了团购。
	2. 比如这次，燃热客户一直在各个商场询价，张钰向我申请价格时我就觉得苏宁那边也报了价，如果我只给张钰放价格了肯定就会让苏宁觉得我偏心，为了迎合两边我告诉张钰，我会给两边都提成，张钰会觉得这事情跑了有利，而且交给王波也会比较靠谱，因此在跟进客户时会更加积极。
	3. 把提成利益分发到位，他们权限谈不拢的有人帮忙谈，满足这两点就能极大地增大成功概率。
	4. 在谈工程客户方面，一般就把价格整到常规价以上一点，如果是中间采购商就把回扣谈到位，再其次就是给采购商私人送东西送到位，然后解决物流安装问题，基本上就算是尽人事听天命了。价格＋人情＋服务  传达清楚了就OK了。剩下的就不去盲目担忧为什么没做成了，如果这三样到位了，仍然没有合作成，那只能说明是我们的产品竞争力还是缺乏了点。
2. 今天做了些啥
	1. 上午，三人开会定事情，报告拿车借钱凭，三千大育买电脑，商场走走回家了。
	2. 中午，阿兰餐馆打包回，老婆马部家中待，午睡过后精神好，急忙上班电话回。
	3. 下午，签订电热工程定，又来燃热意向性，承诺送个好电器，让他考虑自己定。
	4. 晚上，回家蒸煮香肠粽，老婆感冒迟未复。出去散心买点药，批斗我不关心人。
