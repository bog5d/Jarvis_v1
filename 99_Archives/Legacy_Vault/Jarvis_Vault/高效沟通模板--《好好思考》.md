# 高效沟通模板--《好好思考》

回答好这几个问题，引导思考

* **沟通目的是什么？**

：建议大家看书从吸收信息到深度思考。

```
比如： 下面坐的是一群大学生，希望演讲    完了来公司工作，那这就是招聘演讲。
```

* **要达成这目的对方需要知道些什么？**

没有深度思考会怎样？
有深度思考，又会怎样？

1.我们看待问题往往不是事情本身，而是我们的思考方式决定我们看到的世界。 ABC理论
2.什么样才算深度思考？    以我们求职成长为例，来说说关于，努力到pdca，到精益思维到ooda

3. 我们如何能做到这样。   学习：2/8 阅读思考法；工作： 艇高人的思考过程

：
```
     你希望他来公司工作，那么他们需要知道什么？ 他知道了哪些事情，就愿意来公司工作？


```

* **我希望对方听完之后感受到什么？**

：

```
比如说，领导力的一个投入，我需要对方感受到的是这件事情的急迫性，我希望对方感受的是我们对这个事的这种迫切的需求。迫切，讲出来这种感觉。


```

* **对方为什么要重视/关心这件事儿？**

```
比如董事会为什么要关注我们要引入领导力培训这件事情。可不可以告诉董事会的人说，有没有想过公司为什么效率低下，公司为什么出现了员工大量撤走的事情。上一周王经理和张经理在办公室里边大声地争吵，把旁边的人都吓哭了。用事例，这些东西都可以吸引到对方接着听下去。


```

今天采用哪种坡道方式：

\*\* 用“你”：你们都是即将毕业的大学生，现在可能每个人都面临忐忑的心情，到底能不能找到诚心如意的工作，会不会被用人单位给骗了\*\*

\*\* 运用强大而有震撼力的数字： 你知道我们开会这一个小时，又新增了多少移动互联网用户吗？

\*\* 提出问题： 你们当中有多少人用了人生近一半的时间在开会，而且效率还不高？ 如果我们能节省效率的话，拿2个小时一起学习一下，大家觉得值得吗？

\*\* 用观众震撼的方法： 我们正面临一次前所未有的挑战，

\*\* 用坦白的方式： 说实话，我很忐忑，今天在场的都是身经百战的专家。 研究得越多，越不敢讲，原因在于在座的各位资历深厚

\*\* 讲故事： 今天出门之前，我女儿问我。。。。

\*\* 用想象： 想象一下十年后，我们的生活会是什么样？如果我们这样安安稳稳地工作十年，你会成为一个富人吗？你能承担起家庭的责任吗？ 你能买得起背景60平米的房子吗？ 怎么才能获得这些呢，我们推荐的方法是。。。
