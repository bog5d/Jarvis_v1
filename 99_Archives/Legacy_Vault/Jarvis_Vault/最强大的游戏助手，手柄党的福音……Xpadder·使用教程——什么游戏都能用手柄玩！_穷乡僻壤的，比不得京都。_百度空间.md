# 最强大的游戏助手，手柄党的福音……Xpadder·使用教程——什么游戏都能用手柄玩！

Xpadder是一款用手柄来模拟键盘以及鼠标的软件，基本功能与JoyToKey相似。虽然只是一个460kb的小小程序，但是高级功能强大无比，下面会慢慢介绍。。。目前最新版是5.7，这里提供下载：

⒈基本的使用方法：

请先解压到一个固定的空文件夹之中，然后双击运行~~~~并点击Start和之后的OK。

（补充说明，有人会问“为什么我的十字键和左摇杆功能一样，右摇杆和1234的功能一样？”，那是因为你的手柄分“数字“和”模拟”两个模式，一般手柄上会有一个ANALOG之类的键，按一下即可以这两个模式间切换——会有亮灯提示）

配置完成后回到主界面，用鼠标点击手柄上你想按的键，然后映射为键盘或鼠标上的键。

⒉高级应用：（以下的①②③④⑤⑥⑦⑧⑨⑩均为手柄按键）

\*首先是实现更多的按键模拟……

      可能很多手柄党都碰到过这样的问题，想用手柄模拟玩某个网络游戏，却因为手柄上的键太少只有放弃……

      以DNF为例，手柄玩家很少吧，有多少人能安心的接受搓招呢？而还有大多数根本没有搓招系统的网游就基本上不可能用手柄模拟了~~~~

      那么如何才能使手柄的键更多呢？

      我①②③④⑤⑥这几个最常用的键举个例，这里只有6个键。。。但如果把⑤键空出来，使按住⑤的时候再  按其它键即可出现一套全新的功能呢？

      这样就有了①、②、③、④、⑥、⑤+①、⑤+②、⑤+③、⑤+④、⑤+⑥，数一数，有十个键了~~~~

      而如果把⑥也空出来做切换键呢？就有了①、②、③、④、⑤+①、⑤+②、⑤+③、⑤+④、⑤+⑥、⑥+①、⑥+②、⑥+③、⑥+④、⑥+⑤，14个键了……

      其实，⑤和⑥做为切换键的情况下，也是可以同时做为单独的功能键的……以⑤为例，假如你设定了按下⑤时切换到的是第二套方案，那么你只需要在第二套方案的⑤键位置任意设置一个键就好了。。。这在某些情况下其实挺好用，比如你设了⑤和⑥两套键位，但是战斗时基本只用⑥的那一套，那么⑤就可以方便的单独设置出一个技能快捷键了……又或者，你有一个只在空中才能发动的技能，你可以把这个技能快捷键设定到⑥上面也很方便——这些都是经验之谈。

      而如果你觉得同时按两个键也太麻烦，还可以设定成这样：只需要按一下⑤，再按①②③④⑥便是一套全新的按键方案，然后随时按一下⑤又可以回来（这样省去同时按两个键的麻烦，但是需要清晰准确的操作，因为如果乱按一通的话你根本不知道自己是在哪一套配置，那样就惨了）

      以上只是举例，此用途可以延伸到其它按键中，那样一来，基本上任何游戏的按键都够了（最多可设8套按键方案）……

      ①②③④⑤⑥⑦⑧是我们最常用的手柄按钮，事实上，一个普通手柄上初始可供模拟映射的键一共可达24个——按键12个+十字键4个+左摇杆4个+右摇杆4个……

     其中十字键或者左摇杆均可担任“上下左右移动”，而在一些3D游戏中，右摇杆需要设置为视角调整~~⑨和⑩属于菜单按键不常用，⑪和 ⑫算常用但终究有些特殊（3D游戏中用到这两个键的几率比2D游戏会大很多）……

\*这里来说说一键多用

     这个功能并不能说是多么高级，设置也非常简单……效果有点死板，但其实用法可以很灵活……

     基本上表现就是这样的，比如我把手柄的①键在“高级  /  按键设定”中把键盘上的ASDFGH六个键都映射上去，并且分别用“Add cycle stop”命令隔开

     这样的效果就是，我按第一下①=A，第二下①=S，第三下①=D……第六下①=H~~~~然后第七下又重新从A开始。此功能映射的按键个数是不限的，也可以随时使用重复按键~~~~

     比较适用的情况1：用一套固定技能形成连技的游戏，比如永恒之塔，比如DNF……比较适用的情况2：切换武器……

     拿DNF刺客一套普通连技的小前部分举例，“弧光闪(A)+手里剑(S)+影袭(D)+手里剑+剑刃(F)……“那我只需要把ASDSF这几个键，按一键多用的方法映射到某一个手柄键上比如②，顺便假如④映射的是普通攻击(X)……那么我要打出这套连技只需要②和④交替的按就行了，需要普通攻击衔接的时候按④，要出下一个技能了就按②……

     扩展启示：Xpadder可以给键位钩选连发，并设置连发速率，如果你耐心调整好时间，调整好按键，把攻击(X)也融合进来，将可实现按住一个键不动，将自动打完整套连技……再加上还可以钩选“自锁"，那就可以按一下键，然后好好站起来伸下懒腰活动下身体，让它自动打完这套连技……

\*自锁   /   连发的说明

     把一个键勾上”自锁“的用途在于，你只要按一下那个键，那个键就会一直处于被按住的状态……直到你再按一下它，这种按住的状态才会被解除。

     而连发就更好理解了，一般手柄自己都带有连发功能……就是你按住那个键的时候，效果就等于那个键一直在反复的按……直到你松开。而且在Xpadder中，连发的按键速度是可以被设置的，最快1秒按50次，最慢2.5秒才按1次。

![298930f0a7d0c3dca50f5284.jpg](./_resources/最强大的游戏助手，手柄党的福音……Xpadder·使用教程——什么游戏都能用手柄玩！_穷乡僻壤的，比.resources/298930f0a7d0c3dca50f5284.jpg)

     而如果你同时把“自锁”和“连发”勾上的话，效果就等于，你只要按一下这个键，就等于这个键一直在根据你设定的间隔时间，在重复不停的按，你中途去吃个饭啊睡个觉什么的的都没关系，它会重复不停的按直到你再按一下这个键为止……

\*较长脚本的一些使用说明……

    脚本？挂X的那个脚本？没错……这个小小的手柄辅助软件，其脚本定制的强大堪比“按键精灵”。并且定制更容易上手，~~~~

    我只钻研了这款软件两三天的时间，多半是用真三国无双OL和DNF这两个游戏来测试……真三国无双OL是本身就完美支持手柄的，我只是用Xpadder尝试着写挂机的脚本（缺琥珀啦，又有武器想磨，但是自己又没什么时间和耐心去当刷子）。暂时我是失败了~~到目前为止写出来的脚本也只是帮我省去了人工操作，而不能实现真正的挂机。

    “脚本编写”的自定义属性太高，我也懒得长篇大论（其实这篇教程已经够啰嗦了）、或者贴一大堆代码出来吓人……以下只是试一下把感兴趣的各位领进门……

     正题啦，我就用一个很简单的脚本做说明……《曹操·西广场·自动挂湖沼妖怪》的脚本~~~~纯粹用来磨武器的。

     这个脚本的流程大概就是：“ 先按住W（前进）3秒从家中出门到西广场、地图切换的载入时间大概是1.30秒、出门后按小键盘区的/微转一下视角、使角色可以直接按W向前10秒走到文官面前、因为担心TAB键会选到其它玩家，所以到文官面前先按两次ESC再按TAB、然后对选中的文官按Enter弹出选项、然后按↓并按两交Enter选择执行任务、弹出是否执行的对话框时按←和Enter、等待3秒进入任务准备界面、按两次↓加Enter准备、这个任务需要等待3分多钟加上载入和退出时间算它3分50秒、然后通过Enter、←、Enter回到家中、再多次Enter弄完战果报告即可……然后再重复以上的步骤。”

     以上全部步骤需要用到的只有Add Hold Zone(添加按住区域)和添加暂停两个命令……
