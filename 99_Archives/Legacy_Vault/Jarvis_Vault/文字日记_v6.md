# 20170107 文字日记

* 怎么说呢。 参加了一次，同学聚会，并不享受。 嗯，准确地说，还是很失落的吧。  就是完全没有给人一种有温度的惊喜。 
	* 我自己觉得组织这样的同学聚会，貌似没有太大的意义。 倒不是说我自己跑了路，花了时间，不值得。  而是，我觉得在来参加了聚会的同学角度来看，也显得无意义。 也就是说，我觉得自己组织的聚会，我一开始认为是，别人未发现价值，我自己闷着头多付出一点，让别人一来，就觉得，咦，还可以呢，还好来了。   我最困惑的点，在于并没有给人，带来价值感，他们无非也是来捧捧场。 聚会可不是把人聚集拢，就算成功了。 带来价值才是聚会的吸引之处吧。
	* 来的人，其实并没有什么不好。我可能把失望情绪有点加诸于人了吧。 无论是蒋海燕还是李陈，他们的言论观点，我其实是极度不赞同的。 比如李陈的，官场黑暗论，要依托他们那种政府机构才可能成事；比如蒋海燕描述公婆，没法妥协，。  我听起来，有点腻烦感，不过并没有去表达太多不同观点。  但从内心是疏远这些人的。
	* 郑亚君，其实还比较正能量，不过也能感觉到生活并不宽裕。   不过她一直在勤奋地面对生活。
	* 刘巧，其实过得挺幸福。生二胎，相夫教子型。 也没有太大的攀比之心。 和老公幸福且勤奋着，过着典型的都市奋斗族的生活。
	* 我呢，更像是一个观察者一般，来看待这帮同学。   结论就是：以前不是一个世界的人，现在更不可能是。
