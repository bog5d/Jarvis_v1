# 1017 感觉有错 却不强烈 ---绿影不悦

怎么说呢？
我不知道是该追随自己的直觉还是应该多去反省自身，找找自己有无病态心理。

*         我为什么老对王海梦发火？

1. 我觉得这是蛮重大的课题，需要自己去研究自个儿的。到底是我习惯上了对她发火呢？还是，她真的有做得让我受不了的行为？再或是，我对她和王斌的腻歪心生嫉妒？故意 对她发火？
2. 先推敲一下第二种，就是她真的有做法让我受不了的可能性，今天我发了她两次，一次是探讨罚款的可能性，我就是觉得他反馈王平两次不够配合，反馈了两次渠县不够配合，我都去毛了客户他们的。 我可能觉得自己去跟客户毛了且撑腰了王海梦，想以探讨的形式提出来，让他们知道我是给他们撑腰了的。结果，我探讨的时候又很犯贱地顺带指责一通王海梦，这么说起来，好像是我自己的问题哈。 想邀功，不明说，装个逼，反伤人，遇不快，更添恨。只是下属们很少知道，其实领导也需要下属的认可呀。当然不是马屁哈！  
3. 对了，还有一下斗嘴，就是，我自己又去抢功。就是软文的事情，我觉得软文的图片标注很少，基本上都是我标了的几个，而没有深入一下，又把王海梦惹急了。因为她拼图的方式，她还是有新意的，我就一直贬低她了，让她毫无被认同感。这好像是我自己太他妈小贱了点吧，跟下属抢功，向下属邀功。真的都是很幼稚的行为吧，王波。所以，根据这个推测，一定是我自己内心想抢功邀功，又不去正面表达，而是拿领导探讨的方式来显摆，强引话题。
4. 实在是我自己有点贱！能不能不要想到去邀功和跟下属抢功。而且抢功的时候还要否定别人一番，实在不应该，还有王斌那个邀请函，我也是去强调只是复制了我的初稿，王斌也不停地强调没有用我的文稿。唉，王波搞得我真的有点无语，如果你都归功于别人，别人也开心地给你协助分忧，有何不可？我觉得应该改正自己的一个习惯，归功于人。是发自内心的那种，而不是为了去搞好关系而得利地去归功于人，那就成了讨好了。
5. 赞美她人的进步，归功于人。因为自己很可能已经装不下别人的优点了，如果多赞美她人真实的优点，也能让自己还有学习吸收的机会。 固步自封，想法夸已，是会上瘾的，如果不能合理的分享荣誉和利益的话。根本不会有很大的成就的，比如曹操，为什么能留住那么多人才，真的是因为看得起贤能，让贤能也觉得有誉有利可得，才会一直帮他呀。贤能为之用！ 也就是说，我现在的度量真的还没法用到真正的能人，因为我不能分享荣誉，更谈不上分享利益了。 因此我真的觉得，自己还是一个小人。真正的小人，明天尝试，夸人三次，真诚的，别人的进步或者功绩，忘记自己。因为别人对你的真赞形成了需求或渴求时，你已经就不需要别人口头的认同了。
6. 每日三赞！

* 王海梦说自己老公很辛苦呀，我就很不爽，告诉她“不要过于粉自己老公！”，我是真不爽的。当然，如果我能日三赞，这件事就过去了。但是，我就是有点受不了，她老夸自己老公辛苦，老说我对她老公骂得不太合理，不该当着外人骂。总觉得，有点作秀邀功，护短之嫌。一种恶俗之感油然而生，这是真实的第一反应哦。没有其他复杂心思在内，反正第一反应就是很不舒服，是不是只有她老公才是付出心血嘛，别人就都在耍嘛？ 

1. 但我的反应，也很可能就是那个邀功心态作祟，想要别人认可。而她把认可给到了她老公而不是我，也许我才发火的。 如果我是这样，就是自己太要面子了。不过这种看法，我不是很强烈认同的，应该有点，却是不浓。
2. 对她那种只敷自家人，的心态，有比较强的恶俗感。这是我的比较强烈的心态。
3. 说完了我，也得换位一下，也许她就是随口说说，甚至只是说给王斌听听。又或者以自己老公为荣，我就为啥不能支持认可一把呢？ 不知道自己肿么想的。
