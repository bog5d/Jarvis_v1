# Day One Entry: 2015年1月1日

\*今天把孤岛惊魂4耍完了，着实喜欢了一把。里面的风景太美了，有树林，有雪山，有宗教色彩浓厚的洞穴，还有枫叶林，尤其是结尾，找到pagin min后，到皇宫后院后，几个小房子看起来，都特别有异域风情，很是喜欢💖，体验比较到位。

* \*同时我也在想啊，随着游戏技术的发展，以后的虚拟头盔，也就是VR技术，之类的进步到能市场应用后，那些风景别致的游戏，配合超强的代入感，简直就是酷毙了。 所以呀，我必须要有好的游戏习惯，能够非常科学的保持游戏体验，而不把进步影响了。

\*我这会儿，一直在纠结，到底以后写日记，是用这个app还是印象笔记呢，再或者是有道笔记呢。后来索性一想，干嘛想得那么复杂，心情好想用哪个就用哪个，关键是当下的体验，总想把一件事情，固定成一个一直的习惯，反而会扼杀，做这件事的动力，比如写日记，我觉得应该要努力保持写日记，然后深深地反省自身，其实没那么刻意的。如果第一次写日记，尝到了反省自身带来的进步快感，就定下来，自己每天必写日记1000字。结果，第二天，也想方设法写了一千字，但是，发现怎么没有多大的进步快感呢？怎么，没有把自己剖析得很深刻呢？怎么没有焕然一新重新做人的感觉呢？ 然后，就开始质疑了，再然后就没怎么写日记了，呵呵，就又成了屌丝一个，在那里懊悔，每多一天不写日记，就愧疚一天，痛苦一天，更加挫败感。。。。 就是给自己定了过高期望值，总想一开始就做出一件完美之事，殊不知，很多事情，都是凭着兴趣，在做事时候享受着乐趣，就成为了习惯，再经过成百上千次后，就做出完美的事或物或活儿了。 更别说，把一些才开始做的事情定一个固定习惯或标准，让做这件事的快感变成包袱。 比如我的记单词，每天一篇，记了之后不复习，又给忘了，加上觉得这事有点负担感，呵呵就此放下了。我自己一定会继续拿起来的，我坚信。

# 今天看罗辑思维，社群里，有一个叫做袁贝贝的，学习陶瓷艺术的，她决定每天做一个手工艺的小怪兽，然后罗胖给她拍卖，她在2015年，打算死磕一整年。这件事，我触动很大的，这个女孩儿，也是1988年，她的这番精神，是我佩服的，我比不上她。真的有毅力，有自己的风格，敢做，不怕失败。毕竟365天，一天不间断才是最大的挑战，她决定这样做后，可是有全国的罗友看着的噢！ 如果她这365天坚持下来了，真的会成为一个很有腔调的人的，通过这事儿，要学到好多人生哲学和艺术技能，又为日后奠定了基础，当然她都配得上这些，3685的坚持，不是任何人都做得出来的。转念一想，如果，她是真乐在其中的话，应该很容易搞出来的，我会关注她的。

* \*\* 关于坚持，这项人生技能，真的会有很大的威力。比如罗胖，每天六十秒，就是因为每天六十秒，让每个听众听到声音了的，这就跟其他公众号用文字推送完全区分开来了，感觉像真有一个人在跟你对话交流，亲切感完全不一样。而且，真的天天在发，每天都在坚持，像我们这样的听众，都听了两年了，如果哪一天突然没听，都会不习惯了，这是多大的影响了，所以现在罗辑思维能成功，是必然的。创新＋坚持，成为了他的法门，我看好他佩服他，愿意为他花点钱，因为真的折服我了。
* \*\* 我也应该有个值得自己死磕的事情，练一练自己坚持的能力。目前可选有：一，写日记。二，背单词。三，俯卧撑。四，跑步🏃🏃。五，CAD 暂时就想到这五件事，我觉得对个人提升最大就是，写日记吧。 其余的我也没有太大的决心和动力，俯卧撑我是每天在做，也算我一直在坚持的一个小习惯，可以一直保持，如果还有其他可以全身锻炼且简单的训练动作，我想我也会愿意尝试换一个的。 比如，平板支撑啦，或者是小马哥说他朋友推荐的打一套拳之类的，哦，对了办公室我放了跳绳，挂在显眼的地方，想跳就跳着试试。。。。。嗯嗯，扯远了。 回到坚持的，死磕的事情上面来。 今年，我真的要把，日记释放出来。
* \*\* 曾国藩坚持每天写日记，成就了一个内心强大到超然的一个牛人，在年轻时还是刚愎自用，但是一直写日记在修正自己的内心阴暗面，比如白天说了几个黄色段子，晚上开始发誓不再这么干了，比如觉得跟某个人谈那件事时，自己有过于讨好之动机偏离了事情本身，就会大骂自己轻浮，且要自己戒之。这样方成就自己思想上的超然，且真正就事论事的境界。我不敢说达到他那么自律到极致的境界，但我可以通过日记，更多的认识王波自己，至少不会一直弱下去。比如，我自己真的有点过于爱面子了，在众人面前过于装逼呀，脾气太过暴躁呀，自己真的肚量太小呀，这些性格弱点，会更及时的引起自己注意。比如，那天王海梦说我打断她讲ppt，我仔细一想好像确实有问题，而且我的动机还是有点显示个人权威，因此下决心改变这个毛病，改为记录笔记，后头提出来，而且越是对比自己地位低的人，这种习惯更彰显个人修养，提升逼格，所以我会愿意改。 这都是写日记的妙用，故而，我决定，15年要死磕的事情就是写日记这件事。刚好，有日课本，我会把日课坚持下去，我把日课拿来做日记提纲，列出今日有到感触的清单，然后拍照到印象笔记或day one，再开始写，如果写了日课，不想写日记，就只拍照，也不为难自己。确实想偷懒，那就，一句话日记嘛，总要留下点生命的印记。我爱你王波晚安15年，我来了。
