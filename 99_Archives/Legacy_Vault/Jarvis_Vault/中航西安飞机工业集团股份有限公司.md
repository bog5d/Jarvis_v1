# 中航西安飞机工业集团股份有限公司

**企业背景 ：** 
中航西安飞机工业集团股份有限公司（简称中航西飞，股票代码000768）隶属于中国航空工业集团有限公司，是我国大中型军民用飞机科研生产基地。公司A股股票于1997 年6月26日在深圳证券交易所挂牌上市，成为中国航空制造业首家上市公司。2020年12月，中航西飞完成重大资产置换，置入西安飞机工业（集团）有限责任公司、陕西飞机工业（集团）有限公司、中航天水飞机工业有限责任公司等飞机整机制造及维修资产，实现对大中型飞机整机制造资产的专业化整合。
**企业主要产品：**
中航西飞主要从事大中型飞机整机及航空零部件等航空产品的研制、批产、维修及服务，目前主要产品有大中型运输机、轰炸机、特种飞机等，同时还承担了ARJ21、C919、AG600等国内外大中型民用飞机机体部件设计、制造、配套与服务，与欧洲空中客车公司、美国波音公司以及中国商用飞机有限责任公司、中航通用飞机有限责任公司等国内外知名航空公司拥有长期而稳定的合作关系。
**企业背后的故事：**
1：大国鲲鹏-运20    2013年1月26日14时，中国自主研制的首款大运飞机“鲲鹏”腾空而起，顺利实现首飞。自此，中国真正拥有自己的大飞机，标志着中国成功跻身于世界上少数几个能自主研制大型飞机的国家之列。
2：中流砥柱-轰六飞机   西飞是我国轰炸机的研制生产基地。轰6飞机是在前苏联图－16轰炸机基础上研制的高亚音速战略轰炸机。1968年12月24日首飞成功，1969年投入批量生产，成为中国空、海军具有威慑力的武器装备。
3：蓝天勇士：中国 飞豹    "中国飞豹"是我国首次自主研发的双座双发超音速全天候歼击轰炸机。该型机填补了我国歼击轰炸机机型的空白，荣获国家科技进步特等奖。该型机航程远、威力强，有良好的中低空飞行性能，具有先进的火控系统，可实施对地、对海目标精确攻击和空中格斗。西飞充分发挥厂所结合的优势，将飞豹飞机打造成为我国具有完全自主知识产权的海空利器，是我国国防领域的一颗明星。
