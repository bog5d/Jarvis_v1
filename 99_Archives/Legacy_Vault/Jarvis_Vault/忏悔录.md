# 忏悔录

高二时，租房的楼下，有一个网吧！
因为网吧老板也是我们一栋楼的，所以也有几分熟悉。
某日，我买了一份冒菜去这个网吧上网，然后那个中老年女老板，很热情的招呼了我，问我“你也是楼上的哈？你住几楼啊？”
这样寒暄了几句，就找个位置坐下了，然后没过几分钟，我一不小心，把冒菜打倒了，冒菜的汤汤水水，就散落在键盘上了，我一看这苗头不好，于是我赶紧起身，准备在没有发现以前，闪人！！！
于是乎，去结账了，女老板一看，哎哟，前后不到十分钟，按照规律，上机就得按照半小时算，好像是1.5吧。这老板，淡然一笑，哎呀都是一栋楼的，就算了吧！
哎呀，我心里那个，难以言表的情绪，纠结了一下，虽然很是尴尬，我还是决定赶紧撤，匆忙的感谢了几句就走了。。。。。。
回到家，还是内疚了有那么几小时，觉着人家好心好意的免单，我却没敢告诉她我造成了损失！！！不过心里更多想的是，以后别碰见了。
再后来，还是在二层楼梯遇见了，她拿着一碗饭，我硬着头皮打招呼了，她嘴上满是埋怨，“怎么不说一声呢？还好发现的及时，把键盘倒了过来，让水滴干了，不然那个键盘就报废了。。。。”   我却不好意思正面面对问题，就不停的套几乎“咦，今天吃的绿豆啊，还不错哦”   她还是在一直埋怨，。。。但，到最终我也没有说对不起，我认为这件事，我做的确实不够光明磊落，而且错是一错再错！
