# 以德服人

真正要人服你管，给你权利是必须的，我应该做到不越权。以后门市日常制度属于你管辖，我绝不该再做过问。

不过我还是要提及到一点，就是真正旁人服从，更多的还是以德服人。也就是你在给员工安排或提要求时应当是合理要求，或者旁人能够理解的正当要求。
如果你要建立一套严厉的上报制度，是不是自己就应该按时上班。如果对他人严格，自己特权，那就算你有绝对权利，人也不会服管的。

我把你的权利底线守护好，你把自己该约束的约束好，这样才能真正的做到管理有序呀。
  当然如果你是为了面子，那就需要你自己好好想想了。
