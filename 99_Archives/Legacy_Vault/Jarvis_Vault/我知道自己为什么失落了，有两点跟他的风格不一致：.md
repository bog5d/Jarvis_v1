# 我知道自己为什么失落了，有两点跟他的风格不一致：

我知道自己为什么失落了，有两点跟他的风格不一致：
1. 他不希望跟他的现有员工走太近；
2. 希望聚焦，不能超出控制太多；
我以为会给人家带来价值的地方，是对方担忧或者防备的点。 有点献礼献到屁股上了的感觉，不过结果倒还好。

不过传递善意基本做到了，
坦白讲，我还是挺失落的，想被接纳多一点，或者团队组织感。  但这个团队呢，不太鼓励大家和谐相处，老大控制一切，下面老实点。 

算是一个之前的给自己的希望落空了，所以该重新安排。  我之前的希望是，很被对方赏识，甚至被重用，掌握资源。 

但是明显，对方现在对你的定位是： 一个小业务，有点搔首弄姿，过度表现了。 

反正不太爽，没有那种充满干净，我要加油干的感觉。
