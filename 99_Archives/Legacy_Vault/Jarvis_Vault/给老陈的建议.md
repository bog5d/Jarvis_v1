# 给老陈的建议

几个要点

* 招人是第一步

1. 高于行业水平的待遇，比如找老李要一个十万年薪的岗位
2. 多人竞岗，在这种情况下，必须让进来的人有竞争意识，知道被干掉是很可能的，自然会信服你，并尊重你的（不过告诉，老李决定谁去谁留的事情，绝对不允许任何人插手，保证你在新人眼中的绝对权威）。
3. 真的要招到高素质点的人，素养高的人一般都不会差太多，我认为大专毕业是最最底线了，他妈的十万年薪还不配找个大学生吗？

* 第二步，你要安排好。

1. 人招进来不能天天耍起，想方设法折腾这帮新人。安排最脏最累的活给他们干，让他们看到工作最差的一面。如果工作最差的一面都能接受了，这样的员工肯定是吃苦又忠诚的。
2. 每天新人干了什么要总结，并夸奖。那个夸奖很重要哈，做得好的，千万不要吝啬夸赞，精神认可的作用一点不亚于金钱奖励的，而且避免金钱奖励，免得一开始就太浮躁了。如果真有做得很棒的，比如刚来就促成了一个单子，奖励个iphone6，也不过分嘛。避免直接的金钱奖励。

* 第三步，让新人熟悉业务

1. 多让新人去给老业务打下手，借机把老业务手中的资源摸清。
2. 所有流程尽可能让新人都接触完，尽快上手。

* 第四步，换掉老油条。并继续推行，业务人员年轻化，最好是把老油条算换掉，留下的都是被你栽培出来的人，那时候你才有空多陪女儿去耍，上班安排好工作，然后回去陪陪女儿，就ok了。把自己身体空出来，脑袋多思考，就能支得转这摊子了。
