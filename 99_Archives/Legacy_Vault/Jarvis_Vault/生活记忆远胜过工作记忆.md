# 0721 生活记忆远胜过工作记忆

我在想啊，很多时候在回忆过去的时候，肯定很少是回忆起那段时间的工作，比如明年我肯定不会想起今天我去接待了赵正或者潘多钊，而如果提起今天晚上和王海梦一起聊了三个小时，肯定是记忆鲜明的。

* 上午办公室处理了点事情，中午赵正他们过来。转了支付宝2000元到卡上，然后去请他们吃饭，开房。
* 下午就去了保险公司，签好了什么和解协议。然后去交警队，碰到了一个黄警官，信佛的，是皈依了的。不吃肉！然后还送我两本书。不过吃素的人好像精气神是要好一些。
* 不过那个梁姐跟保险公司核对了赔付标准，确实交警队给标准有差异。交警队算出来基本上是保险公司认赔的两倍。然后就是各种去跟保险公司沟通。

我觉得还是有必要跟保险公司那边说清楚，
:现在交警队核算的标准跟你们认赔的标准误差确实太大了，然后家属这边确实不答应。而且交警队那边也声称是按照国家标准来核算的，所以这里面就有标准不统一，确实需要澄清讲明白。我觉得这样吧，这事情无非就两种可能，第一交警队搞错了，第二保险公司搞错了。只要让家属认可是保险公司搞错了，这事情就解决了，也就是需要你和家属一起去交警队说清楚，让交警队承认你们的标准，那样就轻松了。最怕的就是第二种可能，确实是保险公司这边有错误，这样家属这边可能会翻脸，到时候就麻烦了。我明天要回成都，我怕我不能全程参与，如果可以的话，你直接和家属一起去交警队，让交警队那边承认是他们的计算有偏差，我把他们认定书的照片发给你嘛，还有交警的联系方式一并给你。这事情确实搞复杂了，都不容易，我们以解决问题为前提嘛。
