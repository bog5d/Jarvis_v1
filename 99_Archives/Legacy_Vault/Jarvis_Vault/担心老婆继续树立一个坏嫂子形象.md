# 担心老婆继续树立一个坏嫂子形象

梦梦下去了有2天了， 而老高一直没有跟她们见过面，吃饭也不一起。一种回避的姿态出现的，
好吧，我知道她是不想跟她们一起吃饭一起耍。一则是不去见到梦梦他们两口子在那儿恩爱，自己不太适应，而则是不想看到冷瑞继续在那儿谄媚流哈喇子。
青春懵懂期的冷瑞，遇上梦梦那种体贴入微的微信关心，会让他荷尔蒙陡增，所以一心想要对梦梦示好，才会出现为了给梦梦买qq糖而不顾老高的感受。  想想是挺让人生气的，不过那个年龄的他希望他能多被认可并不可耻，也算是可以原谅的。。。。不过在老高这种精神洁癖的人心中就是罪不可恕了。。。这几天表现出不想看到冷瑞的行为，就比较正常了。  
我只是怕，老高一直这样回避，离团的做法会让梦梦和王斌觉得老高不欢迎梦梦，再加上以前霞霞和老高的不和谐，更容易让别人以为老高就是一个和其他女人合不来的，难以相处的女人。 这样心里还是挺堵的，我不希望老高是这样的性格，那样确实不太好。  不过，如果我过于要求老高去做乖面子，和梦梦笑脸相迎，那就不是老高了。
因此，我确实没法要求老高去热情对待梦梦。但是，如果梦梦和王斌觉得老高故意回避他们，我又会觉得老高处得太差了，心中不爽，更可能说是，在梦梦面前没面子。。。。。。。。啊，啊，啊，那就是我太看重梦梦的感受，换句话说就是把梦梦的感受放在了老高的感受之前了，那从内心深处就是我心态有问题呀。
我真正该做的是，尽可能让王斌和梦梦觉得舒适，然后让老高感受到我的爱。
