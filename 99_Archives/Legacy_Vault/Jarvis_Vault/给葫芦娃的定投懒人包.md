# 给葫芦娃的定投懒人包

给葫芦娃的懒人包：

葫芦娃，我打算推着你，把基金定投的习惯给建立起来。 我坚信这是件有意义的事情，😄。
之前是把保险配置给建起来了，（图中左上角）这次我们把养老的配置安排安排。（左下角）  之后可以再试着，把短期理财目标给建立起来。  （右边这两块儿）![unknown_filename.1.jpeg](./_resources/给葫芦娃的定投懒人包.resources/unknown_filename.1.jpeg)

好吧，先说养老理财。

第一步，你先测算下，在现有每月收入基础上，你掉了多少钱，不会很心疼。 
 这一步是为了帮你建立持有30年以上的理财心态，只有在你看来这是一笔不重要的小钱，这笔钱才能担风险，也才有机会获取超高收益。 
假如你算出来是200元，就是说每月丢200，心态不痛不痒的。 

第二步，开始定投基金。   
定投基金，你需要解决两个问题： 

A  买哪只基金？
B  用什么买？

买它
![unknown_filename.4.jpeg](./_resources/给葫芦娃的定投懒人包.resources/unknown_filename.4.jpeg)

你从里面随便选，看上哪只选哪只。 这已经是筛选后的靠谱基金了。   
大星姐和魏芳 选的应该是上投摩根，1年已经有30%的收益了。    
建议，在混合类里面选一只，权益类里面选一只。 把基金代码记下来。  

用什么工具买呢？  就直接用支付宝！
打开支付宝，搜基金，进入基金界面

![unknown_filename.png](./_resources/给葫芦娃的定投懒人包.resources/unknown_filename.png)

![unknown_filename.2.png](./_resources/给葫芦娃的定投懒人包.resources/unknown_filename.2.png)
