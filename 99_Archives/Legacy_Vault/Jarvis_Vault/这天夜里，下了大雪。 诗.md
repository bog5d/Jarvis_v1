# "这天夜里，下了大雪。 诗

诗
[1FEAA1BF-C98B-4592-A282-34C5D42AA745.bin](./_resources/_这天夜里，下了大雪。_诗.resources/1FEAA1BF-C98B-4592-A282-34C5D42AA745.bin)

"这天夜里，下了大雪。后半夜雪停了，皎洁的月光照亮了德丹吉殿。仓央嘉措披衣来到窗前，看到白雪覆盖下的拉萨城，在月光的映照下洁净得如同人间仙境。他一夜未眠。昔日热恋的情人，成了别人的新娘。相思折磨我哀愁的心，让我容颜枯瘦。尘世的缘分就这样断了吗？也许，天注定我成不了一个让姑娘幸福的情郎，还是做个佛爷更好。经师们欣喜地发现，佛爷在上课的时候用功了许多，不再神游物外，总是专心听讲。佛爷本就聪慧，一努力学问轻易便高于常人。“佛爷到底是佛爷，收敛了心性，佛性便自然流露了。”老格西江巴扎巴禁不住赞美道，众位格西纷纷称是。"，选自："仓央嘉措诗传全集"，作者：闫晗
