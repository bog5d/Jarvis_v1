# 坡道：

坡道：
就是开场，把对方拉过来。  来自为什么要听这件事儿。
最常规的开场： 我在人力资源做过多久多久，我们分为五个部分，花2小时来分享，之后有个提问环节，想上厕所的出门左转。 

最低目标是不要成为这种开场。 

\*\* 大家都在人力资源工作了十几年，我们都被一个问题所困扰，究竟怎么样有效激励我们的员工。？

我们每个人都离不开感情婚姻，也看到甚至经历过关系破裂，在感情里的人会困惑，人为什么要分手，分手后的人又困惑，人为什么要结婚。 
所以，今天波哥就这个问题跟大家分享分享。  下午四点半，不见不散。

这是一本有答案，但没方法的书。 知道病因，没有药的书。 

\*\* 用“你”：你们都是即将毕业的大学生，现在可能每个人都面临忐忑的心情，到底能不能找到诚心如意的工作，会不会被用人单位给骗了\*\*

\*\* 运用强大而有震撼力的数字： 你知道我们开会这一个小时，又新增了多少移动互联网用户吗？

\*\* 提出问题： 你们当中有多少人用了人生近一半的时间在开会，而且效率还不高？ 如果我们能节省效率的话，拿2个小时一起学习一下，大家觉得值得吗？

\*\* 用观众震撼的方法： 我们正面临一次前所未有的挑战，

\*\* 用坦白的方式： 说实话，我很忐忑，今天在场的都是身经百战的专家。 研究得越多，越不敢讲，原因在于在座的各位资历深厚

\*\* 讲故事： 今天出门之前，我女儿问我。。。。

\*\* 用想象： 想象一下十年后，我们的生活会是什么样？如果我们这样安安稳稳地工作十年，你会成为一个富人吗？你能承担起家庭的责任吗？ 你能买得起背景60平米的房子吗？ 怎么才能获得这些呢，我们推荐的方法是。。。

\*\* 提问+想象： 既然你们是他们的军队，为什么要跑？
跑了，活；留下，死；
很多年以后，你愿不愿意把你的时间，从这儿到死的时间，拿回来换一个机会，回到这里告诉我们的敌人，他们可以夺走我们的生命，但不能夺走我们的自由。 

\*\* 常规坡道： 今天天气不错，秋高气爽。各位宾朋欢聚一堂，我们都很开心。 这样的坡道，跟听众，一点关系都没有。 跟听众要有关系

甜点：
\*\* 常见甜点： 提问。  经常在没有人问的尴尬氛围中结束。 提问会冲淡中心。 

想让听众感受到什么，才是甜点的左右。 

\*\*  强调那种急迫性： 如果我们今年不做，明年也不会做。

\*\* 一则小事故：语录，奥巴马说，我现在终于可以回去告诉我女儿，我们美国终将。。。。
