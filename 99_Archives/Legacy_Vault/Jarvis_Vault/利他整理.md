# 利他整理

1. 给供应商介绍核心企业资源
2. 我们有银行资源，如果他要办理一些相关业务可以给予一定的帮助
3. 如果他有公司股份，我们可以给予一些股权上交税等等的一些建议
4. 如果对方要炒股，我们认识一些证券的首席分析师，知道一些军工方面比较准确的股票信息
5. 帮助市场人员贴到票后收到钱，闭环订单，完成任务
6. 任何一家公司都希望需要收的钱收回来，需要给出去的钱少给出去，能开票就开票，所以比较大的上市公司，如果他有下游供应商，那么我们可以帮其将他开出去的票贴掉，降低他的现金流出
7. 组局，让供应商同行之间的财务能多交流多认识
