# 乔布斯

1. 问责制度DRI 每一个行动都有负责人 ，我应该把安排好的事情设定一个负责人，没完成则直接追究责任人。
2. 一成不变的做少数事情，这是专注的力量，乔布斯一直专注于某一件事情。我应该学习这点:我需要专注于完成一个创新性销售模式--直通车。
3. TOP一百人，只管理少数人来管理全局。
4. 你发现有些事情做得不对，“以后处理”，那是别的公司。不去忽视问题！
5. 让每个人描述享受过的最美好的服务，而形成”天才吧”，
