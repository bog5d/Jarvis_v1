# 1229 不去打断别人

我也爱打断别人，是个很讨厌的毛病。 王海梦他们在台上讲ppt，我觉得有一点做得不好，我立马就说“这里面有问题，我来给大家讲清楚。。。。。。。。”，然后各种去显示显摆自己的博学。真的有点逗整，
当然，我自己也有一个合理的动机，我打算在讲课的时候，把问题给纠正了，这样大家都学到东西了。
但问题是，这样也许会收获到纠正问题的作用，却失去了发言者的开心，发言者不开心了，你纠正的问题她也不一定会认真听。所以纠正问题，普及知识的作用会大打折扣。 所以，就算我的动机是想纠正普及，打断也不是最佳途径。 
更何况，我的动机里面还参杂了些显示个人权威，显摆个人博学的成分，故而，打断别人这件事情，就更显得可耻了。  这种做法，不仅不会得到别人真正的尊重，也会让自己不清醒不淡定，尤其是在多人的会议上，容易失言。

*    如果我换一种方式呢，比如，我在听别人说的时候，发现问题了赶紧记录下
* 然后再等人讲完了，我统一一起去说，那个我有几个问题要纠正一下哈，第一是。。。。。。。  
* 这样的结果发言人会觉得好受尊重，因为我做了笔记的，同时也让其他的听众觉得王波还是有个人修养的人。于是乎，波哥的逼格就提上去了，
* 波哥为了提升个人逼格还是要，尊重发言者呀，尤其是比自己身份低的发言者，能做到不打断做笔记，后提出的话，会更有逼格范儿的，波哥，慎之啊慎之。

 别人讲话，有疑问记下来，再提出，不打断别人，不打断别人，不打断别人。

突然想到 如果不及时打断别人，也给够了自己足够的时间去思考，也许自己发言时会更有质量，呵呵。波哥加油，尊重别人，提升逼格。
