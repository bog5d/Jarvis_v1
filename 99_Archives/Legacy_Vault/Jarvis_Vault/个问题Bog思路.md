# 15个问题Bog思路

## 一：客户数/客单价/销售额/利润

### 1.1 我们项目的客户数是多少？

    两方面：
    	* 1. 大b端，如研究所，十大航空集团，加起来超过一千家。
    	* 2. 模块级供应商企业一万多家。 
    	但这个数据，我们还是缺少调研数据的。
    

### 1.2 我们的客单价。

* 如果是供金，针对供应商向的，应该是融资额1千万/家；客户上万家，也就是一千亿，然后分润做到1%，有10亿规模营收。

* 如果是SaaS，针对核心企业的，一年初装费是120万，维护费10万。 一千家的话，也有1个亿的收益，和每年一千万的营收。

### 1.3 我们的销售额 。

* 这问题需要目前来回复吗？ 还是一个预测值啊？
* 如果在天使轮阶段的话，是不是可以估算第一阶段吧，600家供应商，每家1000万，60亿的1%，（但是我们在Bp里面的测算时1.35亿）
