# 耍了游戏好后悔

老婆下大竹了，我现在耍了游戏，写日记没什么精神，
今天去了王平，那边。告诉王平，我是力挺姚姐的心态。还是聊的挺和谐的
早上心情很是郁闷，我觉得这个月出货太少，老马也不高兴。我也觉得他在指责我，加上早上的时候，吃早饭没有钱，让自己一大早就有点自尊心受损！！
心里那个难受呀？
对了，今天，上午吃了早饭后，去转了小马哥卡上的7000元. 其中三千用于购买公司的赠品，刀具，4000是本打算还老高的呢

- [ ] 欠小马哥1000元  修手机
- [ ] 欠老高4000元  小马哥本计划还老高呢
已经火速买了网上的12HC2了，哈哈哈，三台，拿回来走流程还是比较好的。1799收货，要赚一点哟！对了，明早还是要把张政的五十元还了，在艰苦的岁月也是要自己撑过去，不然我这样节约的意义就不存在了。
本想每月强制储蓄，却因为存了钱，没钱过平日的生活，所以让周边朋友吃亏多买单，那就失去了节约的本意了，我还是没能学会有规律储蓄的正常生活，那就没起到锻炼自己的作用！

对于门市零售，我真的是有点没辙了。但是，我确定一点的就是，真的是去现场摆展，有点影响门市正常运营。
现在看到，王平门市的零售经营状况，我觉得应该多推蒸汽洗等高端产品了。零售动作有4

1. 户外广告执行出来，蒸汽洗送空调
2. 门市考勤制度严格执行
3. 销售员提成制度改为高端机多提，低端机少提或者不提。
4. 尝试加入砍价会团队圈子，我送赠品饭煲嘛
5. 下去门市聚餐一下了，再不聚就都淡了。5555555
