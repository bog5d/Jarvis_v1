# 10月梦想清单

- [ ] 内心赞美 日行三赞。形成习惯。

* 因为自己已经陷入了邀功索赞的心理误区了，老是伤害比我弱势点的人，而比我优秀之人却会被我排斥开，让我欣赏优秀的能力。

- [ ] 发扬开会艺术。

* 开解决问题的会议。开会前把要解决的问题想好。作为领导要做的事情，并不是把所有问题的解决方法想好，而是把问题问对。
* 会前先提出问题，会上要答案。有篇日记，向三星学习开会。可以拿来参考 （凡是开会必有记录，凡是开会必有跟踪）
* https://app.yinxiang.com/shard/s3/sh/a377a240-0a31-4e65-b088-056e7e85aaee/6a7fb991e2b44291ef82a4e0a4f0d4d3

- [ ] 研究拍照技巧。

* 因为老婆会因为拍照好不好看，极大的影响到心情。所以，如果拍照技能到位些，把老婆拍得美美的，必定会更促进感情和谐，我也少点很多烦恼。更重要的是，遇到想泡一泡的妹子时，此技能发挥重要作用哦。
* 就只需要学习3条技能试试。

- [ ] 做好吃的素早餐。

* 因为我吃素嘛，为了让老婆不成为我吃素的阻力，我就多做好吃的素菜吃，加上她希望我陪她吃早餐，那就做出美味的素食早餐肯定把她幸福到，而且也会增加吃素动力。
* 其实不难的，重心在烹饪蛋类呀既美味，又好看。还有就是粥类，简单又保健，祛湿的啦，养颜的啦，补气的啦，都可以通过熬粥实现，蛮好的哟。

- [ ] 学习冥想，实现心静。

* 很多伟人都是喜欢禅修的，有禅修习惯的人，更容易实现很多价值。比如乔布斯，特斯拉。追求心灵平静的人，在禅修状态下，自己想解决的问题就说不定就悟透了。也许坐禅真的要专门学。
* 不过可以把书上的教程拿来试试，不断提出不能坚持下去的问题，再想办法。

- [ ] 把看过的书拿来复习。

* 很多好书，真的可以重读，而不是买一堆书，都不读完。比如
* 稻盛和夫的 活法，乔布斯传，柴静的 看见，52种思维错误啦，这些书都是值得重读的。
* 我觉得每本书得这一个读后感，只要是真实的感觉，不管体裁，写出来就是。日后再来看自己的读后感，也能知道自己精神层面成长的步伐。

- [ ] 看字幕学英语。

* 我想试试用看美剧，学英语的方式。能够达到不要字幕依然看得很自如的状态。
* 我觉得反复看是关键。有字幕看一遍，纯英字幕看一遍，无字幕再看一遍。至少看三遍，应该就能进步很多了。
* 还有那些简单的英语句子，每天背一点耍。比如what are you trying to say?之类的简单易懂。

- [ ] 我想养成午睡习惯呀。
- [ ] 给老婆一场教堂婚礼

* 做银卡 邀挚友来祝福。
