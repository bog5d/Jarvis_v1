# 不到4000元游泰国，7天5晚，住泳池酒店，附普吉岛、芭东卡塔中文地图、行程费用和海淘返利

不到4000元游泰国，7天5晚，住泳池酒店，附普吉岛、芭东卡塔中文地图、行程费用和海淘返利( http://www.douban.com/group/topic/46584696/ ) 分享自@豆瓣小组 Android 客户端

       这是说走就走的旅行，查好攻略，酒店和机票后，匆忙上路。两人香港直飞普吉岛往返含税2400，2人7天5晚，人均3900，住的酒店都有游泳池（当时不知道有海淘返利，很后悔，感觉亏了好多钱），豪华，干净，旅行"泰"happy。头回出国，自由行，提前6天订机票酒店。

准备篇：（需要信用卡和护照）
签证：本来想自己去办泰国领事馆办签证，但自己办需要机票，担心万一签证不成功，机票白买了，考虑再三还是在淘宝找的代理办，因时间原因，找的广州淘宝代理
酒店：在booking.com先订PP岛1晚的酒店，再定芭东沙滩4晚的酒店，特意挑选有游泳池的酒店，芭东的这个还有健身房，PP岛的酒店靠通赛湾背面的海，很方便，因不想搬来搬去行李，自由行要轻松，订的都是同一家酒店，酒店名字见行程费用表里。
建议1：酒店省了不少钱，是靠穷游的朋友们推荐的，非常不错，感谢他们。
大家可参考穷游网省钱攻略和海淘返利  http://bbs.qyer.com/thread-869194-1.html
保险：在淘宝网订的美亚7天保险，买个保险心里舒服些，飞机晚点，医疗啊，丢失护照，财物等对应都有赔偿
行李：     鞋子-因为不想带太多行李，鞋子只穿了一双洞洞凉鞋，事实证明，洞洞鞋超实用，一双鞋搞定所有，以前有穿人字拖洗澡时滑倒摔伤的经历，这次特意挑防滑的，凉鞋在离岛游的时候不容易掉水里），便宜又好穿。       男：T恤2件，五分裤2条，长裤1条，薄风衣外套1件，事实证明，飞机上很冷，应该带棉外套，回程时很冷，我老公在半睡半醒中冻的发抖，太困了，也没意识到找空姐调高点温度，冻到肚子不舒服。
    女：裙子3条（花花裙子2条，蓝色裙子一条，拍完发现蓝色裙子效果最好，很仙，看看后面的图)，短T恤1件，长袖1件（没穿上），五分裤1条，长裤1条，棉外套1件。再加情侣装T恤1套（也很满意这次的霍利斯特情侣T，穿上很有度假风）。
      普吉11月很热，白天晚上短裤短袖没问题，在房间里有时得开空调，外套只在飞机场和飞机上派上了用场。
      薄传单-1件（在国内旅行也有带薄床单的习惯，折起来和件薄外套差不多，不占多少地方）
      相机防水套，纸巾等，创口贴，蚊子怕怕膏，眼镜等，机票、酒店订单纸版（一式二分，分开放），护照原件，扫描版发邮箱，手机，平板电脑等。
    总共才1个行李箱，1个背包，行李箱9kg，免费托运。
手机APP：穷游锦囊，旅行翻译官（超级好用，输入中文直接转化成英文或泰语，播放给对方听，沟通方便很多了），如果要还价就用计算器或手机带的计算器也行。
为什么普吉岛决定自由行，因为看到攻略上说泰国人民很友好，而且普吉岛治安很好，所以很放心，如果是菲律宾，印尼海岛啥的，可能得考虑考虑。
行程篇：
香港机场：
越途网跟在线客服选座位，4E和4F，前三排都没人。座位可以调斜度，可以斜躺，比较舒服，全程基本都在睡，飞机是空客320.飞机的水啊，食物都是付费的。

建议2：选座位尽量靠前，尽早下飞机，早出来可以少排些队，早点到酒店，出入境单也可以在入境大厅填写，有桌子，先排上队

我们出机场时碰到4个四川的旅友，早上在便利店又碰到了，约好一起吃无法，事实证明，人多吃饭，可以多选点类型海鲜。（现在回头看：如果在泰国吃的地方找海淘返利可能也会有惊喜发现，不过短时不准备去，所以没招海淘返利了。）
