# 1109 梦想何存

最近都没有写日记，因为每天下午回来，都觉得很累很想睡，总觉自己很疲惫。痛恨自己，每天下午的时间利用得特差。
我下午本来就没怎么上班做事，回来就在电脑前颓废到晚上。现在的我呢，对游戏已经没有那么痴迷了，不过依然在电脑手机上消耗时间太多。太他妈多了。

我现在特别想，把自己的时间价值最大化。自己太腐朽了，所有的成就感都是以前的老本，比如我特别喜欢对人讲刚进美的如何干掉李永康啦，如何开发开江乡镇啦，而且很大一部分还是吹牛了的，对其他非工作人际，就摆自己大学时干了好多种生意啦，参加了多少活动啦，我为什么觉得那些事很有成就感，那是因为做那些事之前，我是焦虑担心害怕了很久的，然后还是去做了，因此回忆起来会很美好。而现今的我，已经很久没有感受到，要做一件事情，让自己紧张得彻夜难眠的感觉了，这说明很久没有做高于自己现在能力的动作了，也就是i+1，做得太少了。
必然能得出结论，进步也是太少太少了。
王波想要进步，但，感觉就像只想不做的屌丝男一样，光是想象就让自己满足得高潮了，结果什么事情都没有做。完全是开启的等死模式（有篇文章叫 请关闭你的等死模式，谈到了穿越模式和等死模式，就算反正都会失败，付出时间成本来努力比付出时间成本来焦虑划得来），
还有种心态，怕失败。只要没有目标，就永远不会失败，所以干脆没追求。
唉，无奈的波哥。

我接下来，有两个比较重大的人生目标
一，考取雅思
二，改造房车
除此之外就是生活计划

1. 投资3000到p2p
2. 注射乙肝疫苗
3. 看英文喜剧片10部，去掉字幕学，反复看10遍。

是不是应该写到梦想清单里面哦

先说雅思考试的事情，

1. 王波，你从骨子里面是害怕这件事情的，我看得出来，把雅思考试改成英语挑战计划，听起来没那么有障碍感，说起考雅思还是有点那种要去参加海选话剧的感觉，紧张怕失败。那时的王波，为何那么豁达勇敢，还是正面去做？
2. 哦，想起来了，那时的王波告诉自己，如果不去做，日后自己会看不起自己的，自己一定后悔。但如果做了，没成功，至少不后悔，而且还能附带收获很多，这样子的王波才是那个犀利的王波。那雅思考试，会不会像MBA计划那样搁置呢？我觉得为了MBA看了点书，至少还是会了点系统思维，我觉得很值得的，也就是说哪怕是脑壳发热，干了一件事情，那肯定也会有收获，因为你干了。
3. 雅思考试，对我来讲应该不是特别艰难，无非就是听力嘛，我把听力速度整快点，然后再去适应。
4. 让后口语嘛，我每天背三句口语。加上用软件英语流利说，肯定也会比现在的水平进步很多的。
5. 还有，无非就是单词嘛。那时候我为了想背单词，每天还写了小纸条来背诵单词，后期还是没能坚持下来，是不是这种方法不够简单易执行呢？我觉得背诵单词的方式还需要改进，让乐趣更多一些。快乐学习，方可持续。我觉得我的学习方式不够快乐，所以应该改进。这里面我就增加了一件任务，寻找更快乐的记单词方法。
6. 最后就是写作，关于写作有什么学习方法没有？？目前也没有，我能记起来的就是多用语气连接词。然后就是背诵了。唉，写作还是要找新办法才行。
7. 王波英语挑战计划，不说了三个月拿下。就是三个月，每天2个小时学习。早上只要起得早，每周学习5天嘛，330个小时的学习，用于记单词和练听力那些时间足矣。可以借鉴别人的学习方法，听听力25分钟，记单词20个，看无字幕英语电影一遍。联系说20分钟。就这样，然后到时再细化到每天几道题，说几句话之类的，我就不行拿不下雅思。王波加油↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗↖(^ω^)↗。
