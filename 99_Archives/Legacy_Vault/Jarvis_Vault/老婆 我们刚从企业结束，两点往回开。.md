# 老婆 我们刚从企业结束，两点往回开。

老婆 我们刚从企业结束，两点往回开。
现在心里非常后悔跟你吵架，我昨晚跟你吵时，真不明白你到底要什么，当时的理解就是你不开心，不满意当下，也很不满意我，然后我就感受到一堆指责和否定；
但今天昏沉沉地拜访企业时，脑子里全是你，很担心你。 不该跟你吵架的，你需要的其实是我的关心和关注，只是你不太会表达需求，用指责我的方式来表达需求；
我昨天就停留在你指责我不求上进，不承担责任这方面了； 也觉得被误解和委屈，自己承受的内心压力我没说，但不是没有得，我的想法是咬着牙承受着一些误解，把事情做成；

我从没想过这辈子跟除你以外的人过，我非常感激这个阶段你能陪我度过，只是昨晚吵架这些感激和爱，真的表达不出来（写到这里时 我脑子里又浮现出你要喷我的情形，可能过几天你会说“文笔越来越撇，久了不看书就这样”）

我觉得当下我们的经济情况确实很艰难，也在触底反弹了，所谓贫贱夫妻百事哀，应该就是当下我们的状态，我们一步步熬过来就是。

下次回去认真地求助一下爸妈，可否借一部分钱先把高利息的负债还一部分，这样每月发工资时，钱能变成我们可控； 反正我出差也可以报销了，负债不会再扩大； 这样生活就有了基本可控；

到时候房租我每月承担2000，你也承担一千五，在上海的基本盘就有了；

如果期间我们新基金组建成功，有管理费收入，我相信会覆盖我们娃儿的生活成本的, 我家里目前没有支持能力，所以我很多求助也不敢提起，第一娃儿，的确会多仰仗老爸老妈

唉，不说了，扯远了，爱你老婆； 两口子吵吵架，没这么的，别想得太大了，我不会想跟你分开，你也别老想着离婚，这就是生活磨合，咱们只是猛烈了点，毕竟日的时候也很猛烈嘛
