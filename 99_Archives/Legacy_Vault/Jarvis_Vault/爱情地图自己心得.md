# 爱情地图自己心得

关于爱情地图，有几个问题需要问自己。

* 生活中让我感觉特别自豪的事情。

1. 初中时一个人独立生活，成绩还不错。
2. 高中时，看了卡耐基，学会倾听，在和人相处时颇受欢迎，比如一个叫吕鸥的小学同学，和我聊天后觉得不知为何，就什么都想对我说。我心里暗喜庆幸，我愿意倾听的性格，似乎任何人都会持欢迎之态。
3. 敢于大方地爱妈妈，尤其是老爸走后。比如，给妈妈剥虾，比如在电话中亲妈妈。
4. 大学时，做生意。 尤其是创建皮皮服务队，只是为了找一个伙伴，无意中创建了一支队伍。(遗憾在于没有后期经营好这支队伍)
5. 戏剧社，出演鲁贵，演技受认可。 同时和社团关系处得很好，大爱那段时光。
6. 上班后，并没有为了讨好客户，让自己去学习抽烟，喝酒，打牌，甚至找小姐。 工作后依然保持自己的良好作风，对自己这点很满意。
7. 开大竹门市，一段艰难的岁月，还是做起来了。
8. 追求思宇，自己追求的态度很端正。抱着就算会失败，依然愿意去争取的心态。 这样得来的幸福，会觉得踏实。
9. 养成了写日记的习惯。这是目前我坚持得最久，也是最有价值的一个生活习惯，让我的思想成长，有了可持续性。

* 这些成功/自豪如何塑造了你的生活？

1. 关于日记，我觉得日记给了我定期反省自我的机会。每一天的日记，都会让我反思一下今天有没有做得不好的地方。 还有，为了晚上的日记，有内容，白天遇到一些小心得，会刻意留意一下，以方便我在晚上纪录，这样让我对生活有更多观察，或者说是让我更注意观察生活。 同时，也让我表达事情更清晰点了，有些心得和观点比较模糊，我发现自己在日记中无法，表达清楚。就开始有一些比喻和意象，来辅助表达，慢慢发现自己生活中表达观点，也有这种倾向。
2. 戏剧社，大学做生意，以及大竹开门市。这三件事上的成功，让我不惧失败。这位做这三件事之前，我都是各种担惊受怕，焦虑不堪。 从这些事中，我学到两点，一，要缓解焦虑，就去前线，做事情。二，把事情分解成能完成的小目标，去盯住小目标，这远比担心失败而没去做实事好。 这些心法对我的工作帮助还是很大，比如一般不拒绝领导下的任务，先去做。
3. 关于爱妈妈，这种习惯，怎么说呢，给我带来的改变是，我更加懂得感受爱了。 我对别人的母亲，也是充满强烈的敬爱的，只要能看到其他人母亲关爱自己女儿，我自己内心也容易变得柔和。   这也让我，意识到妈妈一直对我的无限信任，对我人生中的帮助是多么宝贵。     此外，我爱妈妈是我知道她一直爱我的，爱得之深切，因此我自己从没觉得自己缺少爱，进而也让我对重要之人不吝付爱。
4. 关于生活作风，因为没有去沾染牌酒色，一直以一个内心高雅的心态自居，虽然有点装逼，但我就觉得跟那些沾染牌酒色的人不是一路的。 也因为不打牌，我比同龄要打牌的人，多出了好多自由支配时间。 这让我玩了很多喜欢的游戏，也让我尝试把多余时间，用来看看书，打打坐，跟自己对对话，尝试一下提升。  无形之中，多做了不少有价值的尝试，比如现在看书逐渐成了一种消遣了，久不看书心里慌。     生活作风不腐败，直接效用我认为是时间的价值变高了，同样的时间用在玩牌喝酒找小姐上vs 看看书，打打坐，发发呆， 带来的提升是完全不一样的。
5. 关于追求思宇，直接感觉就是幸福真的是可以争取的，也是可以经营好的。 追求前，我确定自己有这想法，就没有藏着掖着，追求中，也是有什么顾虑就直接表达。 比如性不太自信，比如换职业可能会带来收入下滑，比如去重庆她愿意么，我以为很难被接受的顾虑，思宇都很体谅且支持我，这让我觉得人生要幸福也还是不难。  总的来说，追求思宇让我更热爱生活了。  而且，我以前很多天马行空的生活点子，我也想一一地和思宇去尝试，对未来有信心。

* 哪个角色让我觉得自豪？
