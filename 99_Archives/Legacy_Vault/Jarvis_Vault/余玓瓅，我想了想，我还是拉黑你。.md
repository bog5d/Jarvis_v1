# 余玓瓅，我想了想，我还是拉黑你。

余玓瓅，我想了想，我还是拉黑你。 
这两天想跟你的可能性得有点过多了，如果只是约个p，倒还好。 关键，我他妈也有点走心了。 
但如果真要走心，奔着一起过的心态，面临的现实太多了。
第一距离，第二你我要放下当下的感情和选项，第三你家条件其实蛮好的，我虽然也不怂，但也不想去过多磨合这事儿。 
关键是，就算把一二三都处理了，其实我们都付出过多，但感情本身就是一件玩概率的事儿。

我们会幸福生活的概率其实也是五五开，为了这五五开概率，我们这种情场老鸟谁都不愿意这样赌的。

我现在因为你带来的诱惑，  对很多其他选项都不认真对待；  你其实，也也有同样的情况。  我感觉这样下去，对咱俩的人生都蛮不负责的。 

我想了想，还是把你拉黑。 
看缘分，如果未来你是单身，我也单身，恰好在某天不经意遇见了，咱们再搞起来。
