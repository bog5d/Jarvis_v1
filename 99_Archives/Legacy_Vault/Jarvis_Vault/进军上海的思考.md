# 进军上海的思考

老婆，关于去上海这事情，我还是想了很多。
去一个大城市，获得更开放的成长市场，遇到更加优秀的圈子，是对我最大的诱惑。
我始终还是认为，一个人的幸福感来源还是，自己进步带来的成就感。 目前的我，进步确实太慢了！
如果真要去上海的话，我目前的障碍无非就这几个:

* 我作为我们两个生活的经济来源，我的收入是担不起风险的。说直白点哈，从毕业到现在，，你的经济实质上没有独立起来，如果去上海，两个的收入都不稳定，加上大城市的高生活成本，会不会成为我们的门槛呢？ 当然，你学的技能，如果能在上海立足生存，那我裸身过来也是不会有问题的，我相信养活自己是ok的，所以进军上海的前提就是你需要在经济上真正独立，你有不有心理准备呀？
* 选择成本太高了，一个决定的贵贱不在于做了一个决定获得了什么，而是在于做一个决定时放弃了什么。 去上海的话，我放弃的成本大不大。 我享受现状的点主要是

1. 现在的圈子，小马哥我觉得他是能让我在品德涵养方面有提高的人，而且也有自己能把控的下属团队。让我既能享受进步，也能找到当领导的成就感。
2. 名声，享受了一个 “年轻有为的老板”  的社会名声。
3. 稳定的收入，和福利。公司的平均收入在七千左右，加上提供了较好的住房，和偶尔性的外水。如果你经济独立后，我的经济实力会大幅提升，习惯好的话能一年能存够比过去三年还多的储蓄。假如你不需要我出钱过生活的话，就算在还贷款，我也能保持月存3200到我们的共同储蓄卡上，年盈余4万。
4. 门市租金的大幅度降低，会带来未来生意投资的良性趋势。我们现在付了五年房租，每年节省了近6万的房租，经营压力远低于以往，门市给我带来的收益能保持在年纯赚 5万左右。也就是说，光我的收入能在明年底实现9万。 当然你还能结余一万的话，我们两口子能剩下10万左右。 这一点一点都没有夸张的。你可以细想一下，今年上半年，你的月收入大概在1500，我都能明显感觉到生活压力好轻松，我们只持续了3个月存款，就有一万。而且还出去旅游呀那些大把花钱，更不用说你买衣网购那些，再加上后来的你注册公司咯，我撞人咯，你去北京咯。。。等等一些列的大项开销，都是建立在，我和你稳定收入了一段时间的基础上的。（当然你和我都没有很好的节省，也会大大削弱我们的夫妻盈利能力）。 所以目前的收入状态是我还蛮舍不得放弃的，何况这也是能支撑你去折腾的基础。 这四点是我舍不得放弃的。

* 此外，因为做这个生意，欠下了一大笔债务，如果这些债务没能偿还个大部分的话，就贸然离开现有经营项目，既会成为进军上海的沉重经济包袱，也会让借钱给我的人觉得我不靠谱。日后再有需要人家支持的时候，就很难了。所以，非要转型的话，第一个要解决的障碍就是生意债务。

![Screenshots_2014-10-09-23-50-13.png](./_resources/进军上海的思考.resources/Screenshots_2014-10-09-23-50-13.png)

以上几点就是，我现在对于去上海的一些顾虑。说简单点就是:我要还钱，你要挣钱才能解掉枷锁，做自由的小两口。

当然，事情也不能总是看到艰难的一面，在达州能年盈余9万的层次，同层次的话，在上海应该是20万。 而且因为大城市便利的学习机会，可能成长得会更快，也许耕耘2年后，年盈余30万是很有可能很有可能，只要我们不在上海做买房投入，应该也不会过得很差。
而且，老婆大人和我都学会了，节省点过生活的话，那幸福指数肯定是比现在好很多很多的。 
达到这种效果的概率绝对高于70％，所以从投资角度来看，进军上海，是很值得的一个决策。
只有一个绕不开的问题，就是，偿还债务。妈妈和哥哥的可以暂缓，小马哥，四姑孃，陈建的必须得还了，这部分大概是五万。而岳父的一万，我会今年还的。 因此，我们要去上海的前提还需要额外至少五万，才能扫清障碍。
但是，我的收入，五分之三要用于你现在的开销， 而且还需还贷款，所以今年内要还那五万是够呛的。
我认为，最合理的是，如果你想去上海，明年先去，当你在上海实现了基本经济独立了，我就能存点钱起来了，逐步偿还债务。到了，下半年，你发现上海确实是你愿意呆下去的城市，那我就在年底分红后，毫不犹疑跟着来上海，这样我们两个中有一个人收入是稳定的，又能天天和你吃早饭，还可以创造年入30万的梦想。 中间过渡期，大概一年，我至少来上海四次，即每季度来一次，你也不会太寂寞的。
那样我们的感情，需要经受距离考验，不过我还ok，我相信我一直会爱你依恋你，就怕你在社会上成长起来后，对我的依赖减少，进而感情变淡。这倒是成了最大的风险——感情风险。     当然，只要你想去，我不怕担风险的。
