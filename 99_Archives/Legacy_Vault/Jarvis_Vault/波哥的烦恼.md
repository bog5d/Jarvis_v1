# 波哥的烦恼

各位小伙伴，今天11月22日，波哥到办公室竟发现空无一人，要休假也在群里面告知一声嘛？波哥孤独的在办公室写下这篇心情。 好啦 说工作。
近期呢，事情比较单调，我们部门的安排也算不上充实。忙活得多一点，就只有大育，报账和文案类工作还是基本仰仗育哥在处理，而梦梦同学呢，在想方设法找到存在感与价值感，思思呢，一直在王斌身边陪伴（梦梦的眼神中带着醋意的怒火，已经被我看见了）。基本上就是这样。虽然都在忙，但是也都是没有大方向的。 波哥的风格，是不喜欢安排得特别细，你懂得，不过工作进度我还是很看重。本来计划今天开会，谈一谈，结果没人来，那好吧，我就说说波哥目前遇到的问题，看各位小伙伴，能不能为波哥排忧解难啊？

* 我们新开了接近10个新客户，包含直营的和一级商的，我们不希望新客户进来找不到组织，换句话说，为了让新客户更快上手，我们该做些什么工作呢？该谁去做这些工作呢？ 请问李大育 王海梦 王思思，三位大神，此题如何破？ 求写出详细点的方案出来，每个人哈
* 12月份，有乡镇会议要开。我觉得以往的我的乡镇会议方式要改进了。其实，我特别依赖美的那些高大上的视频，在ppt中穿插很多这样的视频，讲了很多品牌文化类的东西，结果容易把会议节奏拉的比较长，而乡镇会议的干货在于政策环节和签约环节，我想比以前有点改进，比如抽奖方式，能不能使用头像抽奖啦？品牌文化和产品讲解能不能分别给不同的人讲？这样会不会，让乡镇客户听得更认真些呢？  我需要各位给些点子。也要写出来？
* 第三个问题是，地毯行动第二轮后，王斌和王思密有木有 新的感触，对于乡镇开发的方式，我们还需不需要继续改进呢？比如价格如何给啦，活动是不是可以承诺啦，是不是应该整理一份乡镇合作协议啦。。。。。等等之类的。这个需要王斌和王思密，写一份出来。
* 第四个问题，开江新客户，承诺有带路演的开业活动，是不是应该谈一下路演公司之类的了？万一合作了，我们要有合作的路演公司能够去办才行啊？我们自己的乡镇有木有必要搞点促销活动呢？  是不是该采取路演形式呢？这个需要王斌，回答。（提醒，乡镇客户头像照片搜集）

各位小伙伴，以上就是波哥的烦恼，各位把自己的答案，直接在这条笔记里面回复，我能看见哈。 拜谢各位大神
