# EverRec 20210816 总对王海梦发火上情绪 其实是我想用情绪掌控别人

00:00:03 - 00:00:50
刚才我又另外想到一个十四块，关于对王海萌的给我们的相处里面的问题，我总是对他发火，就比如说他起晚了，我就特别想表现出对他的失望，不失望之后也让他感受到的失望，进而让他紧张。这背后应该是想通过这种情绪控制它，应该是想控制他。反正不知道怎么样。我看到我的表现你怎么又迟到了？我现在几点？我你到底要做事儿？就这样就这样说之后，他会紧张，彭博给我迟到了。或者说好，这样做其实对他才能一直没有，而刚才我去教他的时候，我就是这样做的。

00:00:50 - 00:01:33
我说的是so你今天是坐回主持人，你早一点起来，多几分钟，思考方面能做好一点，就是在再提醒他，给他做体系或者做备忘。做这个我自己心里也平静很多，也有点少了很多愤怒刷成了我自己拼很多。他肯定是更好的这个事儿。不是，确实是看到兵哥他的一些处理方式之后，给我的启发和影响了。可能不过度解读了，我估计我需要怎么改项目背后的逻辑应该是想控制人家。
