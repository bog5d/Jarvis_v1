# 有点生气 媳妇儿生气出差太早让我改签

今天下午原本六点的航班，一直陪着娃 还是挺开心的，都有点舍不得走。
等到了四点 临出发了，老婆开始不高兴了，坐那儿😠😠，大意是： 

* 你这么急过去到底见谁，非去不可？
* 你就是想早点跑，躲避带娃责任；
* 这种感觉让她想起 在上海时，我连夜出差，留她一个人在出租屋，那感觉很难受很难受；

我一看这节奏，不知如何应对，改签了算了。

但是跟老李沟通时，老李很不爽：

* 那你就别创业了呗！
* 你改吧，反正你过来也产生不了多大价值。

都是不省心的种

我想了下，两方利益和情绪无法兼顾，肯定做不到让谁都满意。
如果选择了留下，还是把家里陪好，老李不满就我也相信他消化情绪后，不会真在放心里。
因此你该做的事情是：稳住。
