# 小白 元仔

小白 元仔
我请教一个问题， 如果我能汇总几十家企业，帮银行吸储一两亿，这中间有套利空间吗？

背景是这样的，我们开发了一套企业saas软件，想定向给到企业使用，不过前期不打算直接收费，不过我们可以让这些企业把自己的存款，存到我们指定的银行，这样实现对这家银行的吸储。
然后，通过这家银行把我们的系统成本付了。

不知道这样是否可行？
