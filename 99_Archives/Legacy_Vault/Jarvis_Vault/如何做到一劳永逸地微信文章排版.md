李笑来曾说：最好的投资是自我投资。

而在这个时代，**最好的自我投资无疑是：写作。**

所以越来越多的人开始了写作。不管是写读后感，写日记，写听课笔记，还是写践行经历，写生活感悟，等等，刚开始的写作小白可能会遇到这样一个问题：

> 自己写出来的一堆文字，放到微信公众号或简书这些平台上，发表出来后还是一堆文字，阅读体验很差。

任何文章要让读者读得爽，都是需要排版的，而排版却让人很头疼。我有一位朋友也在写作，她在排版上花费的时间竟然比码字的时间还多，我感到很惊讶。

大概了解一下，原来她直接在微信公众号与简书平台上写，然后一处一处来排版，光是插入图片，链接，设置不同级的标题样式，加粗，高亮，引用，设置字间距、行间距等都可以搞上几十分钟到一个多小时。

我刚开始写作时也使用写作平台内置的富文本编辑器来排版，还用过各种基于网页端的第三方编辑器，比如135编辑器，秀米，i 排版，新媒体排版等。

这些工具大同小异，提供基于 CSS 的不同主题样式，但花哨的样式很容易让人陷入选择困难的境地，所有的文章发布出来出现不同的排版样式，五花八门，如果设计感差点，还有可能被排成老年人阅读版，这样的排版，令文章识别度大大降低的同时，花费在上面的时间也会很多。

难道那些高逼格排版精美的文章，不需要花大量时间，就能排出来？

真不需要，人家排版可能只用几分钟，因为有一个神器：**Markdown 编辑器**，具体操作步骤就是以下几步：

> 1. 设置 Markdown 编辑器；
> 2. 在 Markdown 编辑器上码字，顺便练习 Markdown 语法；
> 3. 在 Chrome 浏览器中安装 Markdown Here 插件；
> 4. 自定义 Markdown Here 的 CSS；
> 5. 把 Markdown 编辑器上码好的文章复制粘帖到微信公众帐号的编辑器中；
> 6. 使用 Markdown Here 一键渲染，完成排版；
> 7. 插入图片，审核，修订；
> 8. 保存并群发。

如果你此时看不懂上面的步骤，没有关系，以下是详细教程，保证你能学会。

### 1\. 什么是 Markdown

Markdown 是一种标记语言，它的作用是：**让你输入几个特定符号就能实现高效排版，节省大量时间。**如果非要列举它的特点，主要有以下这些：

> * 任何人都能学会，而且是快速学会；
> * 让你专注于码字，写的同时顺便完成了排版；
> * 从此不再需要鼠标，几乎纯键盘输入；
> * 它的兼容性极强，几乎所有文本编辑器都能打开；
> * 它的源文件和渲染后的文件都有极强的可读性；

Markdown 作为一种如此好用的标记语言，在很多写作平台与博客都支持它的使用，比如：简书，有道云笔记，Wordpress，Logdown，Farbox，Gitblog，Ghost、Jekyll 等。

千万不要被它的语法所吓到，它简单到你在半个小时内就能学会使用，一般的文章写作主要用到语法就是以下这些符号（英文状态输入）：

> * 一级标题： 标题内容前加 #
> * 二级标题： 标题内容前加 ##
> * 三级标题： 标题内容前加 ###
> * 加粗：加粗内容两端加 \*\*
> * 斜体：斜体内容两端加 \*
> * 引用：引用内容前加 >
> * 列表：列表内容前加 \* 或 -
> * 加框：加框内容两端加 \`
> * 图片：！加 \[图片名称\] 加 (图片网址)
> * 链接：\[文字\] 加 (链接网址)
> * 邮箱：<邮箱地址>
> * 删除：删除内容两端加 ~~
> * 分割线：三个或以上的 \* 或 -

### 2\. 设置 Markdown 编辑器

当我们学习完 Markdown 这种标记语言之后，接着就要用起来。借助的工具就是：Markdown 编辑器，是指支持 Markdown 标记语言的文本编辑器，你在上面输入 Markdown 语法符号，排版效果就马上出来，也就是让你轻松实现边写边排版。

目前网络上 Markdown 编辑器很多，比如：iA Writer，Ulysses，Byword，Mou，Mweb，Atom，Sublime Text， Wilde，Typora，马克飞象（支持印象笔记）等，还有一些写作平台或工具也内置有 Markdown 编辑器，比如：简书，为知笔记，锤子便签等。

这么多工具，没有任何一款是完美的，你只能找到最适合自己的。

如果你是初学者，我推荐使用简书的内置 Markdown 编辑器就够了，界面简洁，操作顺手。具体设置如下：

> 1. 登录简书个人帐号，设置使用 Markdown 编辑器。
> 2. 新建文章，切换到预览模式，开始码字。

### 3\. 安装 Markdown Here 插件

如果你是在简书中写作并发布文章，其实上面的步骤就可以让你轻松从写、排版到发布。而有很多同学需要把文章同时发布在自己的微信公众帐号，为了完成一键排版，那就需要先安装另一个神器：**Markdown Here。**

这个插件的主要作用是：把浏览器中的文本编辑器里的 Markdown 格式的文本转换成格式化之后的 html，也就是把 Markdown 格式转换成我们平时看到的微信文章精美排版的效果。

它支持 Chrome，Firefox 与 Safari 三款浏览器。我推荐使用 Google 的 Chrome 浏览器，好处你用多了就会明白。以下是使用 Chrome 浏览器安装 Markdown Here 的教程。

到这里，Markdown Here 神器已设置完毕， 你就可以把简书上写好的文本，复制粘贴到微信公众帐号的编辑器中，使用 Markdown Here 一键渲染，完成排版。

OK，大功告成！

到此，排版神器组合：**简书 + Markdown 语法 + Markdown Here** 的基本设置已全部介绍完毕。

值得注意的是，微信公众帐号的文章，如果要插入图片，必须要在其后台编辑器中先上传才能插入。所以，我们用 Markdown Here 完成一键排版之后，需再单独处理图片插入。

所以，神器设置好了，以后写文章，就可以按下面的步骤：

> 1. 打开简书或其它 Markdown 编辑器，写，记得同时根据想要的排版效果输入 Markdown 标记符号；
> 2. 写完了，直接发表在简书上，或复制所有文本到微信公众号编辑器中；
> 3. 使用 Markdown Here 一键渲染，完成排版；
> 4. 插入图片；
> 5. 修订；
> 6. 群发。

赶紧践行上面的步骤吧，相信你会爱上 Markdown 的。
