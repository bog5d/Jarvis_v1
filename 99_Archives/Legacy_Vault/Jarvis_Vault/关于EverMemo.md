# 关于EverMemo

关于EverMemo

2013年7月，我们开始开发这个小应用。初衷只是因为我们都是Evernote的用户，但是不喜欢iPhone上那个又卡又慢的大象。我们只需要一个简单的便签，能够随时记录下脑子里一闪而过的点子。
然后就有了现在你看到的这个应用。我们会坚持快速的记录、组织和分享，不会增加任何与之相悖的功能。希望它也能对你有所帮助。

关于 Mocha
我们相信，人生里的很多答案其实都可以在大家的笔记本里找到。
我们也相信，知识只有在共享时才能发挥它最大的价值。
于是我们开辟出了Mocha，一个完全开放的空间。你可以自由的在这里分享任何的笔记或是记录，也希望当你遇到困难时，别人的分享能给你带来帮助。
当你找到有用的信息时，可以点咖啡杯标记来保存。如果你发现有不适合的信息，也可以点小旗帜举报给我们。
