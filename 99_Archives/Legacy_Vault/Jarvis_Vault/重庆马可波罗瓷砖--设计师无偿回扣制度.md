# 重庆马可波罗瓷砖--设计师无偿回扣制度

昨天跟远古装饰易姐聊天聊了很久，其中她提到重庆马可波罗瓷砖的销售制度，只要一个用户装修的房子使用的是马可波罗瓷砖，不管有没有推荐马可波罗，都有权利拿20％的回扣。

* 这样所有的设计师都愿意推荐马可波罗的瓷砖，因为钱好拿。
* 这跟大竹王者陶瓷是一样的道理，只要工人推荐了的主动把回扣送去。
* 注意事项

1. 成交地方必须是大竹总店（新三中对面）
2. 设计师和工人师傅是否冲突。比如我无偿给了设计师回扣，那工人师傅给我说是他推荐的咋办呢？
3. 价格空间必须足够。

操作步凑:

1. 统一全城价格 （1我做统一价格表，含标价，常规价，套餐折扣价，单品折扣价，恶性竞争价格线。）
2. 印制价格标签。告（含价格，售后服务热线，大竹监督投诉电话，印章，）
3. 搜寻我们所销售客户的设计师信息。
4. 主动送回扣及《给家装设计师朋友的一封信》 （阐述清楚备案提成10个点，不备案提成5个点，以及我们不通过请吃饭来促成合作而是通过价值利益分配达成长久的联盟，我们也按照活动价格执行，不让你们背负用户比价后埋怨的风险。）
5. 扩大设计师同盟数量。定期发促销信息，增进设计师关注度。
