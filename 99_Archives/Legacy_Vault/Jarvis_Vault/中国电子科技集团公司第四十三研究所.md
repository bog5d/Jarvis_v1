# 中国电子科技集团公司第四十三研究所

中国电子科技集团公司第四十三研究所（简称43所）创建于1968年，最初坐落于陕西

凤县

，1982年整体搬迁至合肥，是我国最早从事微电子技术研究的国家一类研究所，也是我国唯一定位于混合微电子的专业研究所。43所数十年如一日，致力于混合集成电路（HIC）及相关产品的研制与生产，为

电子信息系统

提供小型化解决方案，先后主持制定了《混合集成电路通用规范》(GJB2438)等30余项国家及行业通用规范和标准，已成为我国高端混合集成电路领域的领军者，为推动国内混合集成电路行业的发展做出了贡献。

简介

中国电子科技集团公司第四十三研究所目前43所辖东、西两区，占地170亩，固定资产5.72亿。现有在职员工1485人，本、硕学历以上人员超过55%。其中工程师222人，高级工程师110人，研究员高级工程师27人，中高级以上职称占全所职工的比例为24.2%。拥有一支52人的专家队伍，占全所职工比例为3.5%。其中，享受国务院政府津贴3人，集团公司首席专家2人、高级专家3人、专家10人，43所高级专家13人、专家25人。除研究所5个事业部外，43所还拥有3个全资公司。

成就

中国电子科技集团公司第四十三研究所拥有目前国内领先的

厚膜混合集成电路

、

薄膜混合集成电路

、多芯片组件（LTCC）、SMT模块电路、金属封装外壳、宇航用厚膜混合集成电路以及AlN陶瓷材料等7条生产线。其中，厚膜、薄膜及金属外壳生产线均通过中华人民共和国国家军用标准，宇航用厚膜混合集成电路研制线是国内第一条厚膜混合集成电路宇航线。43所还设有EDA设计、试验检测、标准化、信息情报4个中心。

混合集成电路

及电子元器件检测实验室同时具备国家实验室、国防实验室和总装军用实验室三项资质。通过了GB/T19001-2000质量管理体系、GJB9001A-2001军用产品质量体系、GB/T28001-2001职业健康安全管理体系、GB/T24001-2004环境管理体系认证，是二级保密资格认证单位，安徽省

高新技术企业

，合肥市、安徽省文明单位
