# 豆瓣电影TOP250的一句话介绍（民间版）

豆瓣电影TOP250的一句话介绍（民间版）( http://www.douban.com/group/topic/51376846/ ) 分享自@豆瓣小组 Android 客户端
    如果你是真正的电影迷，你不可能不知道“豆瓣电影TOP250”，这是豆瓣评分最高而且评价人数最多的250部电影，换句话说，也就是大多数人认为最好看的250部电影。
    不过，真正能把这250部电影全部看完的人并不多，即使你是电影发烧友，可能也有一部分电影是你没有看过的。对于我们这些忙碌的现代人来说，时间永远是不够用的资源，这当然是其中最重要的一个原因，但是其实还有另外一个重要的原因，那就是我们对这些电影不够了解，不知道它们到底好在哪里，当然也就不愿意在这些电影上面“浪费”时间了。
    现在的实际情况确实如此，要找一部自己喜欢的好电影真的不太容易。如果没有人推荐或者媒体的宣传，仅凭借电影的名字我们是没有办法判断电影的好坏的，甚至在很多情况下，电影的名字本身会对我们形成一种误导，例如，如果你仅仅从《三傻大闹宝莱坞》这个名字对这部电影进行判断，你很可能会认为这只不过是一部普通的搞笑类电影，没什么好看的，但是这却是一部红遍全亚洲的优秀电影，不但搞笑，而且还很励志，甚至发人深省，可是这些信息我们是无法从电影的名称中得到的。所以，仅仅从名字去判断，很可能你就会和这部好电影失之交臂。
    当然了，在当前资讯如此发达的时代，要深入去了解一部电影，绝不会是一件太难的事情。比如，我们可以去豆瓣和时光网去看看影评，可以在百度上搜索一下电影的其它信息，但是这样做又带来了两个新的麻烦：首先，信息量太大，我们需要太多的时间才能发掘出自己喜欢的好电影；第二，在调查和了解的过程中，我们难以避免地要接触一些与剧情相关的信息，甚至在电影还没有看过的情况下，已经知道这部电影的大致情节了。对于希望从电影中寻找惊喜和快乐的人们来说，这实在是一件大煞风景的事情。
    所以，在这里，我们做了这样的一种尝试。我们参考了每一部电影的影评等各种信息，将其中有代表性的内容找出来，对文字进行加工和处理，形成一个不到50字的一句话介绍。通过这个介绍，我们虽然不能了解到这部电影的方方面面，但是却能大致了解这部电影是什么类型，是讲什么的，有什么特点，好在什么地方等等诸如此类的信息。这样，对于一些电影，只要简单看一眼介绍就知道肯定不是我的菜，例如笔者对于那些恐怖题材的电影一向是敬而远之，那么只要看到简介中带有这样的信息，我就不用继续去深入了解了；同样，如果这正是我喜欢的哪种类型，我也可以很快地发现它。于是，找电影就会更加方便快捷，很多情况下，不用花费太多时间去阅读大量影评也能找到自己喜欢的电影，而且可以规避被“剧透”的风险。
    需要说明的是，豆瓣电影TOP250本身也是有一句话介绍的，不过，大家可以对比一下，和我们这个民间的版本相比，现在豆瓣官方的一句话介绍实在是太简单了。
　　下面，就是我们的工作成果，不知道是否真的对大家有所帮助。如果你觉得有什么地方需要修改，欢迎帮我们指出来，在此先谢过了。本文的作者是天天选优网（http://www.xuanyouwang.com），如果您要转载，也请说明出处，并带上我们的链接，非常感谢。
