# 乱写乱画呀

之前已经记录了一篇了，不过要同步很久，那就再写一篇吧。
关于日逼做爱的事情，我开始有一点点心得了，那就是，在抚摸时一定一定要多抚摸，多抚摸，比如上次摸老高的逼的时候，我就心中一直默数，一二三四，儿二三四，基本上都会重复到一百次左右。那样老高慢慢的，就来感觉了。所以，一定要注重抚摸的数量，当然到了日后可以多注重抚摸的质量，这是个循序渐进的过程。
还有好多书没有看完哟，
- [ ] 自控力啦，
- [ ] 番茄工作法啦
- [ ] 销售就是搞定人啦
- [ ] 乔布斯啦
- [ ] 专注力啦
这么多书为什么都没能看完呢。真的太没有恒心了，先把本月的俯卧撑月过完。每天20个俯卧撑，依次增加吧，不过保底20个想增加就增加，保证今天不比昨天少就行，不给自己一种这件事很难得感觉，是开心愉悦的做，才更容易坚持。
每天早上起来，20个俯卧撑，标准的那种。时时刻刻把目标挂在嘴边，每天20个俯卧撑，每天20个俯卧撑，每天20个俯卧撑，每天20个俯卧撑，每天20个俯卧撑，每天20个俯卧撑。一屋不扫何以扫天下，小小俯卧撑不做好，何以坚持完一本书。
对了，这几天看书也没有坚持好，最大的原因还是去的比较晚，还没来得及看就来事情了，然后忙碌的一天就进去轨道开始运行了。每天提前一小时去，是特别好的调整酝酿期。这个习惯必须加以强化。
看来我的三个目标

1. 早起，八点前上班
2. 每天看书
3. 每天20个俯卧撑

如果要实现这三个目标，都只能是建立在早起的基础上了，只要早起实现了，其他几个就轻而易举实现了。
所以早起早起哦。
