# 0513 需要跟自己较较真

上午去办公室，昏戳戳的，眯了又眯，小马哥叫进去聊了聊，觉得开江完全不是在一个进度上，很求恼火。我打算去开江收编杜安乐

做完那么重要的自省日记 都没保存好  他妈的坑爹的印象笔记。

我做完和老婆闹了分歧
 他说王斌他们贪耍行为习惯很差，我就觉得老婆怎么一直在说别人的缺点呢
我还发了这样的微信
你真的在一直说人家的不好，你认为随便说说，就没有问题。你知道面由心生吗？如果你一直想着别人的缺点，自己也不会有一个具有亲和力的气场和面相。
我也不认同他们的一些生活方式，不过求同存异嘛。
如果你多摆摆别人好的地方，你自己也会开心很多呀，干嘛一定要摆这个不好，那个要不得，而且如果我不能认同你所说的别人的缺点的话，还会对你火大，生气挂电话。那我连最基本的自己的看法都不能具有，那是不是某种程度上的专制呢？宝贝，需要你自己看透看开一些，
我这么说，并不代表我就认为他们好，而是我不认为你一直关注他们的坏，会对你有帮助。尤其是并不会带来，你的心情舒畅。我们专注买自己的娱乐生活上，不是更好吗？
我都感觉到好像，分了派别一样，一个王斌派和老高派，会有内耗的呀！老婆

因为当时她挂了电话了，觉得我不理解她。
我在去刷牙前  我就在怀疑老高是一种嫉妒心态，“你有耍的 我还不是有耍的” ，因此就去和冷瑞骑车 开电影，我理解成为一种以牙还牙心态。
我就觉得不舒适塞 心里面，后来我想起读书微信号发过一篇文章，说人就是要给自己找点不舒适，那样会促进自己的提高。所以我就
在想，自己这么不舒适，肯定要原因的，如果我坚持现在的状态就是继续不舒适，如果我换一种想法可能就舒适了，于是我推翻了自己之前的看法，再来思考这个问题，“会不会老高就是单纯的看不起王斌这样的行为呢？”  会不会是我自己有那种小贱的“**你有耍的 我还不是有啊**” 那样的想法，才会想当然的认为老高也是这样想的呢？ 而且 我好像确实是有那样的想法，比如之前王斌和王海梦耍朋友时，我羡慕得很，所以就在想一定要做点事情来让王海梦也羡慕之类  
这么一想  我就觉得  嗯好想确实是我的思维出发点有问题，就跟老高忏悔了。然后非常愉快的聊天聊了很久

因此 不舒适的时候  就跟自己较较真 是很有必要的
