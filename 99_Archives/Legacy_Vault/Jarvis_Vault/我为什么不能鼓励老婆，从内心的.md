# 我为什么不能鼓励老婆，从内心的

怎么说呢？
老婆一直觉得自己的创业计划很有含金量，其实我是认可的。
就是老婆有时说话的神气呀，让我好难接受呀！个人觉得，老婆过去这几年在工作方面一直没什么成就感， 然后因为有了创业点子，并在开始实施了，也许就产生了短暂的成就感爆棚！
哎呀，自己在写日记的嘛，怎么不敢放开心扉说嘛，生怕得罪人呢，王波。其实我就是觉得老婆说话有点“冲”，听到不舒服。又怕打击到了她的自尊心，不敢与之争辩！
唉，心头不爽得很
她说她马上准备，就不去门市了上班了，专心做自己的事情，每次听到这些我就心头窝火。  
难道是我真的太依赖她在门市上的价值了吗？说实话，我内心一点都不承认的！希望我自己不是强词夺理，要面子。
对于老婆的工作价值，我一直不太认可，因为她懒惰的作风和做事拖拖拉拉的风格，长期让我对合作伙伴心怀愧疚，比如每天去门市基本上都是10:30，录一下单据的话，本来一天整得完的，非要拖啊拖。 然后一边录单据一边逛淘宝，越逛越来劲，基本上会报废她一半的工作时间。本来上班就晚，十点半，然后自己的事情又多，再花个半小时摩梭一下，一上午基本上就完了。
唉，反正分内之事是做得很少的，并且我的很多管理制度确实推行不起来，比如我要打卡上班，她是第一个反对的，因为她自己不能按时上班。
还有我要实行弹性工资，就是假如邱艳900元的底薪加200元的考核工资，根据每天的工作计划的完成情况来，考核。
不过她发工资是直接就发了。咦，对了，不能全怪她，我自己也在偷懒，不想去管这些细节活，所以就没能把弹性工资执行到位，哎呀呀，这一点我自己也有责任。
我现在才想到一点，就是做管理，应该更多的去做【紧盯】的工作，也就是指定制度，监督制度执行。不能每天去跟下属此谁更勤快，向他们显示自己是个辛勤劳作之人，然后让自己不觉得亏欠别人。  结果很多制度就像屁一样松懈求。
其实，很多优秀的制度没能做到位，就是监督紧跟没做到位。
  回来继续说老婆，我对老婆工作价值的第一反应是觉得她没做多少事情，爱睡懒觉，做事又摸。  不过我老是忘记她的作用，她对资金安全的管理是相当到位的，也就是说钱由她管到是特别稳妥的事情，我们基本上啥子都不用操心，晓得钱是安全的嘛，至少。
而且我们要的账目基本上是清晰的呀。
不过我也觉得，很多时候她接受不了别人意见的确很是恼人。有些事情，明明知道她是错的，并且有很好的改正意见，她是绝不会接受的并且会反过来警告你“ok ok stop，别让我提醒第二遍，闭嘴！”，唉，真的是气人得很！
  她要做自己的事情我是挺乐意的，这样可以减少因为工作给恋情带来的摩擦，又能让老婆自己去磨掉一些菱角，心智会更成熟，而且更重要的是，说不定就多了一条挣钱渠道了。  真的是利远大于弊的一件事情，因此，应该不遗余力的去帮助老婆实现她的计划。
给自己减负，给老婆圆梦！
别让老婆上了年纪后悔当年老公没支持她。

对于我自己来讲，我就是需要赶紧重新梳理一下14年的格局。 
目前的难点还是在于  批发数据   费用发放 
收钱   记账   防止资金非法流失。
我们的系统能不能解决？
可不可以使用财务系统重新来梳理呀？
