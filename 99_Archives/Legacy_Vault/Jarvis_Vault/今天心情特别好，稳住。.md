# 1115 今天心情特别好，稳住。

确实，今天心情真的很好哟。为什么呢，因为聊得比较开心😊😊😊😊😊😊，哈哈哈。
和谁聊得比较开心呢？

* 先是和王海梦，从大竹回来后，下了高速，我们在南外收费站附近停车了，然后在车里聊了会儿天。主要是关于对姚姐的看法，她说了一些她认为的负面的地方，比如有点绿茶婊啦之类的，我也不反对，因为我也知道姚姐有点点缺点，就是过于有点点“做作”，但我也清楚姚姐对我的好呢，也不算是虚情假意，还是有很大的真诚成分。至少对我不像对别人那么敷衍，做样子。所以对王海梦表示了些许认同，她也蛮开心的。（哎呀皮蛋记得去拿，在车上。）然后很开心地跟她分开了。flykiss 了一下。（开放的态度去认同还是会换来别人的开心，和自己认识其他人的客观）
* 接着又和姚姐王平面谈了一下，虽然等了一会儿王平，回来后，平哥说要开西外好一新，希望要点价格支持。最好是20个点，实在不行18个点也可以。 我说回去申请一下看看。然后就到了老车坝，达州人美食城。哟呵，聊天聊得之乐呵哟，我还是希望王平支持姚姐出去再次创业，就说了些我的观点，王平也不排斥。不过他骨子里认为，姚姐应该听他的，反正还是有点专制。我就觉得，美的厨卫也算不上终身职业，姚姐有更好的渠道不应该放弃。  反正就听到他两口子互相的指责之声。 后来就算了一会儿，然后我就转移注意力，主要说到老高最近还是蛮好沟通的，还有我的门市经营，我充满希望，因为明年成本会很低，我挺开心，也自夸了一下，我对门市的融资贡献。反正挺开心的那种，我也能感觉到王平姚姐，对我的话还是蛮信任的，朋友的信任带来了不少幸福感。
* 然后回凰城一号 停车时遇到了牟浪，打了个招呼，也挺热情。知道了他住二小附近。（可能是心情好吧觉得这个世界都好）
* 回家时，先把小马哥干洗的衣服带回来了。 跟小马哥聊得也挺开心的，我回来说了我希望给王平申请20个点价格，老马驳回了，不过把道理讲的很清楚，就是王平现在的价格优势是具备在好一新生存的，比如向阳电器之类的我们再去深查一下，保证这些不成为王平的阻碍。 如果给了触底的价格，就没办法给予一些类似于送ipad啦，送车子啦，免费装修资源啦之类的政策资源了，反而会因为缺少政策性引导，让他做得更艰难。 我一听觉得有道理哦，我就说“我觉得你说的是对的，那我去给王平反馈一下，不过马部也直接给王平答复一下，这样也好让王平知道我是争取了的，让我对他的私人情意有个交代。” 
* 再后来，小马哥聊到了，他买的电脑将近2万2，向他老爸借2万， 他老爸问他“买两万的电脑真的有必要吗？”
* 答曰 “我觉得我已经没得啥爱好了”
* 他老爸无奈的叹一口气，问了句“还是打农行嘛？” 随后打了2500元过来，我听后觉得那个 温馨的滑稽 氛围，让我笑惨了。小马哥是个无所谓的人，他老爸又是一个特别遭得住的人，真的有点好耍！
* 再后来，小马哥提到，如果我们出货艰难的话，可以想想办法给我们设个三五万的额度，让我们周转轻松一些。又提到QL302 给我们350元，让我们400元发给赵双平。 我觉得特别好，应该处理得掉。 反正挺爽的，心情！
