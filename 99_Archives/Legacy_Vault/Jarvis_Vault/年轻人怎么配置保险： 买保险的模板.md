# 年轻人怎么配置保险： 买保险的模板

买保险的模板

年轻人怎么配置保险：

年轻人的保险配置是相对复杂的，我们先回忆一下配置保额的几个公式：

第一，寿险保额 = 未来应该承担的家庭负债 =当前负债+未来预计开销-当前存款

第二，意外险的保额 = 寿险保额+未来护理及生活费用(按照 100万预估)

第三，重疾险的保额 = 3-5年的年收入 + 重疾治疗费用

第四，医疗险买高额医疗险+基本医保，条件好的买高端医疗+基本医保

成年人的保障按照上面计算就可以。 下面我们举个例子。

A家庭的情况如下：

丈夫小强先生的年收入是20万元，妻子小美女士的年收入是10万元

家庭负债包括房贷100万，车贷10万，没有消费贷

孩子现在6岁，独立前所需费用预计是40万

赡养老人预计一共需要30万

没有存款

那么如何计算保额呢？

小强的保障方案：

寿险=未来需要承担的家庭责任=【房贷（100万）+车贷（10万）+子女教育（40万）+老人赡养（30万）】\* 小强应该承担的比例（暂且按照小强收入/家庭总收入 =2/3计算）=120万。

意外险=寿险+100万=220万，如果小强配置寿险，那么意外只需要100万

重疾险 =3年收入+重疾治疗费用50万= 110万，如果小强购买医疗险，那么只需要60万保额

医疗险= 1万免赔额的高额医疗险

所以小强最后的配置是：

寿险120万

意外险100万

重疾险60万

医疗险1万免赔的百万医疗

小美的保障方案：

寿险=未来需要承担的家庭责任=【房贷（100万）+车贷（10万）+子女教育（40万）+老人赡养（30万）】\* 小强应该承担的比例（暂且按照小强收入/家庭总收入 =1/3计算）=60万。

意外险=寿险+100万=160万，如果小美配置寿险，那么意外只需要100万

重疾险 =3年收入+重疾治疗费用50万= 80万，如果小美购买医疗险，那么只需要30万保额

医疗险= 1万免赔额的高额医疗险。

所以小美最后的配置是：

寿险60万

意外险100万

重疾险30万

医疗险 1万免赔的百万医疗

另外从性价比角度，寿险我们推荐定期寿险，因为寿险第一杠杆高，性价比高；第二人过了60岁，所承担的未来责任也会变少，没必要买到终身。

所以我们重点推荐的还是定期寿险，重疾险我们推荐终身的，因为到了一定年纪发病率高，后面不交也不划算。意外险和医疗险我们推荐1年期产品，经济实惠。

所以30岁小强最后的配置是：

寿险120万， 定期到70岁，30年交，保费：3372 元

意外险100万，1年期保费299元

重疾险60万，终身，30年交，保费：6330元

医疗险 1万免赔的百万医疗，保费：306元

总计： 10307元，占年收入5%左右

所以30岁小美最后的配置是：

寿险60万，定期到70岁，30年交，保费：1020元

意外险100万，1年期，保费：299元

重疾险30万，终身，30年交，保费：2880元

医疗险 1万免赔的百万医疗，1年期，保费：306元

总计： 4505元，占年收入5%左右

上面就是一个完整的保险规划方案了
