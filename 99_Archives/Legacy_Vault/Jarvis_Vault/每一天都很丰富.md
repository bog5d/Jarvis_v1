# 4.24每一天都很丰富

![20140424_162444.jpeg](./_resources/4.24每一天都很丰富.resources/20140424_162444.jpeg)

来吧来吧，记录真实的一天吧。

1. 今天上午，老马要求任务跟上，突然发现大竹让人失望。王平节奏尚且不错，其他就只能说是特别平常。
2. 下午，叫上王平小龚，到魏家收编。
3. 晚上，让王斌上达县来，好好谈谈收心。
4. 和老婆聊天，恨我只谈斌梦。

大概就是这样的一天吧，说真的，王斌这段时间谈恋爱去了，乡镇的状态开始下滑了。小马哥也好好的跟他谈了一下，不过我觉得小马哥更多的还是迎合我一下，因为今天是我提起的，让王斌上来，让他帮帮我，帮我说一下王斌。  
大概说了一下，王斌也还是挺重视的。不过说了之后，我就觉得没那么紧张了，那种紧迫感和气愤感也没有了。晚上，梦梦还是过来找王斌耍了，我则继续回来耍杀手5赦免了，耍的是图书馆章节。到现在为止都还没有过呢。
还有，我觉得还是有点过于羡慕王斌梦梦了，少关注了自身和老婆。其实，我应该继续把老婆和我的生活过得更有声色和情趣。
我隐隐觉得自己还是有种，比较心态，
怎么说呢，就是希望自己和老高的情趣生活是让他们羡慕的，这里指的他们是特指王斌梦梦。这种心理是很有问题，因为这样的话，就算我跟老高做出幸福浪漫之举，也是希望做给他们看，而不是更多的关注内心享受。那如果没被他们看到，浪漫之举，是不是我做的所有努力都没有成果的感觉。所以，很畸形的。应该想的事情是老高真正喜欢什么样的，行为。然后我也去做，去享受
晚安波波。
