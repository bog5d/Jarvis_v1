# Day One Entry: 2015年1月3日

老婆明天回来了，也还是挺期待的。
突然挺焦心门市上的事情了。我觉得自己就是那种犯贱的人。
有老婆在身边时，觉得应该多去关注门市上生意。其实，我一个人的时候，压根儿就没怎么把心思放在门市上。 只是门市遇到问题时，如果我是独立思考的话，会更锐利些，老婆在身边会左右↔我的思考。女人家的非理性成分还是比较重。比如我遇到王斌那天偷懒了，我说出来，老婆的通常反应会是“哎呀，这些正常得很，你也别太着急了，反正以后我们离他远点，不住一起就是了。。。。” 我就会觉得更烦躁，而我一个人的时候，我会思考，这是个机会，彻底推进一下门市管理，先把王斌一顿痛斥，再制定出标准，让他去搞。肯定会比单纯的烦躁好很多的。
到这里有个问题，老婆在身边，我就习惯性去思考工作，那是为什么呢？
想逃避老婆吗？觉得只要陪了老婆，就是过得很悠闲吗？ 习惯性陪老婆的时候愧疚工作。 犯贱呢。
我是不是该好好地享受享受跟老婆一起呢？但很多时候我觉得陪老婆，比工作还痛苦。陪老婆就是逛街，逛街，这种休闲方式真的不是我喜欢的呀，感觉痛苦的呀。 但是，一起做点饭，一起宅一宅还是蛮喜欢的。逛逛公园之类的，我喜欢，她不喜欢。 她喜欢逛街我确实不喜欢而且又要不停花钱，是我痛苦的最大根源。如何，把这个变成快乐的事呢？
我觉得真的好难哟。如果女人逛街是她们的幸福所在，而一起走的男人👨真的是痛苦的，这种行为矛盾如何调和呀？

最大的课题是：如何使逛街这件事让男人也觉得快乐。

我知道能够陪老婆逛街的，一般两口子感情都不会好差。所以陪老婆逛街的必要性还是不言而喻的，只是目前真的觉得逛街是一件很痛苦很痛苦的事。尤其是逛衣服首饰店面，那种枯燥得要命的等待，太折磨人了，真的是。而我逛街稍微快乐点的就是买点饮料🍹呀，吃的呀那些。
说起逛街，我就想起无尽的消费。确实逛街的消费一直让我一个人开销，还是多痛苦的。 15年好想好想存点钱起来哟，但是真的，老婆一直没啥收入，基本上全是我在挣，大小花销都是我来承担，从骨子里面还是有埋怨的吧。 只是我自己都不知道，我是有怨气的。现在回忆起来，肯定是有的。
挣得钱再多，也看不到一分，还是月光族。真的还是挺难受的。
不过还好，因为之前的我本身还是一个不太注意性价比的人，消费随意的人。觉得花钱嘛，花嘛，只要开心，懒求得算值不值得，比如老婆想买个什么大东西，一看，哟还有个两千，那就买嘛。我自己也会是那种，包里有两千元，就认为标价两千以内的东西，都是我可以拥有的。
我错了，我有两千元，可能真的只该买50元以内的，因为可能要还1250元的贷款，加上未来2周的生活。

唉，现在，我能保持一个较低的消费水准，那真的是多亏了老高这段时间花钱太凶了，让我不敢去消费。然后才发现没那些消费其实一样的过得很有质量，幸福感一点都没有降低。 坚信自己的财商肯定比只会消费的女人更高点，女人的冲动消费压根儿都控制不住。只要在她视线范围内的钱，都会以为是可以花费的。假如交了2万给她，她也会把标价2万以内的产品视为己有，当然在网上选购时，就会往3万左右去比较。 这可能是女人的天性。
我应该干的，真的只有默默地把她看不到的财产，合理的打理起来，为未来的幸福生活埋下铺垫。

之前每个月存给她的钱，都是被她花掉了，我自己也没有珍惜我的收入，因为也许我自己拿到那些钱可能都会无意识花掉的。

唉，说起来不愿意逛街，还是不愿意花钱和不愿意动。 让我专心的陪伴老婆如果没有其他的情绪影响，我肯定会好好努力，陪伴好老婆。跟老婆一起时，专心陪她聊天，是我需要练习的地方。 而不是一直神游，端正态度陪老婆说话💬，是我应该培养的习惯。

如何使逛街成为男人也快乐的事情？
如何端正态度的和老婆聊天说话呢？
两大人生课题，王波加油↖(ω)↗。
