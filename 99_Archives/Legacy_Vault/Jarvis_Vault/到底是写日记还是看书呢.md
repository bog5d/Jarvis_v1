# 10.11 到底是写日记还是看书呢

![图片.jpeg](./_resources/10.11_到底是写日记还是看书呢.resources/图片.jpeg)
今晚很纠结哦，又想看书又想写日记。因为昨晚尝试了一下，冥想加指读法，效果不错，非常有利于精力集中，而且会看书看上瘾。

* 先推行，在总结改良。按照保守主义的哲学，推行计划。比如现在的，巡检后发软文，比如，地毯行动。我觉得，只需要先把能想到的该做的事情，先做到，然后过程中逐步改良。 无论结果如何，只要能做的流程都做了，都不会成为打酱油之旅的。 而且，附带收获肯定不少。
* 巡检到底该干嘛？

1. 我觉得应该，先和老板把关系氛围搞好了，然后再附带说一句“哦，对了，陈老板有个事情我还是要跟你强调一下，我们的展区内，不能摆非美的非厨卫的产品，这是做美的厨卫的基本诚意哈，我们出钱把形象给你装了，你还是要帮忙维护好塞，是吧，就当是配合我们工作，把样给上齐。你看这样子，我们是要直接扣代理商周阳的钱的哟。” 
2. 最好是进门了，找个点，先夸老板一下，“你们的门市还是有那么开阔哟”，“你们的卫生环境还是蛮干净的嘛”
3. 先就想到这些，后头想到了在改进嘛。诸如，统计库存啦，培训蒸汽洗啦之类的，则轻车熟路了。
4. 如果没有上蒸汽洗的门店，催促他们上样，承诺一年内一台都没有销售的话，直接予以换货。此种方式可以加速蒸汽洗上样，而且就算一台都没有销售出去，只能证明这个客户该换掉了。

* 换货协议的想法。

1. 突然想到一点，今天在贵福时，那老板说蒸汽洗贵，当我说出上样蒸汽洗，一年内都没有销售也换的承诺时，也把他诱惑到了，但我要求老板必须理解且会讲解才予以退换。
2. 其实不仅是蒸汽洗，要推出新产品的时候都可以这样玩儿呀，比如给一个“新品上样包退协议”，大致意思就是“自上样之日起，至次年今日，型号为（  ）的产品，在（  ）无一台销售，公司承诺补贴销售或者换货，  补贴销售的意思是要求老板以极低的价格销售出去，我们来补贴低于成本价的部分。比如，进价1500的型号，他们最后1000元卖了，我们来补贴亏损部分的那500元，并且上浮亏损额度的20％，即补贴500\*1.2＝600元，而换货的话，要扣除15％折旧，即1500元的进价，到期后需要换货，我们就直接换原进货金额85折的产品，也就是，如果换货，我们只能换价值1500\*0.85＝1275 的货品给老板。  而且要换货或者补贴的型号，必须要老板能清晰的讲出卖点才可以。”    嗯，好想法，波哥是人才。
