# 做出深信不疑的产品

麦克.马库拉告诉乔布斯”你永远都不该怀揣着赚钱的目的去创办一家公司，你的目标应该是做出让自己深信不疑的产品”
这点对我触动挺大的，我想我要打造售后型的代理商，也跟这思路异曲同工吧，要拿下大竹，需要的就是怀揣让客户无忧销售，能在当地有面子且轻松的依靠美的厨卫赚钱，这就需要价格、销售指导、和售后服务。

1. 乡镇门市重装，不，准确的说是升级装修就是个必要的销售指导。
2. 售后的核心就是配件，那一千块就拿来做售后扶持基金，比如乡镇客户需要售后帮助上门维修或指导，按照先有售后体制是不能刺激售后下乡镇帮助的，所以代理商出30乡镇客户出30元，让售后下乡镇有奔头。 进而提高售后质量。
