# 11-3 焦虑 不知道干啥

11-3 焦虑 不知道干啥
惠哥那个，我该准备些啥？
我为什么没有激情的感觉？
我每晚的会议，都是担心被评价，没有重要的结果输出。我该如何姿态端正地工作呢？
其实，每天，能去见见什么人，沟通沟通，对我蛮重要的。 这样会觉得有进步一些。
我真正的困扰是目前没有固定的职业和朋友圈子，所以依然是独自奋斗的感觉。
如果按照这个逻辑，我经常见见王丹这些合作商，是不是会好很多？
或者是去保理公司那些，谈谈合作之类的，应该在这个氛围里面，目前自己依然是独自慢慢在成长， 这个效率太低了。

我希望走出去，多见见人，聊聊合作，见识这个行业。这样会有抓住这个行业的感觉。

这个认识对吗？波哥？

至于为什么我没有激情，可能也跟这个有关。 总觉得，自己是一个人在生活，一个人在工作，并没有陪伴或者团队的感觉。 如果有一个团队的话，你会怎么感觉呢？

这种没有事情做的情况，该咋办？
没事情做，真的很慌。 今天能做什么嘛？ 虽然没做事，但一点都不好受。

但我现在去找个事情做，，就是为了抵抗妈妈没事情可做的焦虑感吗？ 波哥？
