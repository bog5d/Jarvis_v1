# 0616 关系还是复杂了点

1. 还是没有平衡好 有责任感了
	1. 纯付出型         怎么说呢，今晚绿影打电话来说她在文理学院，希望我去接一下，因为不一定好打车，而且有话跟我讲。我觉得出去的话会有种偷偷幽会的情绪氛围，不过内心里面又还是想去，所以挣扎了会儿，就还是开车去了。到了文理学院门口，她朋友春春就送她出来了，  然后她上车说，走去莲花湖转转。然后，就驱车去了莲花湖。我们在车里摆了摆龙门阵，聊得还是挺愉快的，不过我就是老想着要回家去，因为要和老高打电话的嘛。那种既想多聊聊，又想赶紧回去交任务的感觉挺着急的。
	2. 不过意境还是挺好的，在莲花湖旁边，她偶尔抽抽烟，我们在车里摆烦心事，那感觉挺那个啥，姑且称之为浪漫吧。我摆了张旭的事情，想证明自己当初品味真的不差呀。也很露骨的说了，本计划来一发的，结果还是觉得丑了点，就让她回去了，让她回去了就自个儿后悔了。。。。。她就狂吐槽我，说我没品味，就是丑。还有就是东扯一点，西扯一点的，大概九点多的时候，我就扭着要回去了，她说有一条路回去可以很近，但她不告诉我，让我能多陪陪她，我说那行，就绕着跑嘛。后来到了，他家`附`近，就在联通门口停下，又聊了聊。。。我提到了姚姐对我的心态，就是她女儿的事情。 我觉得说得还是有点多了。后来，确实急着想回来，就陪她去买了金橘柠檬汁，然后我就回了。
	3. 说整体感觉哈，挺享受的，很有那种羞涩芳香感，大概就是那种感觉，好像进去另外一个幻梦时空，分别的时候就又回到了正常世界，因为要去哄好老高嘛。    而且我提及到一点，就是老马的情人概念，只谈到那些积极正面的事情，不去谈琐碎的垃圾心情，尽可能展现美好的一面。我说我觉得我们现在的状态，就有点像这样，她说好像是哦。然后，又说哎呀，就把我当情人嘛，我说可以呀柏拉图的那种，她说柏拉图不是爱情吗，我说是指精神层面的。然后就去买柠檬汁了，不过在去买柠檬汁的途中，我感觉是挺不自在的，因为就像男女朋友一样的相处方式，让我很是拘束了，反正我表现也不太自然了，没话题的感觉，我也没有刻意去找话题，无话就无话吧，让她见识真实的我就行了。 然后分开的时候，我说我目送你吧，我不会送你进去的，不然我会感觉怪怪的。她说没事你走嘛！ 我就走了。   到家了的时候，犹豫，要不要给她发个信息，还是不发吧，感觉像男女朋友了，会让关系更复杂。
	4. 她是愿意付出的那一类女孩儿，看得出来是真关心我的感受，有时背后也会挺我，这是假不了的。不过，我因为这样耍了后，背上了，责任包袱就会更不自然了，我明显发现自己的心理变化，比如，想要对她好点啦、老高说想跟我一起住，我第一反应是没法多跟她相处会不会不太好啊、姚姐说要把她女儿带过来跟我学习会不会让绿影不自在啦。。。。。等等的考虑就进来了，这样会让我不自在，活得不像自己的。还是别因为今晚就把相处方式变得很大，不然和她更相处不久的，岂不是很失望，😜😜😜😜。好啦，继续保持淡定的领导角色嘛，别乱阵脚，
	5. 她也不是会欣赏那种扭捏作态的风格的人，如果我不想对她好，她宁愿我表现出来，也不会愿意我隐忍的，我不用去刻意1迎合她的，她会自觉调整姿态的，我相信她这点。
