# EverMemo

6-17随记

今日自问: 波哥，你希望接下来2-3年，从事美的厨电的运营事务吗?
你希望，去一个更大的城市，尝试着发展吗?

假如，你让你再回到2014年，你会选择一直留在达州吗?

还是，去成都闯一闯?

当时对于离开达州的恐惧是，自己没有存下钱，不足以支撑自己换工作，换岗位，担心自己万一回了成都，混得还更惨。  
以我现在的视角来看，王波你当时拥有的那些，根本就不多，没有多到不能失去。 回到成都的话，说实话，我也不知道能干啥，该干啥。  但要是回了成都，比其他同学差的话，自己的那种优越感就没有了。   

但，问题是，我当时的优越感是哪来的?  哦，对了，应该是做了管理岗，有助理，有经销商捧。  这些竟然成了我放不下的东西， 我也用这些来定义了自己。      如果早几年去成都的话，我猜自己的世界会更大。

还有，当时，也舍不得老马这个人生导师角色。  很担心，离开老妈后，自己的成长会停滞。这个，不好说，反正现在来看，我知道自己独立的情况下，也会尝试着成长。    不过老马，确实束缚了我。 

假如，要我穿越回那个时候，去告诉王波，该怎么做选择，怎么过人生。  可能我不会告诉王波，去成都发展吧，而是告诉王波，"bog，你拥有的并不算什么。你拥有的一切，都是失去得起的"

生活到底会糟糕成什么样子呢?  生活还会更糟糕吗?

我会感激老马，在我妄自菲薄时会给我一剂强心针。  不过，我也应该意识到，大部分时候，生活的难题，也是我自己在面对。

我最大的恐惧其实是自己不能在上海扎下根来，到时候还要跑回老马的圈子来。   这是我一定不能面对的事情，或者我会把这看成自己的失败。
我想这一次如果选择了飞出去的话，应该就不可能再飞回来了，是不能再回这个窝了。

独自飞翔的鹰，真帅!

如果，不能拿到融资，一定是不能去试的。   关键是，这样的话，我会有什么心态变化吗?
毕竟，裸辞是非常不靠谱的事情。   

怎么跟老马讲出口，是个大事儿。

和雨哥的感情，估计也是只有放下了。

我想分手了，为什么?
感觉到在关系里，也很孤独。  我俩好像互相都没走进对方内心。
精力不足，自己能分配在感情上的时间好像越来越少了。
性期待也越来越低。
还有，我想试试自己到底会不会离开重庆，去上海发展。  之前因为感情，总想着，不是时候。 我也想知道，自己到底是真的因为感情不愿离开重庆，还是拿感情做挡箭牌，来掩饰自己其实不敢放弃现在，拼搏未知的状态。

记到王波，你有权利结束一段你想要结束的关系，只要你愿意承担选择的后果。
