# 0706 为家庭相处而争吵（劝妈妈）

* 指责我家人自卑。
* 应该是接受我的好意沟通而不是一副我所有的付出就理所应当的样子。
* 我妈就是想一家人聚在一起，有什么错，你家就是这方面做得不够好，你还把这个当做人生信条。

今天又吵架了，我火气没这么大过，来来来，我来叙述一下经过嘛。
今天下午四点回家来，耍了狙击精英3这款游戏，然后因为疲惫就睡了，到了九点左右就接到哥哥电话，大致是讲了个二爸尝尿的笑话。然后问我最近蚀财是怎么回事，我就说了被偷了三千多的事情，然后又谈到今年春节可能要在老高家过一下。妈妈还是多不情愿的，抛出了两点看法：一，老爸会不同意，要吓我；二，感觉我就像上门女婿一样，她还是希望能让高莲来家里过个年呢。
  妈妈会有这些想法肯定是再正常不过的，毕竟一年多久才能聚得到一次，肯定希望通过春节好好的聚一聚，而且我们儿子在她们眼里是很自豪的，也想在春节期间展示展示家庭的兴盛嘛（也可以说成是自豪感嘛）。我非常理解！
当然我肯定也会说服母亲大人的，大致从这几个点来陈述：

1. 肯定不是上门这种说法，我会买房，已经没有谁上门这种社会概念了
2. 她家独生女儿，而且一直一来家庭氛围比较薄弱，我的加入如果能增进她家的家庭氛围我会很欣慰
3. 两亲家相处都不能太自私。你儿子跟你一起过年那是很美好的事情，但是如果为了给她家人情，你愿意暂时放弃一部分这样的美好来让她家增加点幸福指数，是我们的情意和尊重，这样也对增进两亲家和睦有帮助。
4. 我知道如果要从面子角度来看，其他人的看法肯定会是觉得我王波有点不男人👨，啥子都顺到女方，也会让妈妈你觉得不够直起腰杆，但是我不会去在意这些看法的，因为我后半生的生活不会跟村里人有太多交集接触，我只想做对自己最有利得事情，也希望妈妈多理解我。
5. 高莲她家对我确实比较好，把这种当成一种回报，也是比较合情合理的。因为她家从没要求我有车有房，而且在我需要帮助尤其是钱财方面的帮助时，都是不遗余力的在帮助我，我这种舍弃一部分妈妈你因为多和儿子相处得幸福来，让她家多一点开心😊，我相信她家会感激的。只是要委屈一下妈妈了哈。
6. 大致就以上五点，相信基本上能说服妈妈，而且以后每年轮流过年，今年在我这边，明年在她那边，也不会觉得很不公平的。

此外，我需要做的就是多回去和妈妈聚聚了，确实陪妈妈比较少，我在平日里多跟妈妈聚聚可以弥补一些因为没在家过年带来的失落。

哦，对了接到摆，我跟妈妈通完电话后，就跟老高联系了，我一提到二爸糖尿病，她第一反应就是“别找我们拿钱去医哈，我们没钱得”，我说只是摆个尝尿的笑话。（她这种反应总还是让人有点不太愉悦）
再然后，我就说我跟妈妈谈了春节可能要在温江过一次年，紧接着老高就多拽的问“他们囊个说嘛？”  我回答“妈妈觉得就像上门女婿一样，不好得。又说老爸会投梦吓我的”  
接下来，老高又开始一副大爷一样的说“听了你这么说我有两点感觉，第一你们家很自卑，第二。。。。。”
老子一听火🔥就上来了，我妈妈就是单纯地希望过年的时候两个儿子都在身边，一家团聚要幸福点。他妈的就成了又是自卑了，又是我有恋母情结怎样怎样了。
  我在积极的向家里沟通，我很坚定能跟你老高家一起过个年，只是做这件事时能最大限度照顾到妈妈情绪，你他妈就一副心理学家大圣人的批样子，挑剔这挑剔那的，我他妈欠你的吗？
我在做这件事的时候，如果你能感受到为了达成你心愿我积极的在做沟通，看到我的诚意，而且我也一定会沟通好，我只是觉得如果平日里能多跟妈妈他们聚聚，也可以让妈妈幸福感高一些呀，这种想法你又说老子是矫情，说老子恋母。滚你妹的！！
到底是我 尽信书还是你尽信书，看了点心理学相关的就拿起帽子乱扣一通，觉得自己看人看事就精准得很，幼稚！！
然后我们就气愤得挂掉了电话。

气归气，该沟通还是得沟通。
总的来说，老高也没坏心眼，就是过于自我了，看问题完全不去平衡的看，以自己感觉去定义别人的心态。我打算马上给她发一条信息，把态度还是表述清楚。

媳妇儿，我很生气，但必须把态度表达清楚，不然瞎斗气没意思。我觉得很不爽的点在于：我非常积极的在跟家人沟通今年初一在你家过，妈妈觉得不舍得，而想跟我扳一下。我肯定是会坚持我的做法的，我只是想在做这件事的时候最大限度的照顾我家人的情绪，你就说我家人自卑了，又说我恋母了，一堆罪名让我们身上扣，换了其他人是受不了你这种毒舌的。我的初步想法是平日里能多回成都和妈妈聚聚，就多聚聚，这样可以增加点妈妈的幸福感，然后这种想法你也反对，我是会很难受的。你知道不，我从田芳那儿把你接回来，一路上看风景的时候很开心，你看到我开心你就很欣慰，我能感受到。其实，每次我去看了妈妈回来，心情的开心程度比起那种是有过之而无不及的，如果你在我跟妈妈他们相处方面多做一些增进工作的，我是会多感激呀，而你目前的心态是排斥的，这让我真不好受。至于你挂了我电话，我会原谅你的，但真的多正面看待一下我对家人的那种感觉（肯定不是恋母！），你想多跟爸妈聚聚我是一万个支持的。太晚了，不打电话了，老公爱你。
