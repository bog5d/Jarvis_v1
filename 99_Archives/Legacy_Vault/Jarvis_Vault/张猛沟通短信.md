# 张猛沟通短信

猛哥，我厨卫王波。
本想跟你打电话，怕电话里表述不太清晰，就先给你信息吧。 
今年我希望你们旗舰店直接在渠县代理商那儿提货，理由有三：
1，售后服务，和安装维修，会更到位一些。 每一个县的厨卫售后，分了两个体系，一个是县代自己的售后人员，另一个是美的总部自建的售后网点。 县代自己的售后呢，缺点是没有美的自建网点那么严格执行标准化，也许在沟通言辞上没有那么高大上，但优点是很灵活，收费也不会乱来，说白了很多时候都是帮忙售后，能不收用户的钱，就不收。 而美的自建的售后，如渠县雍力，他的优点是，感觉上是专门做维修售后的，要正规些（不过雍力实际上也不是很正规，相比其他区域专业网点）， 而他的缺点是，收费很贵，毕竟他是依靠售后维修过日子，通常都是能不收钱的，他也要收，收了用户的钱，照样去美的报费用，吃两边的钱。 更重要的是，旗舰店只是他的一个小客户群，因此也不会特别用心帮助旗舰店，（大竹和宣汉旗舰店，都是单独给售后补贴200），所以如果使用美的自建的售后网点，不会那么流畅，所以，我建议跟县代合作，借用县代的售后人员，更加灵活，还可以追责。
2，价格。 旗舰店价格，一直都是常规价，在公司层面这是严格管控的。而美的厨卫是两套价格体系，政策价和常规价。每年3月，8月，开盘时，会有重大政策放出，而这些政策价只有县代能享受。举个例子，烟机，常规价假如1000，而在三月份，二十台起提可能会是860元。    假如县代880元提回来，发给你930元，你们都是受益的。 也会大大降低你们的成本。  而在公司拿货的话，这点是做不到，因为三月份八月份，你们是没办法打款三五十万到厨卫来吃政策的，何况公司也不允许给除县代以外的客户放政策。这是建议你和县代合作的第二个理由。

3，任务压力。
厨卫分为厨房电器，和热水器，两个品类。两个品类的补贴好像是一年四万。 但是有要求，完成年任务25万提货，还有报单率的考核。   而代理商提货量比较大，可以用他们的提货数据来填补你们的25万。 说简单点，假如你在代理商那儿，哪怕只完成了18万提货，但是代理商提了240万，我们直接把他的240万抠25出来，做到你们账户上。你们只需要卖好多，提好多，也能安稳领补贴，任务压力转嫁到了县代身上。 
  当然，这样操作，感觉上不是很有面子，毕竟你们旗舰店的身家是远高于厨卫县代的，就像是公司老总，和路边摆摊老板之间的合作。但是，路边摆摊的小老板更愿意跑腿一些，能服务得更好些。 你考虑一下嘛，如果愿意选择的话，我就安排你们旗舰店和渠县县代，一起见个面，把合作谈深入一些。县代那边是很希望，能和你们合作的，就看你们的态度了。  
请及时回复你们的态度哦，等你消息。
