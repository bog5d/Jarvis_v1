# 吃多了 肚子不饿就不吃嘛

我的妈呀，现在凌晨四点四十三，昨晚肚子异常不舒服。

三点半的时候就难受得起来上了厕所，然后又回到床上。  躺了一会儿就又遭不住，赶紧再去了一趟厕所

但是蹲着特别难受，同时呢，肚子里面还是不舒服，又拉不出来。我就想了想，肯定是因为蹲得难受，影响了排便效率，我就找了根凳子，四角方凳，然后半坐在凳子上，然后舒服的放炮。

这事情啊，真得总结一下：

1. 昨晚吃饭的时候明明肚子一点不饿，大家都吃，我就觉得该吃，何况平日晚饭吃得挺少的，基本上是凑合着吃点，那我干嘛一定要跟着别人吃啊？根据自己情况来决定吃不吃，不行啊？以后晚饭真的要清淡，不饿就喝点汤水。 
2. 你想啊，本来晚饭就是不该存在的，只不过从小到大的习惯形成反射了，所以晚饭适度摄入饮食，清淡易消化为主，才是王道。 
3. 此外，抗诱惑能力太重要了。我看到满座子的菜，有狗肉啦，烤鸭啦，腊肉呀，都想去尝试一口，吃了一口又发现，咦  味道不错。再来几口，哦，饭上来了，那就盛一碗吧，唉肚子实在有点撑，那就赶紧把碗里的饭吃完吧，然后本来不饿的人，就吃了大口大口的肉，还加一碗饭。其实完全可以不吃饭，吃点菜就行了。
4. 吃狗肉不能喝茶，狗肉和茶会产生化学反应，形成的组织直接影响肠道的蠕动能力，完成消化不良。这个很重要， 因为在狗肉中，含有丰富的蛋白质，而茶叶中含有比较多的鞣酸，如果吃完狗肉后马上喝茶，会使茶叶中的鞣酸与狗肉中的蛋白质结合，生成一种叫“鞣酸蛋白质”的物质。这种物质，可使肠蠕动减弱，大便里的水分减少。因此，大便中的有毒物和致癌物质，就会在肠内停留时间过长而极易被人体吸收，对人体健康不利。   会议室的饮水是统一的茶水，我也发现了自己的大便比较干，看来吃了狗肉后喝茶，是很大的原因。
