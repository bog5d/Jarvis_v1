# 海南自贸港私募投资税收优惠十问

# 海南自贸港私募投资税收优惠十问

原创 张焱 亚太金融小镇 _今天_

![CFNetworkDownload_3ozVn5.tmp.png](./_resources/海南自贸港私募投资税收优惠十问.resources/CFNetworkDownload_3ozVn5.tmp.png)

**1.双十五的优惠政策是什么，私募基金涉及的主要税种目前有哪些？**

企业所得税15%和个人所得税15%，私募基金涉及的主要税种有企业所得税、个人综合所得税及个人经营所得税三种。什么基金员工的综合所得(包括工资薪金、劳务报酬等)及合伙企业LP的经营所得（投资企业股权转让所得）都能享受15%的优惠。另外，如果是普通的有限合伙企业，其经营所得也能享受15%的优惠。

**2.个人所得税优惠的前提条件是什么，最低社保是多少，个税如何缴纳？**

个人所得税要求个人与海南公司签署一年以上的劳动协议并且连续缴纳社会保险6个月以上（须包含本年度12月当月），最新社保基数是3669.6元，需要1310.06元，其中单位924.75元，个人385.31元，最高是最低的5倍。入驻企业的员工个税在海南自贸港单独汇算清缴，经营所得于次年1月1日至3月31日、综合所得于次年4月1日至6月30日在海南办理个人所得税年度退税。

**3.企业所得税优惠的前提条件是什么，优惠的时间是什么？**

根据三个鼓励类目录，目前创投类企业享受15%的企业所得税优惠，由税务局根据发改委认定，为在中国证券投资基金业协会备案为创投企业及在海南省发改委被备案为创投企业两种，企业所得税按季度申报10%减免后实现15%。

**4.扶持政策的有效期限是多长时间？**

根据财政部、税务总局《关于海南自由贸易港高端紧缺人才个人所得税政策的通知》及《关于海南自由贸易港企业所得税优惠政策的通知》，海南自贸港双十五的优惠政策执行至2024年12月31日，届时应该还会延长期限。

**5.全岛都能享受15%优惠吗，注册在小镇有什么优势，财政奖补何时到位？**

自贸港的双十五全岛都能享受，注册在三亚的备案基金或备案的创投可以享受地方财政奖补，到2021年底能享受三亚地方所得留存28%部分全额奖补，2022年的政策正在拟定，奖补资金在税收确认的次月兑现到纳税人指定账户。

**6.对于私募管理公司，选择何种类型能够享受最佳自贸港政策优惠？**

目前有限合伙类的企业也可以在中基协申请为管理公司（包括股权私募基金和证券私募基金都可），获得的超额投资收益（carry）可以直接分配给有限合伙的个人，享受15%的经营所得优惠。

**7.设立企业需要支付何种费用？收费标准是怎样的？**

入驻小镇五类公司，1/2类公司为拥有独立4个工位的共享办公，第一年开办费（包括设立、服务、代账代税）为39,800元，第二年每年续费（包括服务及代账代税）为9,000元/年，3/4/5公司为拥有1个工位的共享办公，第一年开办费（包括设立、服务、代账代税）为9,800元，第二年每年续费（包括服务及代账代税）为6,000元/年，其他变更费、调档费等见第三部分收费标准。

**8.办理企业设立的全部流程需要多长时间？**

办理1/2类基金管理公司和3类基金合伙产品需要金融局审核，办理流程为两周以上，办理4/5类公司需要一周以上，包括营业执照、四章（法人章、公章、财务章、发票章）、银行基本户开户、网银、税务登记等。另外，对于入驻小镇企业的基本户，必须在三亚的银行开立，企业可以自行选择。

**9.入驻企业设立的合规要求？**

对于所有入驻小镇企业来说，非持股平台类企业，注册资本或认缴资金不低于1000万元，3类基金类企业单个自然人出资额不低于100万元，4/5类普通投资类单个自然人出资额不低于50万元，机构出资额不低于1万元；持股平台类企业，注册资金或认缴资金不低于100万元，股东、员工、高管出资额不低于1万元。

**10.QFLP和QDLP都已经实施了吗，针对哪一类企业，请问如何办理？**

QFLP（合格境外有限合伙人）2020年10月实施，QDLP（合格境内有限合伙人）2021年4月实施，QFLP没有限额，但仅针对中基协备案股权基金，QDLP海南有50亿美金限额，可以是备案股权基金或证券基金，可以通过小镇办理。
