# 区块链和加密 打麻将比喻分布式记账，妙喻

打麻将比喻分布式记账，妙喻

区块链和加密

对于普通人来说，加密的概念比较容易理解。加密就是把一串信息转为密码。而区块链则更加难以理解。

我们可以把区块链理解成为一串记录。就例如说我们打麻将的时候，因为现金来往很麻烦，我们一般都是叫一个人来记账，那么这就是所谓的“中心化记录方式”。但是这中间会产生一个问题：这个人如果偷偷给自己多记一笔钱，我们可能谁也发现不了。

区块链则是一种“去中心化记录”，也就是说，我们怎么样解决记账的人给自己偷偷多记一笔钱这种事情？我们打麻将的四个人，每个人都来记账。打完了八圈我们对账，如果三个人的帐一样，一个人的不一样，我们就根据简单多数的原则，认为三个人的帐就是正确的，把那个不一样的账本排除出去。然后我们把不同时间点的账本钉在一起，这就是区块链。
