# Day One Entry: 2015年3月28日

今晚和王斌冷瑞聊天，感受他们的成长。

白日里，有一个模糊的观点正在产生，如果不拥有太多的欲望，才能真正自由。怎么说呢，比如吃饭，如果随便吃一点，保证基本的饮食能量，却也能安然自得，而不是，没有吃到肉就怅然若失，并且配上多喝水的习惯，会更健康却更省钱，这样就没有因为肉不自由。衣服能穿得洁净朴素，因为不是好衣服而更加注意打理，使之整洁，可能一改了从前的邋遢形象，却也更干净干练了，而且也很省钱，相比起买了不少衣服，却因为太多而不去洗，也没法放好而混乱，这种情况，朴素作风效果远大于后者。 出行，能走路就走路，有公交车就坐公交，却不因为没打到车或拥堵而费时费钱等等，简朴出行却更适合自己内心的自由，等等想法，在脑中盘旋。我觉得，最低的消耗最大的产出，越来越是我所追求的。 不过，我知道自己，想过简朴的生活，并不是那么高尚得想要当圣人，而是我觉得这样的生活会更快的积累积蓄，进而更快的超越周边的类似人群比如同学，同事。 所以，动机也是出于一种竞争意识，并不显得高尚，倒是有更多的投机成分。不过就算动机出于竞争，只要我是开心去做这些的，那就很值得。

还有，如果90%的人都对那些钱财名利特别在意向外，如果自己在钱名方面“性冷淡”的话，就会更愿意分享钱名，更容易获得支持与盟友（不是指朋友哦），这也是一种竞争力。 因此早一点过朴素的生活，早一点对“钱名性冷淡”，才是王道，才是吾心所求，但愿我是对的。

今天，晚上和王斌冷瑞在客厅聊天，我忽然发现他们两个的进步，都已经很多了。比如，王斌在谈事情和沟通方面，已经有基本的评估能力和判断力，知道做最有利或最小损失的决策。 冷瑞，其实是一个自己一个人也能开心起来的人，所以在心情能力方面，是很强大的，也就是说耐得住寂寞孤独，精神层面的意志是强大的，对于恋爱，并没有特别的渴望或者依赖，这一点比起，王斌对王海梦的控制依赖，就要强大很多，我是很欣赏的。
以后，千万不要高估自己在一个群体中的作用，很多事情，王斌冷瑞刘建，是有自己的看法和办法的，比如刘建对于一个号码挂多张单子，她认为也对，但是大竹的实际行情是更真实，不用刻意，比如昨天王斌，想要两千元把事情给了了，因为可以不用再花费太多时间和精力，今天看来他当时的心态没有错，是最优解决方案。 以后，遇到我认为的问题时，先抛出问题，然后听他们意见，别盲目提出自己的解决方案，如果实在没有给出办法了，就采取自己的基本方案。 以后的发展方向，还是，发现问题，提出问题，听别人的表述和解决方案，再决定采用的解决方案。 嗯，我同意自己的观点，加油波波，当领导的就是找到问题，提对问题，下属们的价值就是解决自己的这些问题的。
发展问题，提对问题，听，一直听。
