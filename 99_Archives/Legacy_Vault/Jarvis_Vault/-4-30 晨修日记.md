| 一行禅师 | 回归当下的本质，是在修“止”。 回归当下，放下对过往将来的执念，便是修“止” | 自己尝试冥想，有两个感悟： 一 放下对好的体验的执念，曾以为冥想就是找到平和状态。 但冥想是观自己，暴躁时冥想，就观自己暴躁；宁静时观自己宁静，冥想的本质是观察，而不是追求。<br>二，冥想 就是一种“止”，闭上眼睛，会切断大量的信息输入，以及想干这干哪都年头。 开始冥想脑中虽思绪万千，但都是在处理以前挤压的思绪，并没有新的思绪再进来，我称之为消化积食。  停止摄入新的，消化积压的万千执念和思绪。 |
| 熊逸  | *  我们对佛教的理解，其实并非真正的佛教，而更接近婆罗门教的教义，基本逻辑是：轮回转世、善恶报应、烧香上供、天堂地狱等。   真正的佛教是寻求醒觉知道，跳出轮回，寻涅槃，<br>* 革囊众秽，这是佛教的不净观。 看到美女，控制不住欲望，要把人理解为一个包装袋 套着一堆血杂碎。  这是一种需要修炼的观世界方法，人是不干净的，从出生位置，到因淫欲而生都是不净的，想到这些之后，还会对粉红窟窿有欲望吗？<br>* 佛教观念会绝后吗？会让人类灭绝吗？  儒家一直批判佛教会让人类灭绝。  而佛教根本不care人类灭绝，就像大家都是监狱囚犯，有人努力表现，争取减刑，然后大家一个个地都开始减刑出狱了，这时突然跳出来说，“不好啦，这样下去监狱会空的”，在佛教看来，在六道轮回是苦海无边，只有跳出轮回，不在这个游戏机制里玩，才是解脱。 |     |
| 刘润  | 概念：认知协调； 无知之幕；多数人的暴政。<br>刘润以这几个概念解释清楚了为什么“疫情刚开始，大部人同意如果邻居阳性，就支持邻居居家隔离，自己不会驱逐邻居； 但小区真有阳性了，阴性邻居开始网暴驱逐阳性邻居”<br><br>1. 多数人暴政和无知之幕，如果富人和穷人一起投票，要不要瓜分富人财产，因为穷人多，那投票结果一定是支持瓜分富人财产，这叫多数人的暴政。  但让这些人走到一张无知幕布后面，这幕布会让人失忆，忘记自己的身份，不知道自己是富人还是穷人，那再来投票，因为每个人担心万一自己是富人呢，就不会想着瓜分富人。   这就是无知之幕，本质是说清楚每个人的观点/决策都根植于自己的利益和立场，而不是追求正确。 <br>2. 认知协调，当自己的观点和利益立场不一致时，我们会修正自己的观点，以支持自己的利益，立场。  疫情期间，那些投票支持“邻居阳性，自己不驱逐”的阴性居民，发现，邻居阳性了，如果不去方舱，自己小区的解封会延后，于是心里盘算，还是把阴性邻居赶走吧，但自己之前自己明明支持居家隔离的呀。 这种时候，阴性邻居会想方设法证明阳性邻居是“坏人”，这种时候比福尔摩斯还能破案，比如在小区群说阳性邻居平时不戴口罩，出去溜达，知道阳性还不戴口罩等方式证明别人是坏人，然后驱逐阳性邻居。   这种把观点修正来支持自己利益的过程，就叫认知协调。 | * 不要高估自己观点的客观性，我们绝大多数的观点都是基于立场和利益。 <br>* 听别人观点时，不要看对错，而要看对方立场和利益点。  <br>* 成年人世界，没有对错，只有利弊。 |
| 卡罗尔.塔夫里斯《社会性动物作者》 | 我们自以为不容易被广告诱导，产生消费。 认为自己能看破广告对我们的忽悠，但真相是，我们的行为极大地受广告驱使。  这里面的原理是：制造熟悉感。  我们不会在看到广告后，马上跳起来去买他。 但未来我们做选择时，会优先选他。 试想一下：如果我们要去选水杯，你去超市有10个品牌，你在没有任何水杯广告信息摄入的前提下，每个品牌被你选的概率是1/10，但你出门前在电视里看过一个叫thomos的极限运动水杯，专为极限爱好者量身打造。 那好，你选择这个品牌水杯的概率就会上升到50%以上，如果价格也在你的预算内，那大概率就是他了。 | 有两点启发： 品牌影响的着眼点，压根儿就是现在，而是埋下一个种子，成为客户未来决策的优先选项。<br><br>2 制造熟悉感，才是本质。 我们做商票业务，或者无论做什么业务，增加和客户的见面次数，制造熟悉感，是方法论。 |
