# 关于吃荤和吃素的对比研究

阅读：3852012-08-29 16:14
一直听到我周围的朋友们关于吃荤和吃素的争论，也一直看到网络上、各种报章杂志上关于吃荤和吃素的各种理由的宣讲，我也不确定到底吃荤还是吃素好，因为真的是“公说公有理，婆说婆有理”。
听别人说的毕竟是道听途说，我干脆以自己的身体为研究对象进行了吃荤和吃素的对比研究，自己体验吃荤好还是吃素好。
我的吃素研究时段有两段，第一段是去年8月1日到去年8月31日，整整一个月，未使用任何荤食，均为吃素。今年8月5日到8月12日，也均为素食，未食用任何荤食。期间，8月11日断食一天，喝了一点果汁，有梨汁、西瓜汁等。其他时间则吃荤。
就我个人而言，吃荤和吃素的身体感觉差别非常明显，遗憾，好像没有插入表格功能，所以只能写写，而不能进行直观的表格对比了。
吃素后的感觉：1、身体感觉特别干净
2、身体没有火重的感觉
3、便便正常，且较成形，没有恶臭的异味
4、心里很宁静，感觉也很安静
5、身体柔韧性较好，不容易被拉伤
6、早起没有身体凝重的感觉
7、我腿部经检测有轻微的血管阻滞，吃素后小腿轻微肿胀感觉明显减轻
8、特别断食后，感觉神清气爽，体能并未如常人想象的那样下降，基本保持。
吃荤后则与这些感觉向相反的方向发展，其他的则感觉不变。
暂时就说这么多，我会不断地完善这种比较。
其他地方也有很多关于吃素的宣传，其中他们说的最重要的一条就是吃荤很血腥。其实，对这一条我是持有异议的。
在我心目中，生命都是一样的，动物的生命不见得就应该比植物的生命更高贵，吃植物就不见得不血腥。我们吃的植物一样的是生命。但没办法，借用一句生态系统的专业术语来说“生态系统中本来就是吃和被吃的关系。”我们要维持我们的生命，我们必须要吃。
如果说要真的仁慈，只有不浪费才是真的仁慈。
