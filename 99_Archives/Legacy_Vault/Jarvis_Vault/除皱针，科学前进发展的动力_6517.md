# 除皱针，科学前进发展的动力

最近看了几篇文章，深深觉得自己思想还是不够解放，脑洞还是不够大－－大概可能也许就是一个小破实验设计了这么久都没有尘埃落定的原因吧。今天给大家介绍一篇文章，希望能给也在为实验苦苦挣扎的同胞们带来一些灵感。

要说做实验最”敢搞“的，就个人经验来看，国外的科学家里看欧洲，欧洲看德国。各种人体试验，神马打针吃药、在美帝绝对不过审的manipulation（zhei词儿怎么翻译？），人家搞起来一样一样的。比如，肉毒素大家都不陌生吧？Botox，长一点的名字是Botulinum toxin，几年前还觉得是个高风险的美容产品，现在貌似打一针就像喝水一样平常。它在注射入肌肉后通过阻碍神经递质acetylcholine的释放来麻痹肌肉（神经递质就像是大脑的messenger，阻碍它们在肌肉里的释放相当于让这坨肌肉暂时不再听大脑命令了），注射后的肌肉呈绷紧状态，动弹不得，除皱效果就算达到了。肌肉麻痹是什么意思呢，就相当于所谓的被困在”潜水钟“里，你拼了命想动，但是就是动不了。这就是为啥打了针的阿姨们大都成了半面瘫状，表情用五个指头都能数完。在看到下面这篇文章之前，本狮完全想不到，这么接地气的产品也能为科学做出贡献！

好了，终于进入正题。这是一篇2008年发表在Cerebral Cortex上的文章，由一组来自大名鼎鼎的Max Planck Institut的德国科学家研制（研发？。。开发。。？语文老师。。我对不起您。。）。Cerebral Cortex是认知神经学的顶级期刊之一，so，no no no ，人家完全不是野鸡研究来的。这组人十分聪明地将botox的肌肉麻痹作用与情绪反馈的机制联系起来。很多年前，心理学面临的一个重要问题就是，我们到底是先有身体反应，大脑通过解读这些生理变化构建了情绪的感知（我心跳加速了，我一定是喜欢TA），还是我们先有了情绪感知，再才有生理变化（我喜欢TA，所以我才脸红心跳），还是两者就是同时发生的。

其中有一个研究广为流传。如下图所示，让一组人用嘴唇夹住铅笔，另一组人用牙齿夹住铅笔，然后让他们看同样的喜剧片，你会发现用牙齿夹住铅笔的人普遍觉得喜剧片更好笑。神马意思呢？你的身体帮助你做出了情绪判断。这里插播小清新片里经常有的桥段：没心没肺的女猪脚在男猪脚转身离开的瞬间流下眼泪，她难以置信伸手触摸晶莹的泪珠，然后伤心欲绝地嚎啕大哭起来，才骤然醒悟自己已经不可救药地爱上了他。（让脑洞开得再大些吧！！）

大概就是这样=\]。聪明的读者朋友们（这是什么鬼附体的表达。。我上一本看的中文书难道是十万个为神马吗。）看到这里一定明白肉毒在这个研究里面是什么角色了。想象你现在非常生气，非常非常生气，生气到让你想做出世界上最生气的表情－－皱眉头。但是你今早刚去了美容院，额头上挨了一针，眉毛暂时不大能动。那你还能气得起来吗？

这组德国的研究人员让额头上被注射了肉毒被试模仿生气或者伤心的表情，果然打了针之后额头上啥子皱纹，哦不，表情，都显现不出来（见下图）。（早知道这样就应该去欧洲！！把我妈带上！！不仅美容的开销省了还能顺便赚着！！）

最上面一排：没有注射肉毒的被试。中间一排：研究员提供的照片模版。下面一排：注射了肉毒的被试。Hennenlotter et al., 2008.

然后他们发现，打了针的被试杏仁核活跃程度降低（下图左），而且杏仁核与脑干上一个区域（下图右）的功能性连接的程度下降。杏仁核可以简单粗暴地理解为情绪反应的登记处，脑干则帮助调节自主神经系统，包括呼吸、心跳等一系列情绪反应。

遗憾的是，他们并没有让被试直接汇报他们的情绪状态，因此这些结果只能说明生理反馈的缺失能直接导致处理情绪的相关脑区的活跃程度发生变化，但我们并不知道这些变化的行为后果。然而后来的实验表明，缺少生理反溃不仅能导致情绪感知的下降（简单粗暴理解为不能皱眉头的时候也不大生气了），还能使你的杏仁核对别人表情的反应下降，但是这些变化都会随着肉毒效果的消退而还原（Kim et al., 2014 )。（所以说，身边有打了针的朋友，抓紧机会欺负TA啊！药效没了就来不及了！！）

另外一个可能会让你有点意外的结果是，打针对pre-, sup-, motor cortex都没有影响。这些地方就是大脑直接输出命令的地方。（你，动一下。看哪儿呢，就你，内坨长额头上的肌肉！）军令是照样发，只是小兵的耳朵被堵住了。。。其实很多植物人的状态也是这样，他们的大脑对外界刺激有着和正常人相当的反应，只是失去了对肢体的控制。这个咱们改天再聊。

所以，最后的最后，好像用了肉毒的研究也没有什么三观颠覆的成果－－只是说明了没有蛋也不大可能有鸡，但在有鸡又有蛋的时候到底是先有鸡还是先有蛋？本狮已经糊涂了。
