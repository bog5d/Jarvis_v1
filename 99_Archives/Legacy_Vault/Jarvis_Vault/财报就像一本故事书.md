# 财报就像一本故事书

这是一本读起来最不顺溜的书，一章看完后，不记得这一章的主要内容，不过里面举的公司案例，倒是能看懂。
那说正事儿，这次就不说三点收获了，是学到的一个点，然后尝试运用。

* `如何判断合理的股价—每股收益和市盈率`

作者文在讲“每股收益迷思”时，想要强调不要过分迷恋每股收益这个指标。
  常规来讲，每股收益越高代表公司经营得越不错。 但是，有时候公司为了配发股票股利，或者做股票拆分，使每股收益下降，这是公司为了增加股票流通性，而不是盈利能力下降。 
然后作者顺带一提的，一段话，让我想要研究一下。 

关于如何判断合理股价的一个方式：
在投资决策上，每股收益常与“市盈率”（price-earning ratio，PE ratio，指股价与每股收益的比率）搭配运用，以决定合理的股价。例如预期的每股收益为4元，而合理的市盈率为15倍，则合理的股价为每股60元（4 × 15）。只有在股价显著低于60元时，理性的投资人才愿意购买股票。巴菲特强调要拿捏至少25%的“安全边际”（margin ofsafety），也就是说，合理股价为60元的股票，只有在市场股价为45元（60 × 0.75）以下时才值得投资，以抵御可能因误判投资价值造成的风险。

基于这个知识点，让我去判断一家公司的股票价格高不高，我发现压根儿没办法操作，基本常识太缺了。 
比如 PE怎么来的，查询之后发现是股价除以每股盈利；
![unknown_filename.png](./_resources/财报就像一本故事书.resources/unknown_filename.png)

那么每股盈利又是怎么算的？
每股收益=（净收入 - 优先股股息）/流通股数量
也就是公司利润除以流通总股数。

我试着去翻老东家美的19年财报算算每股收益：
![2C63B29A-79DF-41FD-BEF6-9122A98FF025.png](./_resources/财报就像一本故事书.resources/2C63B29A-79DF-41FD-BEF6-9122A98FF025.png)

![819AD51E-92D3-4C97-ADFC-FC3E8CB5BE33.png](./_resources/财报就像一本故事书.resources/819AD51E-92D3-4C97-ADFC-FC3E8CB5BE33.png)

股份总数：69.7亿股；
总收益：232亿元；

如果按这个计算美的19年的EPS （每股收益）应该是 232/69.7 = 3.32左右呢。

![E1C25562-3171-4B17-97E8-3D0E52E64B6A.png](./_resources/财报就像一本故事书.resources/E1C25562-3171-4B17-97E8-3D0E52E64B6A.png)

我查询到结果是： 3.38元。  感觉方向对了。   误差应该是优先股息的没有扣除吧。 

* * *

有了EPS，接下来算市盈率PE = 股价/EPS 
![04C8A6D4-2BA9-4E09-9DF6-64B1D3AD0284.png](./_resources/财报就像一本故事书.resources/04C8A6D4-2BA9-4E09-9DF6-64B1D3AD0284.png)

58.25/3.38 = 17.23 ，即市盈率PE是17倍左右。  
![4AFDDDA0-CF28-4F90-B22C-903C481ED994.png](./_resources/财报就像一本故事书.resources/4AFDDDA0-CF28-4F90-B22C-903C481ED994.png)

查出来的公开数据是16.9.  

那么接下来，该怎么测算股票价格的合理性呢？

还缺的一个变量，应该就是预期收益。   奇特的是，竟然预期收益，也有数据可查。 
![A2C7CCB8-4E2B-4C98-9164-141E1254E9F0.png](./_resources/财报就像一本故事书.resources/A2C7CCB8-4E2B-4C98-9164-141E1254E9F0.png)

那么合理股价应该是预期收益乘以市盈率 = 3.66\*16.9 = 61.854
根据巴菲特的25%安全边际考虑的话，只有在股价低于61.854\*75% = 46.39时，才值得投资，而当时美的股价是59.75，性价比不高。  

好吧，验证一个市盈PE率和每股收益EPS，花费力气不少，所以炒股这事儿，真的要交给专业人士做。

找到另一个财报研究的点，再根据概念做研究。

总市值/净利润  跟那个是一样的

二级市场股票的算法 没太大意义。 只算一个pe是远远不够的。  
建设性的建议： 创业者是需要学习PE的，PS

ps，市销率， 销售收入， ps是五倍，则按照 年销售\*五倍 就是估值。

行业有非常强的属性，半导体和生物 芯片，有100倍pe，  智能制造 信息科技，20左右
龙头股 和小盘股的pe不一样，龙头一般在20倍以下，小的企业一般比较大一些，小企业成长性高一些， 

重要的概念 peg： pe：增速

Pe：g 分母再乘以100，比如17倍，和20%增速的企业，要算的话，就用17/20 = 0.85。 就是peg
