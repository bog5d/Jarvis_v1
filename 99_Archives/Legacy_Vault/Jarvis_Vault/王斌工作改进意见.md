# 王斌工作改进意见

如果我给你提工作改进意见我不知道你是什么态度，
不过我个人觉得，每次你要下乡镇之前，需要早起来门市，然后开始把要的货再从库房搬上车，再电话沟通是否要其他的货，基本上能出发的话都快接近10点了。
如果你愿意改进一下的话，比如在头一天晚上就电话告诉客户，我明天要下来，你需要带些啥子？ 然后把货搬上车，就算不搬上车先放门市门口，等第二天来的话都可以节省一个小时，基本上能在九点左右出发了。晚上回来肯定比较早。  如果回来的早，再把下一天要去的准备好。就良性很多了。
  能改进就改进嘛！
