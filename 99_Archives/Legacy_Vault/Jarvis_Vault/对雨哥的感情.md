# 对雨哥的感情

自己不知道怎么回事，对于和雨哥的感情，我有点模糊，自己拿不太准。
不知道，我算不算，她进了一步，我却退了一步。  至少我自己是这样感受的。   
反正，没有这段关系呢，我铁定会失落。   但，到了这个年纪了，这些失落，好像也能承受。

不过我更担心的则是：我有没有把别人胃口吊起。 到底是我自己吊了别人胃口，还是人家高冷了，我受不了。 

拿不准。 

反正，第一，她是喜欢我的，这一点我很明确。  
第二，我觉得自己并没准备好。  当下快速回血，增加自己的资产，好像才是当务之急。
