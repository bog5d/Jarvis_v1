# 紫薇斗数是什么？

**紫微斗数**以[星宿](http://zh.wikipedia.org/wiki/%E4%BA%8C%E5%8D%81%E5%85%AB%E5%AE%BF)配合十二宫的[术数](http://zh.wikipedia.org/wiki/%E8%A1%93%E6%95%B8)[算命](http://zh.wikipedia.org/wiki/%E7%AE%97%E5%91%BD)方法，是一种[星相](http://zh.wikipedia.org/wiki/%E6%98%9F%E7%9B%B8)术，亦是[中国](http://zh.wikipedia.org/wiki/%E4%B8%AD%E5%9C%8B)传统[相术](http://zh.wikipedia.org/wiki/%E7%9B%B8%E8%A1%93)中的一支；因其系统里以[紫微星](http://zh.wikipedia.org/wiki/%E7%B4%AB%E5%BE%AE%E5%9E%A3)为诸星之首，故得名。此术认为人出生时的星相决定人的一生，即人的[命运](http://zh.wikipedia.org/wiki/%E5%91%BD%E9%81%8B)；认为各种按一定次序出现的[星曜](http://zh.wikipedia.org/wiki/%E6%98%9F%E5%AE%BF)对相对应的人的命运具有特定的影响，因而分析人出生时的星相就可以判断人本身命运的好坏和时间顺序。

历史\[[编辑](http://zh.wikipedia.org/w/index.php?title=%E7%B4%AB%E5%BE%AE%E6%96%97%E6%95%B0&action=edit&section=1)\]

在[汉](http://zh.wikipedia.org/wiki/%E6%BC%A2)[唐](http://zh.wikipedia.org/wiki/%E5%94%90)之间的[道经](http://zh.wikipedia.org/wiki/%E9%81%93%E7%BB%8F)、[北斗经](http://zh.wikipedia.org/w/index.php?title=%E5%8C%97%E6%96%97%E7%B6%93&action=edit&redlink=1)、[南斗经](http://zh.wikipedia.org/w/index.php?title=%E5%8D%97%E6%96%97%E7%B6%93&action=edit&redlink=1)，与[大藏经](http://zh.wikipedia.org/wiki/%E5%A4%A7%E8%97%8F%E7%BB%8F)中[汉](http://zh.wikipedia.org/wiki/%E6%BC%A2)[唐](http://zh.wikipedia.org/wiki/%E5%94%90)时期传来的“[佛说北斗七星延命经](http://zh.wikipedia.org/w/index.php?title=%E4%BD%9B%E8%AA%AA%E5%8C%97%E6%96%97%E4%B8%83%E6%98%9F%E5%BB%B6%E5%91%BD%E7%B6%93&action=edit&redlink=1)”，就有详尽记载紫微斗数[命主](http://zh.wikipedia.org/w/index.php?title=%E5%91%BD%E4%B8%BB&action=edit&redlink=1)星与[身主](http://zh.wikipedia.org/w/index.php?title=%E8%BA%AB%E4%B8%BB&action=edit&redlink=1)星的排列方法，命主星身主星正是攸关人生劫数关卡的重要关键，因而据以有了道教的“祈安礼斗”[祈禳科仪](http://zh.wikipedia.org/w/index.php?title=%E7%A5%88%E7%A6%B3%E7%A7%91%E5%84%80&action=edit&redlink=1)。

而且在北斗经中很正式的提述有关紫微斗数“十二宫”、“辅助煞星”、以及紫微斗数各种凶恶“格局”所引发种种的命运灾劫；这时候，北斗经南斗经的正式经文与紫微斗数的关连已经甚为紧密；而如果当时经文注解之“经注”能有流传至今，相信会有更详尽的紫微斗数要诀。从14主星以“紫微”为主导，而渊源于南北“斗”经，所以后世称为紫微斗数，实有其渊源意涵。

**紫微斗数**之名则首见于[明神宗](http://zh.wikipedia.org/wiki/%E6%98%8E%E7%A5%9E%E5%AE%97)[万历](http://zh.wikipedia.org/wiki/%E8%90%AC%E6%9B%86)35年（1607年）由第五十代[正一道](http://zh.wikipedia.org/wiki/%E6%AD%A3%E4%B8%80%E9%81%93)[天师](http://zh.wikipedia.org/wiki/%E5%A4%A9%E5%B8%AB)[张国祥](http://zh.wikipedia.org/wiki/%E5%BC%A0%E5%9B%BD%E7%A5%A5)辑成的《[续道藏](http://zh.wikipedia.org/wiki/%E7%BB%AD%E9%81%93%E8%97%8F)》，其中收有《紫微斗数》三卷，作者佚其名，根据其内容来看这三卷紫微斗数其实是“[十八飞星](http://zh.wikipedia.org/w/index.php?title=%E5%8D%81%E5%85%AB%E9%A3%9B%E6%98%9F&action=edit&redlink=1)”之术；清初编《[四库全书](http://zh.wikipedia.org/wiki/%E5%9B%9B%E5%BA%AB%E5%85%A8%E6%9B%B8)》于[子部](http://zh.wikipedia.org/wiki/%E5%AD%90%E9%83%A8)[术数](http://zh.wikipedia.org/wiki/%E8%A1%93%E6%95%B8)累计收有[术数](http://zh.wikipedia.org/wiki/%E8%A1%93%E6%95%B8)五十种，不论是“十八飞星”或“紫微斗数”均为在内。

> > > 紫微斗数在流传过程中渐渐被分为南北两派。
> > > 
> > > > 北派三卷本《紫微斗数》后被道教典籍《[续道藏](http://zh.wikipedia.org/wiki/%E7%BB%AD%E9%81%93%E8%97%8F)》所收录；南派四卷本《紫微斗数》，直到[明朝](http://zh.wikipedia.org/wiki/%E6%98%8E%E6%9C%9D)[嘉靖](http://zh.wikipedia.org/wiki/%E5%98%89%E9%9D%96)年间才由[江西](http://zh.wikipedia.org/wiki/%E6%B1%9F%E8%A5%BF)[吉水](http://zh.wikipedia.org/wiki/%E5%90%89%E6%B0%B4)的[地理学家](http://zh.wikipedia.org/wiki/%E5%9C%B0%E7%90%86)[罗洪先](http://zh.wikipedia.org/wiki/%E7%BE%85%E6%B4%AA%E5%85%88)刊刻流传，即后来的《[紫微斗数全集](http://zh.wikipedia.org/w/index.php?title=%E7%B4%AB%E5%BE%AE%E6%96%97%E6%95%B8%E5%85%A8%E9%9B%86&action=edit&redlink=1)》和《[紫微斗数全书](http://zh.wikipedia.org/wiki/%E7%B4%AB%E5%BE%AE%E6%96%97%E6%95%B8%E5%85%A8%E6%9B%B8)》；现为研究紫微斗数的第一手资料，但是皆为[清](http://zh.wikipedia.org/wiki/%E6%B8%85)[同治](http://zh.wikipedia.org/wiki/%E5%90%8C%E6%B2%BB)时期的[刻版](http://zh.wikipedia.org/w/index.php?title=%E5%88%BB%E7%89%88&action=edit&redlink=1)。从此书内页即可得知其非一时一人一地之作，该书后来曾由[江西](http://zh.wikipedia.org/wiki/%E6%B1%9F%E8%A5%BF)名号“负子子”的[潘希尹](http://zh.wikipedia.org/w/index.php?title=%E6%BD%98%E5%B8%8C%E5%B0%B9&action=edit&redlink=1)所补辑，亦由[福建](http://zh.wikipedia.org/wiki/%E7%A6%8F%E5%BB%BA)[关西](http://zh.wikipedia.org/wiki/%E9%97%9C%E8%A5%BF)后裔[杨一宇](http://zh.wikipedia.org/w/index.php?title=%E6%A5%8A%E4%B8%80%E5%AE%87&action=edit&redlink=1)所添阅。
> > > > 
> > > > 紫微斗数虽在[清](http://zh.wikipedia.org/wiki/%E6%B8%85)末崛起，但是早期的命理学中有[八字学](http://zh.wikipedia.org/wiki/%E5%AD%90%E5%B9%B3%E5%85%AB%E5%AD%97)独大的情形。而随时代变迁，紫微斗数于1980年代之后在[香港](http://zh.wikipedia.org/wiki/%E9%A6%99%E6%B8%AF)和[台湾](http://zh.wikipedia.org/wiki/%E5%8F%B0%E7%81%A3)蓬勃发展，与八字学并称中国近代命理学中的两大显学

> > > > > 斗数理论派别可分为两大主流：一是以[星曜](http://zh.wikipedia.org/w/index.php?title=%E6%98%9F%E6%9B%9C&action=edit&redlink=1)解读为主；另一则以[四化](http://zh.wikipedia.org/wiki/%E5%9B%9B%E5%8C%96)解读为主。星曜解读为主的派系中又以中州派解读星情最为系统
> > > > > 
> > > > > > ，以十四主星所组成六十星系，彼此互相干涉所产吉凶互应，经整理成《紫微星诀》为论理依据。四化解读为主中又以钦天四化最具严谨，以紫微星辰为经，飞星四化为纬，相互对待，互生体用，其论理依据《斗数秘仪》。两大派别均对紫微斗数有所贡献，前者将全书中之星性解读以现今人之社会风俗加以修正，而不流于[八股](http://zh.wikipedia.org/wiki/%E5%85%AB%E8%82%A1)；后者归纳紫微斗数之构成理论，加以诠释，顾其本末。论命形式\[[编辑](http://zh.wikipedia.org/w/index.php?title=%E7%B4%AB%E5%BE%AE%E6%96%97%E6%95%B0&action=edit&section=4)\]

> > > > > > 现今的**紫微斗数**推命技巧较为细腻，除了技术层面的提升，在论命过程也愈趋于[客体](http://zh.wikipedia.org/wiki/%E5%AE%A2%E9%AB%94)性，以往所谓铁口直断的方式也慢慢以[心灵](http://zh.wikipedia.org/wiki/%E5%BF%83%E9%9D%88)层次的[咨商](http://zh.wikipedia.org/w/index.php?title=%E8%AB%AE%E5%95%86&action=edit&redlink=1)对谈来取代；在相较于**紫微斗数**相关古籍中的专断解读方式，现今则采用较温和多元的解读 ，让**紫微斗数**推命术能以更多元的方式呈现人的一生，此外在星曜的使用技巧上，也趋于成熟与多样性，令**紫微斗数**推命术在推断吉凶祸福的部分提升其征验度。

> > > > > > > 
> > > > > > > 定紫微星\[[编辑](http://zh.wikipedia.org/w/index.php?title=%E7%B4%AB%E5%BE%AE%E6%96%97%E6%95%B0&action=edit&section=8)\]
> > > > > > > 
> > > > > > > 得出命造五行局后，推判几倍的命造五行局数可以大于[生日](http://zh.wikipedia.org/wiki/%E7%94%9F%E6%97%A5)数(例如:十六日生人木三局者则六倍, 商数+1, 得可大与生日数)；下一步判断得出来的倍数与生日数之差数 ((商数+1)\*五行局数-生日数)，再判断此差数为奇数或偶数；若差数为奇数，则以倍数减去差数得到一个新的数字；若差数为偶数，则倍数与差数相加而得一新的数字，下一步起寅宫并顺时针数到上一步骤得出的数目，此一落宫点便是[紫微星](http://zh.wikipedia.org/wiki/%E7%B4%AB%E5%BE%AE%E6%98%9F)的位置；依照上一步骤如果改为逆时针来数，便是安[天府星](http://zh.wikipedia.org/w/index.php?title=%E5%A4%A9%E5%BA%9C%E6%98%9F&action=edit&redlink=1)。

> > > > > > > 生日数 = (商+1)\*五行局数-差
> > > > > > > 
> > > > > > > 寅起步数 = ((-1)^差)\*差+商+if(商>0, then 1, else 0)
