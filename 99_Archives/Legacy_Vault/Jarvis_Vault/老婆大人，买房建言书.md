# 老婆大人，买房建言书

亲爱的老婆， 我就开门见山的说吧:我想在达州西外买一套房子。 本来按照我的原计划我是打算把定金交了后给你惊喜（我的理想计划是交房后拿着房子钥匙，向你求婚呢），不过买房毕竟是大事，万一因为意见不合，把本来美好的事情变成矛盾的很远那就很不值得了。
买房子，牵涉这几个问题:

1. 为什么要买这个房啊？
2. 买这个房干嘛呀？
3. 这房子写谁的名字呀？
4. 我们以后是不是要定居达州呀？
5. 我们是不是能负担得起这房子呢？

下面帅气的老公一一来回答这些问题

1. 为什么买房——我想通过购置房屋这种资产实现保值投资，这并不是我现在有多少钱需要找地方理财，而是我觉得现在不逼着自己存钱就没办法存得起钱，我现在每个月6000多，还是没有剩多少钱。就算明年每个月七千多，其实也不见得能多存起来钱，现在看起来自己好像还比较小资，在过几年，小资心理根深蒂固后更不见得愿意去承担房子压力，所以我想不如趁早去承担起来。其实我们看别人买房做房奴压力多大多大，就像看邱艳结婚带孩子是一个心理，觉着人家多老火多老火的，其实甜处只是我们没见着，而且人家已经都完成了“家庭作业”了，我们其实是没做作业却在笑话人家艰苦做作业的同学。所以，我决定开始做家庭作业了。
2. 买这个房子干什么——，也可以理解为怎么用这房子。我大致规划了两种方向，① 简装修后，出租给学生家庭，按行情应该能租个一万多每年，也就是每月一千左右，用租金来偿还一部分按揭款。②用于你办培训班，这就是我建议你在达州做的原因，我看好的那个位置是新达一中，通川八中，还有两个小学，的交汇点，附近还有文理学院，其中靠达一中最近。是办培训班很好的位置，你可以按照你的想法把房子装修成培训班专用格局（顺便说一下只有48.7个平米）。 这样你就可以无房租压力的慢慢练手，把你的大计实施出来。
3. 这房子写谁的名字，这就是个纠结的话题了，老婆，我希望这套房子写我的名字，原因有三，①你家已经买了房子了，在我看来那算是买给你的房子，而我什么都还没有，我迫切需要证明自己，希望有一套属于我的房产。然后所有的房款都有我来凑，这样我才有我个人的成就感（附带提一下，原则上我不会用生意的利润来付首付款）②从资质上来讲，房子确实也没法写你的名字，因为外地户口在达州购房，必须有两年以上的社保缴费，这样没法写成你的名字。③这点也是最重要的，就是下一套买房的时候就写你的名字这样不会算成二套房。下一套房我们共同出资购买，写你的名字，然后就是你的首套房，好不好？（还好当时温江的房子没写你的名字）
4. 我们以后是不是要定居达州呀？这个问题是不好断定的问题，不过你放心，原则上以你的喜好为主，这套房的初衷还是投资，而不是定居。我目前是没有任何打算想要定居达州的，不过近几年肯定是达州。
5. 我是不是承担得起房价——  这个房价是6300左右，48.7个平方，大概就是30万，首付四成就是12万，首付三成就是9万，我现在所有的钱，加上所有借出去的有3万8，我找陈建借2万，小马哥借3万，把你手里的六千（如果你愿意的话哈），还有四千基金，就有10万了，我再找家里借两三万，四成的首付也就是12万，还是能搞定的。  然后每月还贷两千多，压力也不大，后期生意利润大了的话，就提前还清。 总体来讲压力不小，可能要稍微艰难两年，之前我的计划是把我钱给你做培训班生意的，看来你做生意就只能，找家里拿钱了。

宝贝，我当面一直没跟你讲，是我怕口才不好，讲不清楚，反而平添矛盾，这样写下来，发给你，我头脑清醒着，有助于表述清楚，好好消化一下哦，给我你的意见。
