# 2025年2月15日 日记

2025年2月15日 日记
早上刘洋的离职
今天早上，原本计划作为投融资部骨干的刘洋离职了。对于我来说，这个消息既让人遗憾也让我感受到了一些新的思考。刘洋的离开让我意识到，有些职场上的感悟是可以在不期而遇的变化中得到的。比如，在与刘洋相处时，我学到了如何更精致地享受生活。点外卖的时候，原来我只是简单地随便选择一些便宜的食物，而刘洋却教我如何选择烤鱼、豪客来的牛排、小龙坎的火锅等美食，回家也能品味到别样的风味，虽然是外卖，却能在家中享受到餐厅的感觉。
面对工作上的困境
这两天，我在工作上感到了一些迷茫，不知道该干什么。我明白现在最重要的是尽快敲定FA的事宜，推动融资项目。可是，心里又有些犹豫，因为现在推进FA可能会太早，反而不是最佳时机。相比之下，更重要的可能是先完成前期的1,800万闭环工作，确保这一环节稳妥再开始下一步的融资。
此时，我意识到自己内心有一种对表忠诚的冲动，似乎是为了维持与老厂之间的信任关系。特别是在汇报时，我觉得如果我能先把问题处理好再给老唐沟通方案，效果会更好。然而，回想起来，这其实是一种过度的依赖他人评价的表现。其实，我自己在做事时如果有明确的目标和标准，内心就能更稳定，评价他人的声音不会那么重要。
反思自己和团队管理
老唐提到的汇报方式给我带来了一些启发。在面对坏消息时，我意识到首先要降低预期，这样再给出方案时，结果可能会更加可控。而在面对团队成员时，我开始反思自己的管理方式。过往我可能担心自己带入了“穿鞋”的心态，担心失去收入和认可，但现在回想起来，反而要以“光脚”的心态，保持一种随时可以重新开始的觉悟。
特别是在团队管理上，之前的担忧让我对带领团队产生了一些恐惧，但现在我意识到，能够成功带领一个团队，是一项极其重要且有价值的能力。为了加强这一点，我计划深入学习团队建设的技巧和如何与团队成员更好地沟通和配合。
未来的团队管理与招募思考
接下来的目标，我认为可以通过心理契约来管理团队。真正的关键在于如何培养团队成员的成就感，让他们看到自己的工作是为达成一个更大目标而努力，从而激发他们的工作动力。同时，团队成员的需求也要得到充分了解和尊重，特别是在招聘新员工时，要更加注重他们的成长潜力和价值。
我也意识到，许多重要岗位的招聘不仅仅是看技能，更要看能否与核心创始人建立信任关系。因此，我对人才的招募方式也有了新的想法，未来要更早地开始接触和培养潜在的合作伙伴和人才，提前为团队发展铺路。
广州项目的思考
至于广州那边的工作，相较而言，尽管大家都在积极推进，但我认为当前不急于开始对外的宣传。首先，我应该处理好当前的工作，确保手头的事情按计划推进。之后再着手广州的项目，逐步安排好下半周的工作节奏。
日记总结关键点：
1. 刘洋离职带来的反思：通过与刘洋的合作，我学到了如何更加精致地享受生活，同时也认识到，工作中的困境有时是激发新思考的契机。
2. 工作中的迷茫与思考：面对FA项目和融资的推进，我开始意识到，过度着急可能不是最佳选择。需要闭环前期工作后再做决策。
3. 团队管理的反思：我意识到自己过于依赖他人评价，反而忽视了内心的目标。团队管理中的成就感、信任建立和目标明确是关键。
4. 人才招募与培养：未来要更加注重团队成员的成长潜力，提前识别和培养人才，为团队长期发展奠定基础。
5. 广州项目的推进计划：广州的项目目前无需急于对外展开，更要关注当前工作的推进，逐步调整节奏。
对生活和工作的小建议：
• 强化内在自信：你在面对工作中的变化和挑战时，可能会有一些不安和迷茫的情绪，但更重要的是建立自信和清晰的目标。当你自己对未来有了明确的计划，外界的声音就不会那么影响你。
• 做好团队建设：作为团队的领导者，带领团队成功的关键是给予团队成员正确的引导和支持。你已意识到，这不仅是技能的问题，更是如何在团队中营造信任和共同目标的问题。
• 放轻松，拥抱变化：当有人成为过去的一部分，不代表失败，反而是新的机会。你可以从每一次变动中获得成长，下一次的带人和管理会更加得心应手。
可行的待办事项：
1. 整理FA项目计划：在现有工作框架下，重新审视融资计划的启动时机，确保所有前期工作完成后再推进。
2. 加强团队培训：找出团队当前的技能空缺，制定系统的培训计划，提高团队的整体素质。
3. 优化人才招募策略：开始与潜在的人才进行非正式接触，逐步建立信任关系，为未来团队发展储备人选。
4. 完善汇报沟通技巧：调整汇报方式，尤其是如何处理好坏消息，确保团队和上级能更好理解你的决策。
5. 广州项目的计划调整：整理广州项目的时间表，明确下半周的工作重点，确保工作节奏稳定推进。
