# 这会儿登机了 老婆；

这会儿登机了 老婆；
你昨晚在群里说那么多气话，是因为感觉在关系里吃亏了吗？
我在想日子其实在越过越好，我们在六村与人合租，环境很不习惯，那时你最大的怨气是跟人合租卫生不好，要收拾厕所垃圾，要跟人抢着做饭，这些事儿让你能大发雷霆；

现在我们住的环境大幅改善，前期我俩不规律，没能让你按时吃上饭，你常常大发雷霆，我出差你一个人在家，万一有点啥，我也很担心；
我们决定让冷大姐过来，现在处得还行，而且也保证每餐正常饮食，身边随时有个人；
你也向家里求助，把之前转不动的财务情况极大缓解，我看到的是我们的日子在越来越好；

我能想到让你不满意的，是陪伴； 近期高频率出差，陪你的时间少了；

我知道怀孕让你一个人承受了很多，你敏感的点比如我觉得吵到我了，其实是单方面解读，那次喝了茶晚上你睡不着，确实我睡着了被弄醒不舒服，换过来想，也有你晚上不舒服要喝水，我也会起夜的吧，我想表达的是两口子相处，有1万个细节，其中8000个细节ok的，2000个有不ok的，你总盯着那2000个细节，以此为基础假设这个男人不够爱，不负责，不值得，这让我也觉得委屈啊。

我能理解你的不满，刚耍朋友时没钱，但至少陪你的时间还是有的，妈的，现在是既没钱也没时间陪伴，你独自面对很多；

我想说的是，现在是一个很关键的阶段，我们出差多，是真因为项目有进展，一二期基金，和中介类业务（缪称之为战略咨询类业务），我相信3-6个月内一定会有实质性进展，换句话说一定能见着钱；

如果你觉得陪伴不足，那好，我就守在你身边，你认为能解决我们婚姻的核心矛盾—收入问题吗？

这个阶段，对你我都不容易，但我有信心，两口子相互扶持着熬过去，就到一个新阶段了，我们在婚姻里也会更成熟。

现在，你想回重庆，你思考过核心是什么吗？ 是我的陪伴不足，还是叠加了职业上的不如意？

我感觉你是想找到一种，“老娘曾经从不为这些破事儿低头，现在虎落平阳被犬欺，我要远离这些，找回从前”的感觉，

你回重庆，我支持的，你要每月打钱，我也同意，现在还没见到钱，前面几个月的开销，我欠着，记账，求助老爸老妈帮衬帮衬，我一定给老爸老妈补上，到时我们商量一个费用标准，比如每月打4000；

我相信老爸老妈，也希望我们熬过这段时间，他们能帮的会帮的,其实老爸老妈对我们的支持意愿，本质上取决于你对我们婚姻的笃定；

如果你不笃定，他们支持也没底啊，万一支持着，结果两口子离了，他们也懵啊。

我认为我俩的婚姻基本盘是很好的，两口子性格合得来，也不藏着掖着，双方父母也支持，两大家子心基本上一起的； 男的，自己想创业，处于投入期，经济情况较差，女的也为了男的，到了同一个城市，有个班上，两人怀上娃儿了，时常吵吵闹闹，拌拌嘴； 这样的婚姻，没太大硬伤；
婚姻中，没所谓公平不公平，你认为自己付出多了，不公平，其实你穷尽一下所有伴侣可能性，无论在哪段关系里，结婚面临经济压力，怀孕面临身心压力，都会遇到的，只是我们什么心态面对； 如果这个男人给力点，做饭做家务，按摩加揉腿；不给力点，就是出差，打鼾加放屁； 但你面对的基本挑战，也是这些。

当然，我想说我在陪伴，家务，按摩揉腿没做好，另一方面，冷大姐大远来照顾，和我其实接纳了你很多负面的情绪，也是一种对你的爱啊。

我不是想要表功，只想让你换个视角看待，咱们这个家虽然没那么好，但决没你说的那么糟，我坚信如果上辈子你也是这样和我过的话，你最后也没后悔。
