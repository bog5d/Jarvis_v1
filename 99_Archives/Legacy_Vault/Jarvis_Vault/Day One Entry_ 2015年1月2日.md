# Day One Entry: 2015年1月2日

\*老孙去了大竹，看到了很多问题。我还是大部分都认可，比如门市卫生一团糟，员工风气散漫。至于说开慈善机构解决就业，这点来说呢，还是偏激了点。
门市上摆了一大堆货物，一直没有收走，其实我也是看到这个问题的。每次不是王斌说要发下去的，就是忙得很，空了搬。如果我够果断强烈要求他们搬起走，肯定也是能搞定的，然后我也犯贱的觉得认为先支持王斌的安排，所以王斌在安排没有提及到这一层时，我没做要求。我很后悔自己没有果断要求做我认为对的事情。自己认为对的事情，干嘛要妥协呢？ 不是有库房吗？还堆在门市上？ 他妈租个库房来空起耍的吗？
我也知道，我自己也怕搬货时，我太累。现在每次下大竹，都只想快点离开，真的不舒服，吃不好，睡不好的。痛苦得很！ 可能是我自己有逃避思想，不想动，想懒散，故而没有强调那些事情。 我负有领导责任的，我怕累，想偷懒，有时怪王斌他们懒，我自己我有责任。

# \-\*\*门市卫生这事情，接下来十天，每天让冷瑞上班拍照，连续看到没有跟销售无关的东西在销售区域。

# \*\* 门市风气问题，我认为还是王斌安排和跟进问题。安排不仔细，也没有跟进到底完成没有，久了也不会把他的事情当回事。

其他高大上的想法，暂时没有，先把能做的做好。 卫生每日拍照，风气问题加强公布进度。 这么定了。 他妈王斌一天到黑，要死不活地看到急人得很，有朝一日能分开就分开，真的不想被负能量影响到求了。
