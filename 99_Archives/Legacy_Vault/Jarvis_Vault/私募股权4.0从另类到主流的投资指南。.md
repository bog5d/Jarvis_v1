# 私募股权4.0从另类到主流的投资指南。

![Evernote Snapshot 20220127 160228.jpg](./_resources/私募股权4.0从另类到主流的投资指南。.resources/Evernote Snapshot 20220127 160228.jpg)
私募股权有一个很重要的特点就是，少数的骨干经理人在这里面收益丰厚，但同时私募他对被收购的企业的员工也是非常的慷慨，就是他对人的分红啊，利益分配这个事儿，做的非常的到位，
这个时候我就有可能说，如果我们可以通过股权这个事儿。跟很多的相关方建立，利益共享，而有利益共享，还有更容易形成合力，它会形成一个，嗯增强回路，相互增强的增强回路。
![Evernote Snapshot 20220127 161507.jpg](./_resources/私募股权4.0从另类到主流的投资指南。.resources/Evernote Snapshot 20220127 161507.jpg)
这里面有一个重要的知识点就是私募股权，它其实是承担了为小微企业投融资的重要的角色，而小微企业其实是最难融到资金的。

![Evernote Snapshot 20220127 161347.jpg](./_resources/私募股权4.0从另类到主流的投资指南。.resources/Evernote Snapshot 20220127 161347.jpg)

![Evernote Snapshot 20220127 164232.jpg](./_resources/私募股权4.0从另类到主流的投资指南。.resources/Evernote Snapshot 20220127 164232.jpg)![Evernote Snapshot 20220127 164233.jpg](./_resources/私募股权4.0从另类到主流的投资指南。.resources/Evernote Snapshot 20220127 164233.jpg)
这里面有一个重要的观点就是，私募股权基金，它的收入，管理费，他带来的那个费用足够高的时候，会让这些私募股权它去追求基金规模，因为基金规模上去它的管理费收入足够高，他们就不会去投资更好的标的，或者风险更高，但收益更大的标的转而去投资更安全的交易，就就会偏离股权基金，对一些初中，这个也是当下私募股权，他的一个，跟三方的利益不一致的地方。
![Evernote Snapshot 20220127 164518.jpg](./_resources/私募股权4.0从另类到主流的投资指南。.resources/Evernote Snapshot 20220127 164518.jpg)![Evernote Snapshot 20220127 174013.jpg](./_resources/私募股权4.0从另类到主流的投资指南。.resources/Evernote Snapshot 20220127 174013.jpg)
书中有一点说得很对。即使投资者并不满意收益水平，管理者依然可以过得很好。
![Evernote Snapshot 20220127 174013.jpg](./_resources/私募股权4.0从另类到主流的投资指南。.resources/Evernote Snapshot 20220127 174013.jpg)
做过权责市场是一个缺乏效率的非有效市场，因为这里面有投资效率的不确定性和特定性不对称以及流动性比较差才行，那这个事儿他有个好处就是私募股权里面的投资收益的离散性特别大，也就是说做的好的人特别牛逼
另外对于。相对于证券市场的基金投资者他们的收益严格的和市场行情挂钩，市场行情好还没好，市场已经跌他们就跌，所以所有从业者的收入其实是相近的就怎么说，你再聪明才牛逼，你的聪明和智慧在里面，很难发生高额的回报和变现。

![Evernote Snapshot 20220127 174259.jpg](./_resources/私募股权4.0从另类到主流的投资指南。.resources/Evernote Snapshot 20220127 174259.jpg)

基金的组件一共分为4步。
组建基金管理有限公司，组建基金管理中心，有限合伙组建基金的投资中心，然后到银行开户。
