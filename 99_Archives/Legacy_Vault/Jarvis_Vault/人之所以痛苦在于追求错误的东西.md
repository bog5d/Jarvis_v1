# 0913 人之所以痛苦在于追求错误的东西

- [ ] 绿影欲回  心中失落。
- [ ] 是我不坚持还是她真不愿。
这种情绪真的是很难表述的，是一种极端矛盾复杂的心境。怎么重头说起呢？
今天原计划是，自己一个人下渠县把旗舰店的上样落实了，因为绿影把两边的上样都搞砸了，因此就计划去把尾收了，顺便骂她一顿给点脸色。
然后，找她去拿价签，再去火车站坐火车下渠县。
哎呀，说得太啰嗦了，情况就是她很想跟我一起下去，我表面很凶很冷漠，其实也希望跟她一起出去，甚至也希冀晚上发生点什么的。
结果她穿的一件蕾丝打底衣，乳沟若隐若现的，让这段时间性亢奋中的我，一下子来了赶脚。
然后中途因为晚点没赶上车，自己开车到了渠县。本来计划在渠县住一晚，甚至都明说了开一间房，心中期待也蛮深的。 
说说自己的心态吧，是期待发生点什么的，但又想起老婆的爱了，因此本身就想要不想要的，还在犹豫挣扎的呢。
她呢，她的心态我并不知道，  在身体上是有明显互动的，比如他戳我腰，我还是会碰她的胸，都还是挺正常的。。。。
总之就是，我默认她是愿意并期待的。
中途给渠县旗舰店张猛电话时，我觉得张猛不是很热情，就想今晚把事情落实了直接回大竹，不过要回大竹的话我还是挺不愿意的。有种飞到锅里的肉又跑了的感觉，所以回大竹的计划并不坚定（也许这就是我骨子里很贱的一面吧），但我对绿影表达的是要回大竹的想法的，她也认同，然后我就想，要不先去旗舰店落实了事情后，如果天还没黑的话，就回去大竹，天黑了再说吧。
 
结果，我们完成事情后，天已经黑了， 于是呢，我就说天黑了我是真的不想再开车回大竹了，毕竟山高又黑。要不就住下吧，她笑了笑那好嘛。然后我又觉得这事儿吧，好像有点不好，我想起老婆了，所以就神色凝重的想了会儿。
然后，我说如果你想回去的话，可以坐组合车回大竹，然后明天来开车就是。
她说，用不做啦，实在不行就喊王斌坐组合车过来，再把车开回去。
我想了想，又是神色凝重的犹豫了，她就不停地说，开玩笑哦，住一晚两个人三百多，你还要去北京，能节约还是节约。
我听起来就特别不爽，因为我觉得这好像是一个借口，一个不想跟我一起的借口，这是对我很强的一种彻底的否定感，于是心中特别不爽特别不爽，一下子就不想搭理她的感觉，真的，完全不想搭理她。也有种被放鸽子的感觉。

唉，那种强烈的不爽感，让我特别特别堵，一种第三者被无视的感觉。老子，话都不想说了。

然后，她说要我陪她买衣服，结果帮我选了一套衣服，她并没有买到称心的衣服。心情稍微好点了。

现在回过来看，也许她真的是不愿意我太花钱。
`也有可能是不太好意思花我钱，因为我不停在强调要去北京要花多少钱。 她可能认为我也会``有点心疼钱，  不好意思再去享受那个酒店之夜。`
她也可能有矜持，不想表现得过于主动了。

好了，不说了，我觉得也许我想要的是错的，也有可能她是想而不好意思的吧。反正是一段充实的心理历程
