# BP逐字稿1.0

各位领导及合作伙伴好，我是泽天智航的投融资负责人XX，很荣幸跟大家汇报我们泽天的企业情况。

我们泽天的方向是做国防装备信息化 智能化，自主可控这方面； 从大行业来讲，属于军工电子，同时，我们在该行业的两个细分领域属于国内民营企业的龙头和引领者，一个是信息化装备的核心处理单元比如指控火控的主控单元，一个是算力单元，比如海j舰船公共计算环境算力单元，核心处理和算力单元，又被称作信息化装备的大脑和神经系统。 
这个我们后面会再展开细讲；我们的核心处理单元和算力这些年海军和空军应用相对较多。

首先跟各位汇报一下企业的3个优势：

首先大家都知道，军工是近几年少有的不受口罩和经济周期影响，持续保持增长的行业。 
其中的国防装备信息化是千亿级的赛道，而军工装备核心处理单元是现代国防装备的大脑，

据华安证券预测，25年规模达到785亿；

而且泽天在细分赛道民营企业头部，是国内唯一打通核心处理单元，到核心分系统，到整机再到底层驱动核心代码和底层实时操作系统的自主可控生态闭环的企业。  同时是国内军用舰船公共算力单元领域的为二团队，另一家为浪潮信息。
而且我们在交付时浪潮的启动时间四十几秒，而我们的不到十秒，就是因为这里面的底层驱动核心源代码，是我们根据这个场景自己写的。后来浪潮还想找我们买底层驱动的核心代码。 

这是我们部分批产型号展示，泽天6年，累计参与了200多个科研项目，其中约50个已经定型，有十几个重点型号，目前有八个型号项目开始大批量生产和列装，这八个在未来3-5年将形成20-30亿的订单。 这是不含目前定型中的那些型号项目的情况下。 所以我们团队内部对未来充满信心。

国防装备自主可控 信息化 智能化一定是大势所趋，从2个维度的来看，一个是国家层面，2022年，习总书记在求是杂志发文说”推动关键技术自主可控，事实证明我国在这方面大有可为“，其实是着重认可了这几年我们国家走自主可控取得的进展，并且未来一定要坚定地走自主可控，为什么会专门发文认可自主可控呢，毕竟刚开始决定跟老美脱钩，走自主可控时，不像现在这么有底的； 

二是资本市场层面，中航证券也公布了军工电子行业的深度研究报告，从产业链角度论证了信息化装备的核心处理等产品是武器装备的“神经中枢”，从资本角度重点看好军工信息话 智能化，认为未来是军工信息化和智能化的大时代。 
因此我们泽天也时常感觉自己很幸运，我们一直深耕在这个大势所趋的行业里面。

接下来分享我们的产品。
在详细介绍产品之前，先简要说明我们产品的分类逻辑。我们的产品结构是“3+1”，包括三类硬件和一类软件。

在硬件方面，首先是核心处理单元，这是我们的主要产品。其次，我们有一系列的核心处理分系统。第三类是军用整机。
众所周知，军用产品的生态链从原材料到元器件，再到处理单元，分系统，最后是整机。我们的产品线涵盖从核心处理单元到分系统，再到整机，以及底层驱动核心代码和底层实时操作系统。这些软硬件结合形成了我们这家高科技民营企业的特色。
所以，我们的第一类大产品是核心处理单元等硬件，第二大类是软件和操作系统。
接下来，我们详细展开核心处理单元的产品情况。

接下来，我们来讨论我们的第一个产品，因军工行业的特殊性，我在分享产品时，若涉及到军，我会省略方式，希望各位理解。

这是海''j某代台的核心处理单元，专用于Z挥控制和H力控系统。

首先，让我来说一下这个产品的背景。
在过去，我国海J在指挥控制和H力控制系统方面是相对独立分散的功能系统，Z控侧重信息收集、处理和分发，H控侧重目标定位跟踪和W器制导。

后来海J在方面走得比较前卫，开始把Z挥控制与火控系统做了统型，在圈内呢 就叫做第一代标准化指控台，也叫一代台，其中一代台，二代台，用了进口的分系统和部件。
从第三代开始，要求100%自主可控，而且所有舰船都要装，这个量很大。 当然这么大的型号总体任务，不是民营企业接得住的，当时就有近20家国家队PK，最后3家胜出，来分这个市场，这三家分别为中船716，中船709，这两个都是船的总体单位，另一个是被称作亚洲第一所的中电14所，这三家三流水。

而我们则是给这三家的指控火控系统配核心处理单元。

另外，关于它在舰船上的配置方式是这样的，按照每艘舰船的舱室，以舱室为单位来配置，一个舱室配5/6套，含备品备件7/8套左右。 而每艘舰的舱室从数十到过百的都有。

第二个产品是通登控制核心处理单元，通登是通用登录艇的简称，都知道东边，我们要完成祖国统一的话。肯定不是焦土战，发射一堆导弹去炸掉，大概率是登岛作战，所以通用登录艇需求大。  每条艇配备5-8个，含备品备件8/10套。 未来局势变动的话，量会比现在更大。 

第三个产品是陆J机电车终端核心处理单元，关于陆军车辆，大家对诸如坦克装甲，猛士越野等往往比较熟悉，其实有一类车数量很大，就是机电车， 主要为作战提供移动式的机电支持，比如移动发电，通信支持，维修保障。 泽天研制控制总端产品的核心处理单元，通常情况下单车配备4台，含备品备件可达5/6台。

第三个产品：海J公共计算环境算力单元：该算力单元是军用信息化“云-边-端”框架的核心基础单元，

简单说说什么是“云-边-端”，以我们手机用AI为例， 比如百度（文心一言），或者chatgpt，当我们在手机端使用的时候，手机只是作为数据采集设备，然后把数据传到云服务器，服务器完成海量计算后，返回结果，手机作为交互设备，把结果呈现给我们。

 手机和云计算服务器之间就是端和云的关系，为了解决延迟问题，增加边缘计算减少延迟，甚至在服务器中断时提供服务。 同样，现在信息化和智能化装备，也要用“云边端”框架，比如海J以整舰为单位，建立中心化云计算服务器集群，其他协同的装备则负责采集数据和执行指令，第一终端装备不用再重复配计算单元，减重减负； 第二作战方式会更加智能。 

目前海J和陆J正大力建设以“云计算”为基础的智能化系统，陆这边，则是以营地为单位，建中心化云计算服务器集群，在内部叫做“战云”协调；
而我们的算力单元，打个比方吧，算力单元的样子有点像很方正的黑色的液晶显示屏，一般6-8个算力单元组成一个算力机组，算力机组组合堆叠成一面墙，就是算力阵列，多个算力阵列排在一起，就形成算力集群，舰船一般以阵列为单位来配置； 而陆 的营地则是以算力集群为单位配置，有点网上看到的谷歌大数据中心的感觉。   目前海和陆在这方面需求潜力巨大。

以上为核心处理单元系列的几个代表型号产品，接下来跟各位分享一下我们核心处理分系统级别的产品，
第一个时舰载核心处理分系统，海的水下舰艇用双冗余处理分系统。 
水下舰艇在运行和作战过程中，这个系统需要处理很多信息，比如战斗系统数据，密码机数据和外部通信总线数据，这些数据必须实时备份下来，所以需要做热备份处理，同时要保证极端环境下的高可靠性，所以要做双冗余系统，如果A系统卡壳，B系统马上启动。
该系统，水下和水面舰艇会陆续装配。

第二个是，空中防碰撞系统。 我们的军机在飞行时，需要实时感知当下空余有没有其他飞行器，根据监测数据来提供飞行姿态的决策，比如是水平机动还是跃升机动，这个是军机标配了。

第三个是海的鱼雷核心分系统，该系统为水下制导武器的核心处理分系统，主要负责水下武器的发控流程和任务载荷设定，现在水下武器越来越智能，比如引爆系统，以前更多是撞击式的弹力点火，命中接触后撞击引爆，现在更多的是非触发式地电子化的安全引爆系统。而这些水下制导武器，既可以舰载，也可以机载，这些产品作为消耗品，潜力巨大。 

![unknown_filename.7.png](./_resources/BP逐字稿1.0.resources/unknown_filename.7.png)

接下来是我们整机系列产品，空方面的机载测控仪，该测控仪主要用于飞行参数的实时采集和记录，实现航空故障的诊断和预测。该产品也是战机标配类的。

重点跟各位汇报分享一下这款无人机任务管理机，通常情况下无人机分为飞行控制系统，任务载荷系统和任务管理系统，而我们的产品就是处理任务管理这方面的。 我们的任务管理机因抗恶劣环境，比如高低空，高低压、盐雾，震动冲击和复杂电磁环境下运行可靠，且支持长时间飞行，在无人机里用得比较好。大家知道，俄乌战争后，无人机的重要性被大大提高，今年被定为了无人机元年，无人机未来的需求极为客观。 

这是舰船上的区域状态监测装置，该产品特色是能适应海上复杂的环境，监测船舶的航行参数以做航行决策，同时该产品可辅助增强舰船的战术运行能力和海上作战支持能力，每舰船一般配4-5个。

以上是我们的三大系列的硬件产品，接下来说说我们的底层实时操作系统RustOS，前文提到我们坚定地认为军工行业需要这么一套高可靠性的实时操作系统，为什么呢？
首先以前我们用的实时操作系统是老美，那家公司叫风河， 产品是VX works， 航空航天、国防、汽车和工业自动化应用很广，我们现在自主可控肯定不能用他的，必须自己有。 当然目前国内也有友商在做类似产品，比如南京的翼辉，推出的sylix，这家公司就凭这套软件估值超30亿，另一家是麒麟操作系统，300多亿市值。 既然有了我们为何还是要研制推出这套系统呢？

回到刚才说的高可靠性，我们以特斯拉的刹车事故为例，当时一辆特斯拉model Y，失控加速跑了几个街道，造成2死3伤，但实际上车主踩了刹车的，而且还把档位也拨到了P档。这其实是源于特斯拉的刹车采用的是电子刹车系统，当你踩下刹车，不是从机械层面制动，而是给底层系统发了一个电子指令，而这些电子指令会被转换为一个个逻辑运算的任务切片，当你系统任务载荷过高时，导致内存过载时，这些优先级高的指令没法及时被处理。 对于国防装备来讲，这种情况是不允许的， 而我们要推出的RustOS， 通过独特的所有权机制，可从底层避免了内存泄漏问题，我们可以自信地说这是目前国内安全性和可靠性最好地底层实时系统之一。

从行业生态来讲，国内这方面也只能算刚开始，我们希望这款系统能为军工行业做好一些生态和基础建设。
