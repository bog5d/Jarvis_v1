# Day One Entry: 2016年3月13日

# 160313 备西外开盘 追问生活

自己做了一下人生意义的自问，发现进步和学习新东西，重复出现了。
也就是说我自己看来，进步和学新带来的快乐还是比较多。 因为没有jin b进步和学新，会让我不快乐。
因此可以根据这点更多的做一些新尝试，尝试一些真正觉得比较难一点的挑战，才能收获更多的成就感，比如学英语啦，比如看26本书啦，这些挑战真的都不简单，还有走路啦。 做成其中一件事，一定会带来无与伦比的快感刺激。
真的这是我找到生活意义很重要的事情，还有学乐器。
想到这些心里充满了期待，自己想干的事还真的有点多。

今日自问
真的对燕子就没有更多相见欲望了吗？
应该是占有欲降低了吧？ 也可能是重新调整了一下关系定位吧。觉得她也没有我自己脑补的那么多美好了，我总把她比作巫师里面的特丽丝，深爱着杰罗特的美丽女术士，但其实她素养还是不够高的。
也可能是自己在范他们面前显摆过这段关系，被范秋羡慕过，然后就把她当作自己的一大异性缘成就，而更增加了占有欲。突然闻之她有相亲对象后，开始气馁，从骨子我还是希望她对我保持专一吧？ 如此矛盾的心情才开始折磨我。
不想继续见她还是有面子成份在的。
自觉没面子吧。唉！
惭愧，境界不够。有种想通过不见来惩罚她的心态。

也不强迫自己啦，随心而行。
