# 161217文字日记

1. 跟老孙谈了退出他股份之想法，他欣然允之，并称预料之中。 
2. 老孙的坦然，对比王斌的小为难，滋生对比之心。对王斌总想一次性拿完资产，有点埋怨情绪，觉得还是没有我那么一般，考虑他的感受。 
3. 其实所有的痛苦，和埋怨，都是自己无能的愤怒。 在对王斌的埋怨上，我所无能的点在哪儿呢？  
	* 觉得有些事情需要他办，他却没给我办。比如去和万和夏姐谈，比如去看库房，比如门市转让。 这些事情我希望自己能偷懒，不去触碰这些事。
	* 但是王斌在这些事情上，并不积极。 当然也可能是太忙于其他事情了。 比如发货。 不过我感觉，王斌心不在此了。 没有那么大的肚量。
	* 我自己当下痛苦的，就是无法承受王斌不那么有感情地对待我。我甚至还想他会感激我这些年的相处，总想着他离开后也能给我好评，可能是太想要他的好评了。 所以没那么果断，也是自己太在意自己羽毛，太想当个好人。
	* 我自己明天还是应该给他提出来，哪些事情需要他做，而自己则赶紧开始招聘。 明天去谈净水客户和寻找库房。去东柳碰碰。
4. 晚饭后，和宇宝宝去东湖开车慢慢溜达，边开车，边牵她手，有一种静静享受二人世界的美。 思宇还在感叹，以后结婚了，还会不会带她出来兜风呀。 我希望自己能做到一直能够带她出来兜风。 其实宇宝，嘴上不说，内心还是很享受二人兜风的，尤其是她会选超级好听的歌，我们在车上感受。
