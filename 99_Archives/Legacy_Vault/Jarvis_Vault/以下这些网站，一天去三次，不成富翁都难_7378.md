# 以下这些网站，一天去三次，不成富翁都难

【编者按】：人一生，成与败，学习环境、创业环境极其重要。月光族上网常做的事情是看看娱乐花边新闻、QQ闲聊、论坛灌灌水、上社交网站穷开心……，十年后，我相信他们还会是月光族。倘若这些人每天去下面这十个网站，一天三次，我敢保证，他们不成富翁都难！
1、创业学院
创业学院（www.chuangye666.com）是一家搭建于2013年10月30日的新兴网站，该网站拥有全网最全的视频及文字资料，包括了小吃、小本创业、股票金融、记忆口才、两性健康、手工工艺、舞蹈、医学、气功等等，资源总量超过3000G，价值超过30万元，这里强烈建议大家开通该站的VIP会员，目前公测期间只要30元就可以开通永久资格，下载海量资源了。
2、 MBA智库网
这是一家日浏览量接近100万的网站。MBA智库网（ www.mbalib.com ），顾名思义，网站集合了大量关于金融类、管理类、营销类、经济类等宝贵资源，对于中高层管理来说，在这里无疑可以享受到知识海洋进阶的盛宴，尤其在百科一栏中，大量的经济理论、模型、效应、法则会让您在周围人看起来很有管理大师的范儿……
3、创富邦
如果说MBA智库网是文字版的商学院，那创富邦（ www.cfbaa.com ）就算是彻头彻尾的视频版云存储商学院了。看完资源目录表后您一定会很吃惊，这里面各行各业的培训视频未免也太全了吧，成功学、管理学应有尽有，对于上进心强、学习欲强的朋友估计要兴奋上好几天。唯一美中不足的就是可免费看的资源并不多，网站建得也不够大气，互动性也差了点。但鉴于资源极其丰富、画质也是高清的，从这几个角度来看，还是十分值得推荐的。不过似乎还有机会免费得到移动硬盘……
4、 中国经济网
富人总是能把握机遇并且有效将它转化为财富，是因为他们用非常敏锐的眼光去看待国家每一项出台的新政策，绝大多数巨大商机都是政策所带来的。中国经济网（ www.ce.cn ）作为国家经济的门户网站，发布信息的权威性和时效性是毋庸置疑的。不管您是炒股、投资还是创办企业、还是正在寻求商机，我觉得这个网站是要常去转转的。
5、 投资界
投资界（ www.pedaily.cn ）的性质跟创业邦很类似，也是一家股权投资门户网站。很多机构愿意花几千万元甚至几十亿的巨资到一个新兴公司中，必定是因为这家企业将来的盈利及发展空间是非常具有想象力的，通过分析和解读这些企业，相信您肯定能从中找到灵感。也许就是一个小小的灵感，就能改变您一辈子的命运。
【编后语】：以上几个网站我经常去，每天一小时，真的是受益匪浅，学习高人的商业智慧，也让我的薪水两年内翻了五倍，感恩这一切。当然，肯定还有更好的也许我还没发现，希望有心人能在我这基础上给予补充，再此表示感谢！同时希望看到这篇文章的朋友，如果本文对您有帮助，还请您不要吝惜几秒钟，转发或者分享让更多人一起来受益！世间广洒爱，四季都是春！
