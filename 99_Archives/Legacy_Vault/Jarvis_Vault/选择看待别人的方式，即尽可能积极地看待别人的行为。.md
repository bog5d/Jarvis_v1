# 选择看待别人的方式，即尽可能积极地看待别人的行为。

选择看待别人的方式，即尽可能积极地看待别人的行为。

突然觉得，把别人的动机想得美好一点，真的自己要平和很多。

自己以前为什么总想看到别人的坏，来寻找成就感呢？寻找别人的坏，来瞄准,来抨击。反正都是看别人的坏。

我记得曾经，冷瑞问我一个问题，前女友说他是在催工资，我就拉下脸责问他是不是在催工资，然后自己一肚子怒火就涌上来；

曾经自己看王斌，总觉得他怀疑我，不信任我，不满我，甚至可能会有单干的谋逆之心。

而老高（前女友），这么多年在我耳边，传递了很多对周围人负面的看法，说我哥哥虚伪，嫂子要强品味low，妈妈说话阴阳怪气不明说，老马（我领导）不如我强....

当时她对我说了很多人的负面看法，我现在想起来很是气愤。 总觉得她有把我带偏，

但是按照笑来的说法，其实无论是我自己对冷瑞 王斌（下属），还有觉得被老高带偏，这些看法只是让我更加不平和而已。于我自己也没进步。

最近一两年，自己在真诚方面做得还是可以，所以和冷瑞王斌的沟通，已是坦诚无碍了。

今年选择和老高分手，发现分手后看到的世界完全不一样。（老高其实很多时候对他人的看法真的是负能量满满的）。

有时候想起来，会埋冤老高对我的影响，但其实也有自己的可怜之处，比如父母关爱太少导致的安全感极度缺乏，使得她更希望控制和抓住一些东西。

这么一想之后，其实内心是平和的。

现在思宇看待她妈妈和继父，有时也有点负面，虽然她的负面看法其实是有根据的，但如果我们能抱着拿不准别人动机时，就尽可能积极地去理解；当别人表现出不够善意时，相信对方有无奈之处或者非针对性恶意；

比如王平，老是说一些对我不满的言论，定是有伤害他之处而不自知；对于他的不给面子，何尝不是他不够自信，觉得我太让他不自在呢，也是他有难处吧。

如此一看，发现这个世界真的没有那么多勾心斗角，暗流涌动。
