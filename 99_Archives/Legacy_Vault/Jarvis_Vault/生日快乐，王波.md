# 生日快乐，王波

王波，生日快乐，二十五岁了。过去的二十四年，不知道是怎么过的，总结一下，应该是随波逐流。
工作没有提前计划，是缘分带来的。恋情，没有标准，也是缘分带来的。
    我不知道，我恋爱图的啥，好像没有图啥，吵架的时候才带来这个思考。每次吵架时只想快点不让她生气了，快一点哄好，自己需要安慰还是需要支持，好像就忘了，当然她也不会给这些，偶尔想想，心中自是百味杂陈，难以言表。一个男人的不停让步，真就代表了这个男人有错吗？

    如果我的委屈无法表达的话，我想自己也会有憋不住的一天。 
 但，抗拒是无效的，我想这就是为啥很多男人红杏出墙的动机吧。  算了，只有一个脆弱的男人方回以此为借口，在外寻花问柳。王某，还不至于这点觉悟。不过真心委屈了，在二十五岁生日这天真心委屈了。好吧，诚意成胸怀吧。
我希望今年学会新的CAD技能，存十万元钱，还有割掉包皮。
生日快乐，王波。
