# 可算作促销动员信吧各代理商注意，各代理商注意，最近厨卫销售相对

可算作促销动员信吧
各代理商注意，各代理商注意，最近厨卫销售相对疲软，公司计划在5、6、7、三个月内执行渠道促销。 本次促销以店面促销为主，促销主题以“买美的蒸汽洗，送美的机器人吸尘器” 为主。 参与此活动的网点，为代理商自有专卖店，县城重要网点，核心乡镇网点。 我们会免费提供，吸尘器演示台，吊旗，地贴，X展架（含架子）、横幅等促销物料。
但，请注意，这次并不是跟往次发物料一样，发下来就完了，是一次真枪实弹的促销执行。 有几点注意：
第一，所有物料，我们免费提供，不需要代理商掏一分钱。
第二，代理商各自区域内的下游网点，我们也可以提供，需要代理报准确的需求。请注意，是准确的需求。不是拍脑袋觉得自己要多少份。而是跟自己网点，沟通后，愿意参加，并且会执行到位的网点，我们才会发放物料。
第三，代理商报上来的需求，我们都会无条件提供，要达到两点基本要求：1，物料保持3个月，也就是在8月份之前不能扔掉。2，代理商自己的人，必须到乡镇配合布置物料，并拍照交回来。
第四，我们在下次巡检的时候，如果巡检到报了物料需求，却没有执行物料的网点，一家罚款500，罚代理商。 因此，代理商在问乡镇网点的是否参加这次促销时，一定要是乡镇老板，愿意配合执行的，才报上来。大家电的网点，不建议参加，因为空调的物料多了后，势必撤走厨卫物料，而我们是照常罚款的。
本次，促销活动的原则是宁缺毋滥，那种不能很好的执行物料的，就别参加；给了物料又不执行的，就罚款。 我们宁愿精致的布置1一家网点，而不是粗糙的布置10家网点。
