# 国内券商=国外投行？三分钟解决你关于投行的所有疑问

投行是相对于商业银行而言的，商行我们平时接触比较多，主要是拉存款和放贷款，另外还有一些支付、结算的业务。

投行这个词汇是从美国舶来的，美国典型的投行代表包括华尔街的高盛、摩根斯坦利和美林，他们提供**广义上的投行服务**，包括狭义上的投行业务，证券经济业务和财富管理。

**狭义上的投行业务**，也就是国内券商公司的投行部门会提供的服务，主要是股票、债券的发行，以及兼并重组的咨询业务。

当然啦，国内的券商公司主营的还是二级市场的证券经纪业务，和传统投行并不能混为一谈。

我们也可以按照投行的规模和业务来做一个分类。

**BB**

国外的知名投行业内称之为 Bulge Bracket，简称 BB，就是我们常说的九大投行，分别是：Goldman Sachs、Morgan Stanley、JP Morgan、BAML、UBS、Credit Suisse、Deutsche Bank、Barclays、Citi。

**非 BB**

非 BB 的比较有名的主要有HSBC（汇丰银行）、BNP（法国巴黎银行）、Nomura（日本野村证券）。

**Boutique**

精品投行,是专注于某一比较细的领域，往往指的是专业做并购、上市、发债等细的领域。在国内比较有名的是：Rothschild（罗斯柴尔德）、LAZARD（拉扎德），国内的证券公司也可以分属此类。
