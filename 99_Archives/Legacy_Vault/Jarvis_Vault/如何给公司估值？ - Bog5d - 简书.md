# 如何给公司估值？

因近期要跟某合作商谈城市合伙人的事情，所以开始计划合作条款，期间遇到公司估值方面的知识盲区了。

做了点功课，发现公司估值方式还挺多的，什么pe，ps，ev估值法。 这我还是没弄透，ps估值法稍微懂了点

![unknown_filename.1.png](./_resources/如何给公司估值？_-_Bog5d_-_简书.resources/unknown_filename.1.png)

看了看爱尔眼科的城市合伙人案例，
爱尔眼科跟城市合伙人的在股份划分过程中，需计算公司估值时，采用的是ps估值法（市销率）。

简单说，就是跟年销售收入挂钩的。
比如你的年销售收入预计第一年是1000万，第二年是1200万，第三年是1500万，可以折算年均销售收入为1233.3万。 （年均可不是估值哦，毕竟入股分红，又不是只分一年）

再乘以ps倍率，假如是2，那么估值就是2\*1233.3=2466.6. ps倍数可以根据销售收入规模来分等级。

![unknown_filename.jpeg](./_resources/如何给公司估值？_-_Bog5d_-_简书.resources/unknown_filename.jpeg)

如果此时，你拿着100万想要入股，是不是基本就能清晰自己的股比了呢？
