# 杭州小记

### _杭州小记_

前些天去了趟杭州，单纯地去玩玩，来上海半年了，还没有解锁过周边的城市，于是杭州成为了解锁的第一个城市。
其实一个人要迈出去，也是挺难的事情。 有时候想出去玩，到哪个地方去旅旅游，要给自己很多理由才能出门去。
_比如，那边天气好不好，到底有什么好玩的，好吃的，谁会跟我一起玩，需要有很多前提条件才会出发。_
我出发前，在犹豫要不要出时，就只确定一件事，走出去，一定会有别样体验。 然后买票出门了。
结果我都干了些啥呢？

* 看书和思考。 出发时，有种旅游心态，思考问题时质量比平日高了很多，有一种跳出来审视自己的感觉。
* 到了杭州，去美食街解锁美食。（但实际上，不算什么特色美食）
* 结交到得到学友，特意外。 他看到我的电子书上面的猫头鹰LOGO，知道我也是得到粉，相聊甚欢，还力荐我去得到大学。 因为得到大学，是需要引荐人的，他可以帮我引荐。 （换做以前，我立马去了）
* 跟小姐姐游西湖，独自冒雨登宝石山。
* 解锁美食，葱油爆虾、葱包烩、奎元馆的虾爆鳝面。 据说那葱包烩的典故是用面皮把秦桧包在里面吃掉，所以在民间流传下来。
* 还不忘给重庆的小伙伴买点特产寄回，给自己点个赞
