# 当前军用屏蔽视窗广泛使用不锈钢丝网夹于两层玻璃之间，比如将不锈钢丝网镀黑色镀层，…

当前军用屏蔽视窗广泛使用不锈钢丝网夹于两层玻璃之间，比如将不锈钢丝网镀黑色镀层，或在高透光率的pet表面镀1.5微米铜层然后进行蚀刻，目前工艺成本高，残次率高，不能做大面积蚀刻；
与国防科技大学合作开发，利用纳米压印光刻技术制出高透光的（透光率大于90%）电磁屏蔽视窗；
技术和工艺已迭代到300mm到1300mm，将该技术不仅用于高端军用屏蔽视窗，未来有望替代进口的多孔铜箔。

电子电力新能源产业飞速发展，铜需求迫切，但我国铜矿紧缺，铜价高昂，行业亟需铜包铝结构降低材料成本；
铜铝复合箔现为卖方市场，相对传统铜箔有极大的成本优势，可替代目前民用电力市场，新能源汽车电池市场（锂电池），太阳能电池市场主流在用的传统铜箔；
在具备更好的导电散热、节能、扛电化学腐蚀等优点上，成本仅为传统铜箔的一半；
