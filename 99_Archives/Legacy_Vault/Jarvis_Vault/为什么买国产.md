# 为什么买国产

很多人觉得，爱国就是要大声喊出国产手机世界第一。但我不这么认为。即使知道iOS系统在某些方面可能比安卓更优秀，我还是会选择购买国产手机。为什么？因为对我而言，真正的爱国是在可以选择的时候，依然选择支持国产。
我现在用的就是国产手机，小米、华为，你们可能都用过，或者身边有朋友在用。其实，这些国产手机真的挺不错的。我在这里也想呼吁一下，小米粉和华为粉不要再互撕了，我们应该骄傲，因为这两个品牌都是中国的骄傲。
不仅仅是手机，我还想呼吁大家，在日常生活中尽可能地支持国产产品。我敢说，绝大多数国产货在质量上都不输给进口产品，价格还更亲民。服饰、洗衣机、冰箱、空调，国产的品牌和产品都有很多优秀的选择。
说到中国高铁，大家都觉得很骄傲，对吧？那为什么在买洗衣机或者家里装修的时候，我们就自动地认为西门子比海尔好呢？我最近装修房子，装修公司一开始推荐我全套进口电器和材料。我跟他们说，我要买国产。他们跟我说，电路安全这些最好用进口的。我当时就不乐意了，我说我要最好的国产，这是我的选择。
再说到买车，如果你真的因为工作需要，必须要买几百万的豪车，我也不好说什么。但是如果就是普通的家用代步车，二十万以下的选择那么多，为什么非要选日本车呢？到了2020年，还有人觉得日本车高档？在我眼里，除了BBA，其他的车型，无论是丰田本田，还是比亚迪吉利，差不多一个档次。但性价比呢？完全不是一个级别的。
所以，我建议大家，在选择产品的时候，能买国产就买国产。实在有些产品国产做得不够好，那也可以选择欧洲货，尽量不要买美日韩的产品。对于那些只买美日韩产品的人，我想送他们一句话：'商女不知亡国恨，隔江犹唱后庭花。'
