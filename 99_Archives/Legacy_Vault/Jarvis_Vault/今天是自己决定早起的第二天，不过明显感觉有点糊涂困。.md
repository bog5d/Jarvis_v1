# 今天是自己决定早起的第二天，不过明显感觉有点糊涂困。

今天是自己决定早起的第二天，不过明显感觉有点糊涂困。看文章时，大脑不太专注。

之前留言，总觉得是给别人看的，随着留言次数增多，越发觉得是帮助自己梳理一下，看完文章后，大脑留下了多少营养的。 

过去三年，一直每天坚持写日记，哦不该说坚持，是享受写日记。 

日记这事儿，其实就是自己跟自己的对话，不过我并没能从写日记这事儿里面，提升多少写作能力，而是自省能力。 经常会在日记里，为白天发生的事情，懊悔。 

对个人性格帮助，还是挺大的。

不过我写日记太花费时间了，于是最近改为语音日记了。 十分钟内，把今天的一些思考记录下来。 

不过换为语音日记之后，我又担心没有经常用文字，自己的文字能力会下降（虽然也没觉得自己之前文字能力有多少进步。）

看了文章后，我发现自己原来思维方式里，有误区。我内心默认了自己，这几年写日记，无形中提升了自己的文字表达能力，但这也许是个误区。 思考习惯或许是有长进，文字能力倒不是必然有长进的。

我从没有过，格式化写作，也没有过类比，韵律的讲究。包括分段落这件事，不过也是随意为之（咦，突然联想到以后要写文字日记时，可用markdown，分段清晰些）

哈哈，看完这篇文章的收获：

1.决定在以后的文字日记中，使用markdown

2.看到好的例子，搜集到印象笔记中。
