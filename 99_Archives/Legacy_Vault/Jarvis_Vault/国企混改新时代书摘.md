# 国企混改新时代书摘

![Evernote Snapshot 20220127 144613.jpg](./_resources/国企混改新时代书摘.resources/Evernote Snapshot 20220127 144613.jpg)

国企改革里面有三个点很重要，第1个点是管资本为主，第2个点是划拨，部分国有资本充实社保基金，第31步是发展混合所有制，那这个就意味着说以后政府他可能跟着作为资本方了。
这一段也把为什么要做，呃国企混改的背景，讲来讲就是说，我们国家国有资产的管理体制其实是没做好，就是没有，就是这个客体是被搁置了，因为在我们产业里面，我们国家的市场形态里面实物形态的国有企业，它仍然是我们国家经济的主要形式，也就是说各级政府它首先作为市场的管理者，同时也是这些企业的拥有者管理者和控制者，那么这这个事他会形成诸多的矛盾，而而那个混改的方向应该是想把国企它作为这些把政府作为这些企业的拥有者，但是他们不作为，经营者他们拥有的方式就是用资本去控制和拥有这些企业，但并不管理这些企业。

这是为了实现政资分离和正企分开，简单讲就是想把这些企业的所有权和经营权分离，这个倒是一个大格局的事，因为当下很多国企里面的员工他作为经营者，因为这里面没有太大的动力，所以经营的并不算好，但是这些企业资产呢又得是国有资产，最好的办法就是，国家，他当董事长，请职业经理人来管这么多国有的资产。
![Evernote Snapshot 20220127 150024.jpg](./_resources/国企混改新时代书摘.resources/Evernote Snapshot 20220127 150024.jpg)
这儿有一个新的知识点，就是我们国家目前国有资产的管理形式，是分两种。
第1种是，像全国社保基金这样的国家队的，资产管理公司，来对这些国有资产进行控股，这种控股的方式就比较灵活。
第2种则是国资委全权管理，就是管人管事，管资产，就是国资委既要管企业又要管市管资产全报完了，这个很不理我。
最重要的是什么呢？谴责他管理的这些，股权，用股权的方式扣你资产，这些股权资产它是具有流动性的，还可以灵活的，在市场里面进行交易，而国资委管理的这些国有资产它是不具备流动性，所以它的资本效率就很低。
