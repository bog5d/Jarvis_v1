# 我想要起草一份协议，你是资深合同专家，我想起草的协议要点如下：

我想要起草一份协议，你是资深合同专家，我想起草的协议要点如下：
1.甲方为出资人，乙方为募资人；
2.募资人为了自己在股权投融资方面的事业拓展，需募资个人发展用的资金，这些资金用于支持募资人在上海开展业务时的生活和差旅费用；
3. 出资人交由募资人的资金，募资人以借款形式，作为出资人的债权；
4.同时，募资人赠与出资人分红权，募资人以自己未来在股权投融资的项目收入作为分红池，出资人按照每5万享受分红池中1%资金为基准；分红受益期8年，分红受益时间以募资人第一笔项目分红到账开始计算8年；
