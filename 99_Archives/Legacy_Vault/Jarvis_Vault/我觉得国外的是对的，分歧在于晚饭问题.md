# 我觉得国外的是对的，分歧在于晚饭问题

“ 绝招二
如何让刚出生的小宝宝一觉睡到天亮
美国医生再三批评我：你不是好妈妈，隔壁的孩子一睡到天亮，隔壁的妈妈是好妈妈，你不是！每次去看家庭医生我都挨批评，原因是我晚上要喂奶1-2次，抱怨养孩子很累很辛苦。医生告诉我一个观念：人类晚上不需要进食！我傻乎乎问，为什么呢？医生没办法，只好大声说：因为我们是人类！我印象极深，妞爸早就笑弯腰了。很显然，我担心BABY晚上会饿根本是多余的，10-12小时的连续睡眠，对孩子来说比吃奶重要100倍！我们被老大妞妞折磨很惨，晚上吃一顿奶的坏习惯一直延续到快2岁。养老二的时候按照医生说的，彻底让我和老公解脱出来，从刚出生到一个礼拜我家妙妙被训练距离5个小时才吃奶，2个礼拜之后，孩子就能睡天亮了。晚上不需要吃奶！！！（三个感叹号是请妈妈们切记的意思） ”
我的观点如下：
  让新生儿一觉睡到天亮——这根本不科学！！！（三个感叹号表示露珠很激愤！）
刚出生的宝宝无论是生理需求还是情感需要上来说，都不可能长时间离开妈妈的怀抱和奶。一晚上喂奶1-2次算什么？这就需要抱怨养孩子很累很辛苦了？那个医生的观点真是太好笑了，人类晚上不需要进食？成人的胃和新生儿的胃可以比较吗？成人也要吃晚饭的好吧！而且，成人睡觉的时候如果饿了，可以忍忍就算了（更不要说有的人根本忍不了），新生儿能忍吗？新生儿根本无法控制的情绪，饿着能睡整觉吗？
睡眠确实重要，但是对于新生儿来说，他们的睡眠模式就是一两个小时，两三个小时一醒。没必要要求一个新生儿一觉天亮！这一点无需训练，因为宝宝自己就会慢慢夜晚睡眠的时间长起来的。我家宝宝从来是自己想睡就睡，想吃就吃，大概在一个月左右，就能从晚上12点吃完一次夜奶后，睡到凌晨5点再吃奶了。
我看过很多断夜奶的例子，有些断得太残忍了。我没给宝宝断夜奶，不是怕宝宝做不到，是我就做不到。我宝宝现在每晚从八点开始睡觉，大概两个小时会吃一点点，他没有真正醒来，只是闭着眼睛找奶，我会给他吃一点，他在梦中吃两口，一来填填肚子，二来感觉到安全，然后会睡得更安稳。如果宝宝要就给，不要就让他睡，没必要弄醒喂奶，但也没必要他要吃奶却偏偏不给啊！！！
 
最后，法国医生的建议是按时喂养，新生儿每三个小时吃一次母乳，一边吃10分钟，两边都要吃。这个方法我做不到，我还是支持国内的按需喂养，两种方法都是对的。不过由此也可见，法国人的观点是在母子关系中，重要的母亲，母亲的乳房要得到充分的休息才满足宝宝的口欲，而我们国人的观点是母亲是为孩子牺牲风险的，宁愿自己辛苦也要把孩子放在第一位。
