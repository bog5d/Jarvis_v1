# 争名字 王梓

哥哥给萱萱取名，王梓萱。 老高总说哥哥剽窃创意，因为自己曾经给哥哥说过，想给自己娃儿取名王梓，然后哥哥就抢了她的创意。
我听这个观点后，我就跟哥哥通话中，说到觉得他剽窃了老高创意。
然后，和哥哥大吵一场。
 后来，哥哥说到，妈妈这几天都忙晕倒了，我却在这里争什么名字创意，真的有点不懂事。

现在想来，我特么是怎么了。

为什么那么听老高的呢？

就是怕她把事情闹大。但为啥她的观点我会信呢？

总觉得曾经的自己就他妈是一坨屎。没有自己的底线，完全是无条件妥协。
跟老高的相处，让我和家人越走越远。
