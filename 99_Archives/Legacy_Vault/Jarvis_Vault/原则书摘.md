# 原则书摘

![Evernote Snapshot 20180412 090652.png](./_resources/原则书摘.resources/Evernote Snapshot 20180412 090652.png)

冥想 真的很重要，在那么多地方看到冥想带来的益处。 

我自己目前体验冥想带来的好处是：冥想完的一两个小时内 心超静，不浮躁。

但坐不了那么久，腰不舒服。 

![Evernote Snapshot 20180412 091247.png](./_resources/原则书摘.resources/Evernote Snapshot 20180412 091247.png)

帮助别人理性：这一点其实不难做到。 用之前提到的两分钟法则，用两分钟重复对方观点“你的意思是不是要做A，因为原因b，c。 ...... 而我的观点则是要做W，因为xyz....，我们观点的不同之处是V”

如果做到这一步，能够极大地避免杏仁核绑架。

![Evernote Snapshot 20180412 093549.png](./_resources/原则书摘.resources/Evernote Snapshot 20180412 093549.png)

人们行事风格迥异，原来真的可能是思维方式甚至是大脑构造等硬件上面的差异。 

所以，我们在做团队安排时，非常非常有必要，做好性格测试，数据化地去了解这个人。 然后安排正确的人，到正确的岗位。

要把王海梦，唐刚，王思密，李大泽 李锟都拉进来，好好地做一下性格测试。
