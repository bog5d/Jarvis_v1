# 铸网风暴--建立设计师无偿回扣制度

上次我思考了关于建立设计师在自己负责的业主选择了美的厨房电器，就无条件得到回扣。
此计划本次正式命名为:铸网风暴。
铸网风暴不是简单的制定一两份制度就能搞定的，需要一个系统的工作计划才能达到效果。
每一个设计师就是一个大点，这个大点能辐射连接十几个业主，每个业主又能推荐另外的业主选用这个设计师，所以就形成了一个以设计师为纽带关系网。
不过这个关系网并不是我们美的所有的，故，我们需要拿下这些关系网，也就是铸就利益网链，形成稳定的价值体系，设计师愿意主推，我们主动分配利润。在大竹卷起一场“铸网风暴”！！
这必须分两步走
一，制定有利润空间的统一标价。并为所有县城和乡镇网点，贴上这些统一价格签。
二，打听我们用户的设计师，为设计师主动送上回扣和操作规矩。
