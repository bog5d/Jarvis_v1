# 0614 生日快乐 真正感激vs勉强迎合

1. 光写日记不看
	1. 这日记是倒着写的，先写的第三部分 ，真正感激与勉强迎合。不过我现在写日记的动力不足，我觉得很大的原因在于自己没有去看自己写的日记，因为自己都不为自己的日记感到惊喜或者不够珍惜日记，所以不太坚定的去记录自己的生活。
	2. 其实，无论自己的笔记，还是搜藏的印象笔记的重要摘抄都可以随时翻看，有些文章真的很有用的。养成没事看笔记的习惯，很重要。
	3. 翻看五分钟计划。也就是等人呀，无聊的时候翻看一下的。就像记单词一样，利用碎片化时间。
2. 开车总结 刹车不踩
	1. 今天小马哥坐我的面包车，在路上提醒了我几点开车注意事项
	2. 多使用三档以上，因为低档位的齿轮小，跑同样的距离，会需要更多的转数来支撑，会费机油和齿轮的。
	3. 我在起步的时候喜欢踩着刹车，启动一点离合之后才会放开刹车，然后马上就加油门。其实用不着这样的，平地上起步直接放开刹车，只松离合，车子就能起步了，然后再给点油，基本上问题就不大了。
3. 小马哥陪看电影
	1. 说说老马吧，今天生日嘛，老马还是挺给面子，我大概一点过的时候从旗舰店回来，老马本来在耍游戏的呢，见我回来还是出来陪我聊聊天啊，说说话呀的，不过刚好我想上大号，就中断了 。
	2. 然后他跟我约好晚上去看电影，我说好，大概五点多的时候就过来看我还在睡，就回自己房间了，
	3. 我起身后，说走嘛去看，他问了一下场次，我一查，咦，我想看的 明日边缘 和X战警都在，而且一场是在六点五分，一场是在八点一十。所以我们就去看了两场。
	4. 老马还说要坐我开的车，呵呵😊，那就去嘛，我觉得他愿意坐面包车真的是想给我面子，也想看看我开得怎样。他觉得这样更有意义点吧
4. 过清闲的生日，适合的
	1. 今天生日，本来要去参加装家军团小区摆展卖卡的呢， 不过要先去旗舰店协助开业活动，就去了，因为下雨就没有去小区摆展，看来是老天要我休息休息，感谢天意眷顾。
	2. 然后我心中最想要的生日就是下午舒舒服服地在家里，打打坐，思考思考人生呢。不过王海梦和王斌却安排好了晚上要去吃饭，然后K歌。还送了我一个钱包，挺好的。 我心中还是蛮感动的。不过我的确不想晚上再去耍呀，唱歌🎤🎤那些。我就告诉梦梦说，我最大的生日愿望就是在家好好地清闲清闲。我知道，说了这些，梦梦和王斌肯定会有所失望的，但是如果我为了接受他们的好意而不去说明我最想要的，是不是从内心里面更不坦白了，而且自己也不会从内心去感谢他们。所以，我觉得今天说明了自己不想晚上去，而改成了中午去吃。然后中午就一起吃了火锅，中途大家一起聊天聊得挺愉快的，大致就是我们摆游戏，李大育知道的游戏确实不少，我们聊鬼泣4，但丁，维吉尔。觉得挺好的。
	3. 经过这事情我觉得：一。有时候你想法设法去给别人的，不一定是别人最想要的。你想努力给的，并不是别人真心需要的，因此多真正的去理解别人的需要，比如下次我要给朋友生日惊喜，我真的应该好好关注一下别人的状态，而不是去想我平时是怎么过的生日，或者怎么耍的。二，应该提出自己不想要的，别因为对方是好意就一定要接受。如果对方的好意，不是自己想要的，就非常正面的告诉对方，特别感谢他的好意，我内心真正想要的是什么，如果我接受了你的好意却违背了我自己的心意，相信也不是你想我达到的效果，而且我不能从内心里面去感激你，甚至会觉得是我在迎合你的好意，所以如果我们改一下的话，我肯定会从内心里面更加感激你的。。。。
