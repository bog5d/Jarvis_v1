# 老是推迟

我只想说，每天想要写日记，都是拖拖拉拉的。一旦看到手机屏幕就被屏幕控制了眼球，不能正常思维，正常行事。

其实吧，写日记这事儿，我越来越发现是，更多的记录下的东西，就是离写日记时间发生的越近的想法和事情则被记录下来的概率更高
![图片.2.jpeg](./_resources/老是推迟.resources/图片.2.jpeg)
在网上买了耳机，我觉得好大好沉哟。

![图片.1.jpeg](./_resources/老是推迟.resources/图片.1.jpeg)
![图片.jpeg](./_resources/老是推迟.resources/图片.jpeg)

你说日记我到底该记什么呢？本来是想记录清楚，但是以叙事的方式记录的话，会让在写日记这会儿特别无聊。
但如果，我只想写下我此刻的感受和心路历程，在以后回过来看日记的时候，是不是会特别无聊呀？波哥同志。

我太希望能活开心了，不吵架是很幸福的事情哟！

今天又和老高大吵一架。至于起因经过，我都快记不起来了。
