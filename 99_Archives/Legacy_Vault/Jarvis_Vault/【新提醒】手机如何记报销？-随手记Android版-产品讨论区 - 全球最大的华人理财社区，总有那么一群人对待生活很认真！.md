# 【新提醒】手机如何记报销？-随手记Android版-产品讨论区 - 全球最大的华人理财社区，总有那么一群人对待生活很认真！

很多同学不知道报销的款项要如何来记账，今天就让桃子手把手来教大家吧~！n\_n~!

首先要明确一点，报销是我们帮公司垫付，不能算作个人支出;报销回来的钱是公司还给我们的，也不能算个人收入。所以，**报销在财务上通常处理为转账，不体现在收支中。查账时直接检查报销账户余额和流水即可。**好了，明白了这一点以后，我们一步一步来记账吧！

第一步：**建立**一个**“报销”的债权账户**：随手记首页——》账户——》右上角添加，名称：报销。类别：债权账户

**记报销的支出**：首页点记一笔——》正上方菜单——》代付，银行卡（或现金）转 报销账户
