# 原来单身的时候是成长最多的时候。

有时候我也在想应该趁着自己单身的时候，好好的成长自己，就是无论你是在有伴侣的状态或是恋爱状态还有家庭的状态，其实个人成长是最慢的时期。

现在自己是单身的状态就应该把这个机会用起来，因为有大把的时间是投入到自己身上，我应该珍惜自己还在单身的时候

有这种大把大把的时间用在自己个人成长个人事业上的这个黄金阶段。

这也是为什么我总是在担心自己又陷入到下一段感情里面了。 
有感情需求，但是又怕进入之后过不了自己的人生。
