# 思考

思考与行动
“特立独行且正确”与“自以为是”的区别在哪里？“自以为是”为什么常常是危险的？

> * 特立独行且正确，是观点被别人否定过后，自己再次验证后发现自己是对的，进而更坚持。  自以为是，是观点被否定后，不去验证，而是把反对者当作敌对情绪去看待，别人越不认同，处于对面子的维护，更加去维护观点。 
> * 两者的动机区别巨大，特立独行且正确是想真正追求更加合理；而自以为是则是在意别人是否认同。
> * 自以为是当然危险，因为会把自己锁住，进入为了维护正确而维护，不会吸收别人的观点，不会多角度去看待问题。直接危害：一则自己认知不会进步。二则，会拥有糟糕的协作关系。
> 	

你有什么观点是“特立独行且正确”的？在这条路上付出过哪些代价？

> * 哎哟，这个问题我找不出什么大安全，难不成是因为这么多年，自己都没有自己核心的观点吗？ 自己都是随波逐流之人吗？  想想也真是。
> * 想到一条：我坚持认为打牌是一件很浪费时间，且没有重要价值的活动。所以自己一直不打牌，就算有很多人想要教我麻将时，我都不愿意学。 
> * 因为之前的客户圈子都打牌，我不打。其实少了很多社交机会。 所幸的是，自己因为不打牌，有了更多时间做想做的事情，游戏，阅读......   其实代价真的没什么啦，不过就是去开会时，他们显得很有玩的，我则是回酒店听书睡觉。关键是真的无损失感。

如何才能从“特立独行且正确”的观念中获得商业价值？（而不仅仅是一时的快感，即，所谓的“仅供刷刷存在感”……）

践行自己坚定的观点。对呀，如果一个观点，我自己觉得很对，坚信不移，那么自己一定是在去践行的路上了呀。 比如坚信要活很长，终生学习很重要，那就去做呀。 比如坚信大数据是未来，那就起学呀。 没有抛下一切去学，本身就是此观点的坚定程度并没那么高。

若是你真的能够明白，“很多的时候正确本身然并卵”，那么你的性格脾气都可能发生哪些变化？
通过是否愿意践行，来验证自己的观点。

你身边谁经常有“特立独行且正确”的观点？其中有哪些观点值得你认真研究？他有什么特点？他又是如何做到的？
