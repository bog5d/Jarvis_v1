# 10月26日 感触颇深。

![图片.jpeg](./_resources/10月26日_感触颇深。.resources/图片.jpeg)

昨天听了罗辑思维。本想把当时看了的心得写下来的，但是 总是在晚上时觉得身体疲乏无力。再次需要自己早睡，和午睡了。这个课题跟随自己太久了，也没给解决。
偏话少提，我真正关注的时罗辑思维在描述张勋复辟帝制时，失败的原因。最重要的就是段祺瑞不买账了，本来段祺瑞要给支持的，因为段祺瑞只想干掉黎元洪嘛，   到后头，张勋复辟了，给段祺瑞封的官比自己小了很多，这让军阀头子段祺瑞受不了。成为很主要的一个原因。
所以罗振宇在总结的时候，就提到一点
:你得建立一个利益结构，让这个结构里的每个人都得到比原来更多的利益，才会持续得下去。说白了，做一件事得让兄弟们能有好处，不然内讧大大滴有。这里面我就有几点心得了。

* 大多人内部被打败

1. 最近看三国演义，很多军阀最后被干掉都是内部人背叛，而背叛的很多人，共同原因就是跟着老板没吃饱。比如陶谦的兵看上了曹操老爸曹嵩的钱财把曹嵩给杀了，酿成大麻烦。 郭汜 逃跑时，也是胡车儿相约叛逃，分赃不均又被杀。还有吕布在下邳被围时，殴打献酒的下属，结果下属外接曹操，直接给捆了。这里面好些人，都还是能够突围或者有机会活过去的，都因为内部分裂了。
2. 但像曹操这样的，大军阀，兵多势众。却没见过多少叛变的将领，大部分人都是忠心耿耿的，至于张绣杀了他典韦和曹昂的叛将，那确实是曹操睡了他嫂子。没给够足够的尊重。  能把内部打造好的人基本上还是不会太惨。
3. 至于怎么打造好内部，就是利益分配合理，当然前提是赏罚分明。😄干得好的不要吝啬钱和赞美，即物质和精神犒赏。赏罚分明，切记只罚不赏。重奖轻罚！

* 曾国藩的成功学，严己宽人。

1. 曾国藩在以前团练湘军的时候，就跟当地的巡抚那些合不来，老夫专事奏责之事，长期弹劾那些让他不顺利的官。并且要粮饷大权，才会继续干，就那样跟皇上较真。后来皇上有了备胎，加上形势也没那么急了，就把他冷落起来了。
2. 第二次出山后的曾国藩俨然换了一个人。出山前把大大小小的官员都给拜了一遍码头，很是尊重同僚。更重要的是发生了一件事对我启发很大，就是他们打下了太平天国之后，要报账。负责报销的部门官员肯定是要吃回扣的，他们谈好了回扣值，大约八百两。可谁知道恭亲王，后来说，国家衰败成这样，这个账就先不报，挂账上，日后再报。 财务官员们就傻眼了，不过曾国藩呢，觉得虽然账没有报下来，不过之前谈好的八百两还是给送过去。不落下话柄！这真的是个人才。
3. 还有在之前攻打下了太平天国后，曾国藩手下的将领，把财宝洗劫一空了。但皇上要追查财宝哪里去了，曾国藩坚持说算了，都是苦兄弟，苦出身。没了就没了吧。皇帝  恭亲王他们也拿他没法，只好作罢。他肥了兄弟，自己也不会太过损失的。 很可能他在给财务官员送八百两银子的时候就是从那个里面拿的钱呢。
4. 手下送礼，也不装清高说不拿，就取个十分之一，表示收了。下面的人有面子，自己也不太过违背本心。

* 主动减少利润，我自己和史蒂芬裁缝。
	* 罗辑思维提到两个他朋友的事情，其一，他朋友签了一个大单，利润特别高，高过了行情。罗振宇告诉他朋友，赶紧改签合同，不然这生意只能做一次。 他另一个朋友承包项目时，发现自己提供的价值低于了合同约定的价值，按理说合同都签了，就算自己赚了吧。可他那朋友，主动退了一部分钱给那个人。我觉得干这事儿很有逼格，也是我很佩服的地方。
	* 这让我想起了，我自己在川农作二手书生意时，那时候，我们收购作废的书，然后把有用的专业书籍留出来，单独一本一本的卖。而那些公共课程的书，我们就拿来卖给废品回收站，然后我们拉了一车废书给回收站，回收站的人肯定还是压了价格的，这我知道，但是我发现我们的书最底下一层是被水淋湿了的，然后我又折身回去，还了那老板娘1.5元钱。就走了。我内心觉得自己干这件事挺帅的。而且我希望自己一直保持这种有逼格的个人风格。尽管有人会说你装呀，你傻呀别人反正挣了你钱的呀，之类的，但是我退小利，得大智慧，我自己觉得值那肯定就值。
	* 还有卡耐基这本书上的，史蒂芬裁缝，主动提出还有漏洞，跟员工主动承担犯错后损失的钱，这些都让客户或雇主更加信任。也让自己更加地真材实料，我觉得可以坚持这种逼格。
