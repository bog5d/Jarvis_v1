# 阴影记忆之张杰的耳光

六年级时，应该是1999年的样子，从爱群村小，搬到了麦头山中心小学。
刚过去的时候，因为是从很多地方村小来的，所以我们原来的班被拆散了，我分在六年级二班，当时张杰也是跟我同在二班，张杰和我是村小时的同学，成绩不错，二三名吧，反正挺厉害，就是有点心胸狭隘，脾气强硬。  不过，既然同在一个陌生地儿了，我两就亲近了，每天放学一起回家，然后吃饭也一起，我到哪儿他到哪儿，他到哪儿我到哪儿。
我也知道，搬了新环境，我自己心境也是变化很大，我很清楚的记得搬到中心小学后，我的虚荣心被前所未有的放大，特爱做出各种动作博大家关注，而且习惯装得自己家境不错，我猜是内心不自信，俗话说缺啥补啥嘛，就装作自己很幽默，装作自己很受人欢迎，明明跟一些人不熟，却要在人多的时候装出一副跟人家可亲近的样子和人家开一些熟人之间才有的玩笑，比如摸摸屁股拉（因为这个还被一个叫做辜粱鸿的用铁丝钩打过）。唉，反正我知道那时候的我着实有点讨厌，也好那时候的我的把该讨厌的事情都经历了也算是好事。反正就虚荣心特强。。。。
心态背景交代完毕了哈，
我记得那时我的成绩在六十多人中排名三十多吧，张杰排名十多，反正女生里面成绩好的就是女神，比如辜瑜曦啦，杨红霞啦杨欢啦这些，然后就长期把成绩好的男生和女生拿来配对（早期的八卦），有个同学就在那儿说“辜瑜曦和李红兵是一对，杨欢和杨凯是一对…………”，反正就是按成绩水平排的，在说这个的同学把那几个好成绩的女生（也就是女神些）逗得哈哈大笑，我也想博女神一笑啊，就打起兴致，对女神些说“对啊对啊，辜瑜曦和李红兵，杨红霞和张杰吧，哈哈”，张杰也在旁边之前说别人时他笑得很欢，现在提到他了，他表现出一副特别不快且冷酷的样子，看着我问我“你再说一遍”，然后怒视着我，按常理我应该识趣的打住，谁让旁边有一大堆女神啊，而且这些女神还是乐呵呵的在笑着呀，我不可能表现出一副弱者像就打住呀，（也为了防止尴尬，我继续笑着），又说了一遍，结果张杰站起来，按着我的头，像电视剧里面的那样狠狠的给了几巴掌，口里恶狠狠的说着“你再说一遍，再说一遍”  ，可恶的女神们还在笑着。。。。。。。
其实那几天心里好难受，每天上课脑海里想的都是如何把张杰打得半死不活的，然后又担心因为打架，自己被开除拉，在想到父母的难受。。。反正就是想得特别多了。那段时间的心路历程，真的是特别黑暗，还好我脸皮够厚，没有因为这些而种下自卑情节，厌世心理。很大的原因在于跟成绩最好的李红兵成了好哥们，心中的自豪感加深了些，同事也因为这样找到了自己的朋友圈子，就固定下来了，接下来几年里面，李红兵的一些正能量还是影响了我很多的，从这一点我真的应该感谢李红兵。
