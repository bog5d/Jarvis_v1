# 10月31

人们恐惧的往往是恐惧本身

* 这几天下了大竹，是等老高从成都回来了，我直接去火车站接的她。至于要等她回来了就接她并立马下大竹呢？原因是，如果我在上班，她从成都回来的话肯定会在我这儿赖着耍两天再下去。这样我就担心王斌他们闹意见了。
* 然后就是在门市上，呆了两天。前天在门市上和冷瑞李博邱艳开了开会，主要就是给了大家一个愿景，我告诉大家，我的志向是去拿下一个城市的总代理，所以大竹是我的学习场所，几年后我总会离开大竹，大竹的事业是会交给大竹人的。所以我告诉邱艳让她把美的可以当做一个事业来做，如果我要开二店的我需要融资的时候，可以考虑让她入股的。不过我只会会把入股比例控制在30%.
* 并告诉了李博，如果我高升了，我会尽量带着他。希望他好好成长！我说的这些其实是我心里话，不过有一点是需要机遇的，比如拿下一个地级城市的总代理，这是有运气成分的。不是光靠个人努力能达成的，不知道这算不算妄念！把自己的市场做好吧，然后成为标杆。【其实，我是不是该问问自己，拿下一个地级城市总代理，我还欠缺些啥？缺啥就补啥呗。】

  FIGHTING

* 还有昨天是王斌生日，下午吃蛋糕时，冷瑞和王斌互相摸蛋糕，上演了一出奶油大战，肚子给我笑疼了 ，昨天晚上还去KTV唱了歌，挺好玩儿的。我唱了曹操，感觉还是挺有力的，不过应该多学几首歌曲了不然去K歌都不知道唱啥了。
* 对了老婆觉得我的交际圈子质量太差了，都是不够上进的朋友，这是有点道理的，我内心也反感那种可以去结交有利的朋友，不过主动结识优秀的人肯定会成长得更好，因为高人的思想和自己的思想杂交后是会有化学反应的。一个有上进好学的人，会从好思想里面吸收积极正能量的。

        我觉得有必要多联系联系优秀的前辈或同龄人，把自己的自满洗掉，找到人生方向。

* 另外一点是，这几天性生活越来越没劲了。对老婆提不起兴趣，好像是做爱钱顾虑有点多 1，担心老婆下面太干2，担心小鸡鸡在专心为老婆抚摸后硬不起来3，怕插入后隔了包皮的小鸡鸡会受挤压而疼。4，怕到时射太快，没面子。唉！！性生活都有点不和谐了。我想买海蒂性学报告来研究研究了。
* 还有关于自己的恐惧，最近胃疼，我推测是十二指肠溃疡，反正肚子一有点饿就心口位置疼。我就开始胡乱担心了，总之就是为不知道的未来忧虑，怕老了身体不好，然后就在那里恐惧，唉！！！我们恐惧的往往是恐惧本身。好像那个牛奶炖香菇有奇效的嘛，我找找试试
