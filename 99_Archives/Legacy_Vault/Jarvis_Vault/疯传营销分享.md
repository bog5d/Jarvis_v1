# 疯传营销分享

王海梦：
李杰：
1:可视化:一种信息的传递方式，如何让客户帮我们传递产品信息(比如凸显我们品牌的独特性，勾起客户的好奇心等等)
2:社交货币，我们的产品要为我们的受众带来社交货币让他们为我们推广(如内在的吸引力，打折的方式过了这个时间点就不能享受优惠了，或者让他为我们推广而受益等等)
3:诱因，让我们的产品跟高频事件挂钩！
行动项:思考我们产品的社交货币，打造一个属于我们的，跟我们产品挂钩的高频事件，提高客户帮我们宣传的积极性
总结:做产品比拿客户大的多，所长我们要宣传我们的企业价值观，这样才能把品牌做起来
罗成：
1消费消费者行为剩余（可视化）比如鞋子要有特点让别人看得见，麦当劳，品牌衣服。结合我们公司可以让供应商发朋友圈，返一定购物卡，给我们做推广。
2和高频事件绑定营销：渴了累了喝红牛，不在状态和脉动，怕上火喝王老吉，送礼就送脑白金，睡觉之前听故事。结合我们公司，要做到当供应商缺钱就会想到我们给他融票。
3调动别人高情绪营销：比如美航毁了我的吉他。残疾人登山调动敬畏情绪，比如手机支持国产买华为，抵制苹果。男人的不容易，女人的无理取闹。教育改革等。军工行业的人很多都有爱国情绪，我们这个符合国家政策-让银行资金投入实体经济，解决很多中小型企业资金压力，核心企业也会有成就感。能和核心企业产生共鸣。
4要做品牌，有故事的品牌：比如三打白骨精就会想到西游记。武松打虎就会想到水浒传。爱情永恒，就会想到砖戒。对于我们公司，比如某某企业，面临倒闭，我们拯救了某某企业，解决了某企业的金融危机。
