# 想读原版书却啃的很吃力是吗？推荐一本入门书！

[![7d7810f5e5257a4a8c5a3d1ad4522957.jpeg](./_resources/想读原版书却啃的很吃力是吗？推荐一本入门书！.resources/7d7810f5e5257a4a8c5a3d1ad4522957.jpeg)](http://www.douban.com/group/people/ruby925/)

### 来自: [蕾梅黛丝](http://www.douban.com/group/people/ruby925/) 2013-09-26 10:24:32

每个英语读书爱好者都喜欢尝试原版书，但很多时候查了很多单词，连几页都读不下去，而且查出的单词也没什么实际用处。很多时候是即使单词都认识，读下来整句或整段依然不知所云。于是就想从简单一点的入手，却发现，读童话吧，单词一样不简单，还不实用，读杂志吧，不是相关领域理解不了，语言还很精炼，混着讽刺和隐喻，不在那种生活环境或工作领域的很难理解。读经典名著，那更是要有很好的英语理解运用能力，才能上升到欣赏文学的状态。本人就是这么一步步尝试过来，发现最后还是一本都没读下去，很苦恼。男朋友是澳洲本地人，看我总是读那些“大书”，一再摇头，说“你就不能读点正常的书吗，你的阅读水平也就是7、8岁孩子的水平，你读的那些童话年代太久远，应该读一下现在澳洲小学生的读物！”被他这么说我很生气，首先，神马叫做正常书？我读的哪里不正常了？再说了我阅读水平怎么就不行了？学了十几年英语，来澳洲读了三年护士专业马上毕业，雅思过七，怎么就不能看成人书，何况好多名著的中文版我还看过。。。。。。。然后男朋友给我推荐了一本Roald Dahl写的Matilda，让我每天读给他听（留邮箱，我会吧这本书的电子版发给你们，但是是epub格式的。你们自己也可以搜搜看，等我有时间把这个作者的其他书电子版找到也会发给你们）。开始我很排斥，因为这本书的内容讲的是一个很聪明的小女孩和她的crude parents。我一看便皱眉头了，问他怎么会有这样的书，写父母不好的。男朋友说我们生活在两个星球，我是来自党国天下，形成的思想自然不切实际。。。。。。。很多事情是真实存在的，不是被和谐掉就真的和谐的。而且Roald Dahl的著作是澳洲小学生的必读之物，每个澳洲年青人小时候至少都读过Dalh的一本书。所以抱着怀疑的态度，我决定看一看。一下午便读了近一半，除了少数单词不认识需要查一查，并不影响阅读和理解。并且故事有趣又贴近生活，着实有我小学时看的读物的感觉，类似郑渊洁爷爷的皮皮鲁之类的书，从孩子的角度看世界。我觉得完整的读完一本原版书并完全理解后对培养读原版书的兴趣和习惯是一个很好的入门，所以一定要找到合适自己的书。当然每个人的阅读习惯和话题兴趣都不同，Matilda这本书你们可以先读下看看，如果对胃口的话，可以试着读更多Dahl的作品。然后再一点点提高难度。我自己也会慢慢进入状态，然后让男朋友给我推荐之后12岁左右读物，16岁读物，然后分享给大家。希望很快，就可以和大家原版书爱好者或初入者一同成长，变成“成年人”！ （^o^ ^o^ ^o^）
