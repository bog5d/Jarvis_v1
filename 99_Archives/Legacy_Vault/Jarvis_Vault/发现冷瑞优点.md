# 0511 发现冷瑞优点

今天事情也特别多，但最让我感触深刻的还是冷瑞端正的态度，真的。
先说说今天发生的杂事
上午，老婆不满王海梦对我有暧昧嫌疑，为此争了一上午，我还是觉得自己有点稍微自傲了，对老婆的控诉还是不太重视，认为自己魅力还是吸引了女性的主动注意，这又不怪我。。。。。我知道自己有自负嫌疑了，但，我还是没法深刻的去妥协，或者去向老婆承认自己还是有享受小暧昧的心里。我，错了没有？我内心里不愿意去认为自己道德无底线，靠，纠结😖😖😖😖😖😖。男人有本事就变坏，好像我有点征兆。不过我知道，我不会对王海梦动心，但是绝不会迎合老婆去把她辞了，我真干不出来！！  先观察观察自己吧。

中午，周阳开车过来，坐他车一起来南充，然后找了个苍蝇馆子吃饭，和冷瑞住进了712房间。

下午，开售后会议
![20140511_164103.jpeg](./_resources/0511_发现冷瑞优点.resources/20140511_164103.jpeg)
讲了很多事情，零冷水啦，24小时服务啦，单台退换啦，封单啦。。。。。
还发现，那个鑫泰兴的张晓琳有点漂亮气质，哎哟，不错哟。 吃饭时，有个遂宁的小老板（好像是唐万成的下属），在向我说免费给用户清洗烟机热水器带来了大量的回头客，我问到假如找个方法报点单据结费，是不是可行的呢。那个小老板告诉我，如果你是老板的话，是不会在乎这点钱的。把旁边吃饭的张晓琳笑惨了，因为她知道我就是大竹的老板。。。。我也很淡定的笑道:哦，是这样啊 ，看来当老板就是不一样呀！！！
看他嘚瑟样，我笑了，因为当年的我也是这样。。。也笑我当年的傻叉。

好了，到了晚上就是重头戏了，冷瑞的进步。

总体来说，冷瑞是一个态度非常端正的人，说两点吧。 一是，那天我准备批斗王斌前的晚上，我问冷瑞，你对斌哥最满意的事情是什么？他很语无伦次的回答，他昨天去发了两万多的货。在问他对斌哥最不满意的事，我预料的是冷瑞会说自己受的委屈，结果他说就是这两天上班太不及时了，压根就看不到人。不管有没有做作，心态是为门市的。二，今天要通关考试，他从一开始就在背诵资料，我其实并不太看好他，甚至没打算跟他一组，我就自顾自的去背了，然后快到准备通关考试时，他给我电话问我到底跟谁一组，我就索性让他跟我一组算了。
结果他过来告诉我，我把这些都背完了，我心中一惊，是不是哦，来我考考看。
就给抽查了一下，嘿，结果算答上来了，因此我发现了，原来这男孩儿，真的很踏实的一个人，就缺正确的引导。
后来在要准备通关时，我自告奋勇放了第一炮，虽然冷瑞说的比较小声又断续得很，不过他卖力得很，特别希望得到认可重视。当然跟我的风格也比较合得来吧，我是一直愿意出风头的，冷瑞就不觉得愿意去表现会被我鄙视👎，而且出风头甚至会被我赞扬，所以就放开去备战通关，很享受那种努力了获得回报的感觉。我认为那就是吸收了正能量！

然后紧接着，我们又第三个出场，前面两组都没有过，冷瑞老是说普通话，然后忘了出示收费标准和未把蓝卡讲解到位。 不过大部分评委觉得可以通过，林红说严格说是很难让你通过的，吧啦吧啦的点评，说心里话我挺喜欢林红这老头的，嘴巴直，不记仇，语速快，像办事儿的那种，还有张天平是他副手，也很好的一个实务性人才。
然后，冷瑞特别关心分数，我就告诉冷瑞，在来这儿开会的大部分人当中，你一定是这里面学习吸收得最多的，我们不求拿到那个奖励，只求自己在过程中学到更多。。。你真的最赚了。
看他虔诚的眼神，我真的觉得他开得很值得。。。
