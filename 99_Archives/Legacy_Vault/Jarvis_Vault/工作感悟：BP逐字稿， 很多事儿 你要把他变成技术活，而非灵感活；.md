# 工作感悟：BP逐字稿， 很多事儿 你要把他变成技术活，而非灵感活；

工作感悟：BP逐字稿， 很多事儿 你要把他变成技术活，而非灵感活； 

  - 比如每一页写逐字稿时，就问自己问题，

  - 这一页必须表达的点是什么？  

  - 若没有感觉，不知道如何开始，是不是相关基础知识匮乏，若是，请研读相关知识？  

  - 还有哪些素材和案例可以支撑？   

  - 如何把这些素材和案例串起来？

  - 有没有可以删掉的废话？
