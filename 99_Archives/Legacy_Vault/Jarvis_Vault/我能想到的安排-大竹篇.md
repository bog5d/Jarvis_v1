# 我能想到的安排-大竹篇

今天再次和老马沟通了任务的事情，14年的大盘任务不是很乐观。老黄等客户的陨落，和大竹这个大客户的下滑还是影响整体出货。但是王平的崛起，还是给公司带来了
不少的利好消息，所以还好有涨有降，今年任务没至于崩盘，应该不会比之前的全年规划差距太大，何况还有年底那一拨的开盘没执行，因此总体来讲，我们今年完成任务，
还有有期待的空间的。
对于大竹，上半年已经就那样了，不再去纠结啦。我们总结教训，关注数据，看看接下来该干嘛。
我现在把能想到该做的事情，记录下来作为提醒。                

* 县城维护到位，重点在于康婷国美，导购蒸汽洗培训灌输一下，物料价签执行到位       。
	1.                   县城全部执行统一价签，绝不妥协。
	2. 康婷电器厨卫导购的负责人，承诺蒸汽洗加提50元每台，当然，培训蒸汽洗哦。
	3. 物料有的 都用上，不用心疼。                                      

* 王海梦数据分析后，看到的问题。
	1. 陈忠瑜这样的客户，不正常下滑，这种市场不能空等，属于就算把正常的乡镇代理扶持给到位，也不能产生正常销量的市场，应当释放出来了，要么陈忠瑜自己提货，要么只有加重散货客户比例，斌斌看着办，这种客户要么鞭策，要么调整，不下死命令一定要干掉他，但是找一个备胎客户台必要了。（比如吴国军这样的 客户是可以暗中扶持的）
	2. 主流客户的提货量在下滑，少数几个客户在增长，石桥，欧家，妈妈镇，当然赵双平也是微有增长的，这四个客户我认为属于乡镇客户里的A级客户了，应当重点扶持，出了点什么售后问题了，销售问题啦，无条件给整好，提升客户的忠诚度，有活动资源就先给这几个。
	3. 我们的散货客户数量，和提货量都在大幅下滑。这肯定跟小斌斌的维护方式有关的。本来今年比去年的维护频率就低了，散货客户的维护更是少之又少，而散货客户不是一个固定的合作商，维护得不够，势必就会怠慢美的，毕竟人家的选择还很多，不是只有美的，这更印证了地毯行动的必要性了，地毯行动的作用之一就是丰满我们的散货网点，请斌哥务必重视啊。
* 门市安排
	1. 导购忠诚度如何，好不好用，是不是可以长期合作的人选。
	2. 小区的门贴执行程度如何？ 现在贴了多少了？能不能深度执行一下，比如确定到冷瑞 31日，t贴X小区3栋，200张。这种具体的委派目标。
	3. 新门市的装修收尾木有，安排老蒋测量新门市形象尺寸。
	4. 我们的库存中 残次机多不，给冷瑞设定万成截止日期。
	5. 督促高滩门头。
	6. 前期账目录入，方便对账
	7. 乡镇欠款能收多少就收多少，真的揭不开锅了

* 地毯行动相关

1. 开江，涌兴首轮扫街后的，短信回访。
2. 第一轮后，来了门市上的意向客户，电话关心一圈，想合作的那种确定一下合作时间。
3. 第二轮地毯行动的日期安排。是否需要再物料方面有所改进？比如李大育提到的，在邀请函上加入一个电话小条子，还有斌斌提到的加入新的产品型号。
4. 车子是不是需要看看 有不有保养需要，保证经得住折腾才行啊。
