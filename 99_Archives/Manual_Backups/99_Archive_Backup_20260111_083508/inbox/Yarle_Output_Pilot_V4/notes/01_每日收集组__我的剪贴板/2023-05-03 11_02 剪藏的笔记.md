# 2023-05-03 11:02 剪藏的笔记

\# 跑步相关知识分享 ## 跑步的好处 !\[跑步的好处\](https://source.unsplash.com/1080x720/?running) - 手脚冰凉的女性，会因此改善 - 心肺功能大幅增强 - 进入心流状态 - 运动可带来持续的内啡肽激素 ## 修正跑步错误 !\[修正跑步错误\](https://source.unsplash.com/1080x720/?running+technique) - 我们跑步有很多默认错误，把这些修正好，是能少受伤和很享受的 - 比如我们“坐着跑”，比如过度跨步，这两种情况的解决方法是学会“重力跑”，利用重力驱动身体向前，而非肌肉带动自己向前；而且过度跨步带来的剪应力极其冲击膝盖骨，伤害很大； - 如何培养重力跑呢？倾斜身体找重心，左右腿切换观察； - 再如我们跑步发力方式是“蹬腿”，这样很费力；把蹬腿改为提腿，发力节省30%； - 如何培养提腿呢？用胶带，寻找后腿肌肉发力的感觉； ## 培养跑步习惯 !\[培养跑步习惯\](https://source.unsplash.com/1080x720/?running+habit) - 要培养跑步的习惯，内心建设比技巧更重要 - 我们要找到自己为什么要跑步的源动力，比如我，其实是想精力更好，现在自己就像一部破旧手机，早上出门，中午就没电了🪫 - 另外，习惯不能被消除，只是会被替代；我们不能指望自己在现有生活方式不减少的情况下，增加跑步这个习惯，只能是用这个替代掉其他习惯，比如耍视频，睡觉等；
