# 2018-08-23 07:58 剪藏的笔记

西方学者提出过这样的概念， 真正的城市化、深层的城市化应该是生存方式和价值观的改变，应该是农村的人口进了城，并且接受了城市的生存方式和城市中主导的价值观。如果没有完成生存方式和价值观的转变，那就是虚假的城市化。 而我们中国发生的事情是什么样的呢？ 我们有了流动，但是多数农民工并没有定居，他们也没有敢想定居这回事儿，家也没有办法安在这里。农民工在这里打工挣钱，日后是准备离开，回老家、回农村的。 从实际情况上说农民工的家庭也是分居在两地，丈夫在城里打工，妻子还在农村，或者丈夫跟妻子共同在城里打工赚钱，孩子还在老家，就是所谓留守儿童。这是一个正常的家庭生活吗？ 少数人如此可以理解，因为什么情况都有。但是如果一个民族里面有相当大的百分比，相当多的家庭都过着这样两地的生活，或是把孩子都交给农村的爷爷奶奶，这样的事情不值得思考吗？不需要改变吗？
