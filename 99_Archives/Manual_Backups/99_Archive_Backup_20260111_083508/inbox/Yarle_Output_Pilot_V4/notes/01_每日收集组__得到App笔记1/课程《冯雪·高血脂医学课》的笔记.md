# 课程《冯雪·高血脂医学课》的笔记

冯雪

### 03｜诊断：为什么高血脂诊断那么困难？

  2022-02-11 21:56:50

血液通道里的B船多，则意味着搁浅概率增高，则意味着脑梗 心梗增加。

我们在第一讲就说了，高血脂是一个“症”，而不是“病”。它本身既没有带来不舒服，也看不出当下的身体损伤。它只会年复一年的，增加你心脑血管疾病的风险。所以，我们要综合各种各样的因素，结合你的血脂数值，才能算出你心脑血管疾病的风险。

  2022-02-11 21:53:07

低密度脂蛋白胆固醇，本来是罪魁祸首； 但是运送它的B船总是搁浅，留在血液里，才是问题关键。 也就是说B船越多，搁浅可能性越大，心梗脑梗的概率才大

不过，最近医学界又有了新的发现，B船，也就是载脂蛋白Apo-B，更为重要。因为这艘B船上，什么都装，甘油三酯也装，胆固醇也装。这艘船由于一些病变，会“搁浅”在我们的的血管壁上，然后船上的货物就散落出来，淤积在血管壁上。研究还发现，Apo-B这个变量，甚至比LDL-C这个变量更加精确。

  2022-02-11 21:52:08

只要增加患糖尿病风险

其次，甘油三酯的嫌疑，也可以很快排除，因为甘油三酯和心脑血管风险的关联不大。相反，它主要会增加我们患糖尿病的风险。

  2022-02-11 21:51:28

结论是这个嘛。

大量医学研究也发现，高密度脂蛋白胆固醇是“好胆固醇”，这个指标比较高的人，心脑血管疾病的风险就比较低。

### 02｜机制：脂质如何在身体中代谢？

  2022-02-11 21:44:52

高血脂的本质是，流通环节的脂质增多了。
但如果脂质老老实实变成肥肉不乱窜，倒也是好的。
那么吃了就容易胖的人，是不是以为不容易高血脂

血脂升高的本质，就是处在流通环节的脂质过多了。

  2022-02-11 21:44:13

这个思路科学,目标不是消除脂质，而是不让他乱窜

如果我们把血脂的来路、转运和去路串起来，就会有一个重要的发现，那就是，对于高血脂这件事来说，脂质有没有真的离开身体，不重要；它有没有在血管里乱窜，才真的很重要。

  2022-02-11 21:43:18

也就是说运动优先消耗的是糖，而且没消耗完的糖 还会被转化为脂肪存起来

但是，这条去路也不容易，因为我们的身体里有胰岛素。胰岛素，不但优先拿糖类来供能，阻碍脂肪的分解消耗，还要倒打一耙，促进脂类的合成，把用不掉的糖类、脂肪都转化成身体中的脂肪，储存起来。

  2022-02-11 21:38:01

高密度脂蛋白胆固醇是把用不完的胆固醇拉回去消化掉

过，你在体检单上应该还见过另一个指标：高密度脂蛋白胆固醇。这是另一种小船，但它们却是高血脂的克星，因为它们对胆固醇的转运是逆向的：它们是把用不掉的胆固醇从身体各处收集起来，再转移回肝脏，最后变成胆汁酸，排出体外。

  2022-02-11 21:36:59

甘油三酯，胆固醇

再来看内源性血脂，就不一样了。内源性血脂的船，要把脂质从肝脏输送到体内的各个组织。这次要去的地方多，自然就要用小船，才比较划算。
这些小船，根据它们装的原材料比例不同，就有不同的名字。装甘油三酯多的，就叫甘油三酯。装胆固醇多的，就叫低密度脂蛋白胆固醇。这些内源性血脂的名字，就是你在体检单上看到的指标了。它们不分早晚、不分地点，长期在我们的血管里来回穿梭。它们，才是高血脂危害的源头。

  2022-02-11 21:36:18

我们去测的血脂，压根儿就不是吃进去在转化的血脂，而是内源性血脂。

它装了这么多货，为什么危害小呢？因为它运得特别快。快到什么程度？正常人血液里的乳糜微粒，半衰期不超过15分钟。你吃完脂肪没多久，它就把活都干完，下班了。所以，我们空腹去测血脂，体检报告上根本测不到乳糜微粒，也就不可能造成多大危害。

  2022-02-11 21:34:27

这意味着肝脏健康程度 跟高血脂密切相关

可能跟你的想法正好相反。你吃进来的脂肪类物质，成为外源性的血脂，它们的危害却是微乎其微的。而我们今天谈的高血脂，几乎100%都是内源性血脂造成的。

  2022-02-11 21:32:48

那么内源性血脂呢？外源性血脂这些大船会一路卸货，最终到达肝脏。刚一到肝脏，肝细胞就把这些大船分解了，再重新组装成了新的船，按照身体的需要，决定是否再次进入血液。只不过这次的船比刚才小一些。这些小船，就是内源性血脂。肝脏，正是内源性血脂最重要的来源。

  2022-02-11 21:30:49

绝大部分脂质原材料，都是不溶于水的，所以就需要找到一些溶于水的蛋白质来组装一下，让脂质“坐”到蛋白质这艘船上，形成一个小球，这样才可以顺利下水。这些蛋白质，因为是用来装载脂质的，我们就把它叫做“载脂蛋白”。

  2022-02-11 21:29:50

上一讲我们就说过，血脂是血液里的脂类物质，而不仅仅是脂肪。脂质在我们身体里发挥着各种各样的作用，每一个细胞都需要脂质。
