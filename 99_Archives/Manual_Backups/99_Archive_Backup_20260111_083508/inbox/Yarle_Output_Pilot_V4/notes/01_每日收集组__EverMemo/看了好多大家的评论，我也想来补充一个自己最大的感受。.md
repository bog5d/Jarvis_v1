# 看了好多大家的评论，我也想来补充一个自己最大的感受。

看了好多大家的评论，我也想来补充一个自己最大的感受。

像笑来老师说的，很多人自学不下去，很可能是因为”预算”不够，没有做好”投入足够时间和精力的预算”的心理建设。我深有感触，很多时候我们基于兴趣去学习，却因为没做好这样的准备，没有认清学习是这样”需要长时投入的”事实。进而以为自己不合适，就放弃了。然后”换一个地方继续检验自己的智商”

事实上，人生漫长，何必惊慌。不要因为害怕”浪费时间”从而放弃自学，任何一门手艺，花个两年也就入门学会了，花个五年或许能到精通，至于成为大师，那是一辈子的事。
