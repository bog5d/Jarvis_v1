# 兴哥，春伟兄，刚牛晟翔来电沟通，同步几个要点，

兴哥，春伟兄，刚牛晟翔来电沟通，同步几个要点，
1\. 初步规划下周下半周到长沙，届时把长沙这边军代一起邀请到企业，悉知；
2\. 关于54所的业务，可以作为第一阶段重点推进目标。 基于此，牛问我几个信息:
    1. 景嘉微目前在54的产品，我们是否能做？
    我回答：除了GPU采购景嘉微，大部分产品我们有能力做；

    2.如果他们协调的结果抢占景嘉微一些份额，是否会有影响泽天和景嘉微的关系？
      我的反馈是： 1. 孙总以前在54所积累的人脉，很认可孙总，目前来看，54所和我们泽天现有产品契合的业务有数千万的潜在订单。 受限于没有进入合格供方名录，此部分订单尚未转化，若协助打通，此部分订单大有可为； 2. 关于会否影响景嘉微，具体涉及到产品规划，孙总这边更清楚，建议后续当面详聊。

      3. 牛问54所打通合格供方，需要沟通哪方面人，军代能否起作用？
      我的反馈是： 每个所有自己的合格供方流程，54这边孙总相对更熟悉； 军代一般来讲，对进入合格供方有影响力。

    4. 牛建议，下周来长沙我们展示产品时，更多展示跟通信主题相关的产品（因装发资源侧重这一块儿），这样的好处是一旦在装发负责产品范围内，他们相对直接一些，把转化订单到泽天。 我们今天在线时产品讲得过泛，他们当下不知道怎么将现有资源和我们业务关联上。
