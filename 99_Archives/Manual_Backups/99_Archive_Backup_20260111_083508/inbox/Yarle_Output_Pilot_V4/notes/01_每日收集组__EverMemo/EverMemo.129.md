# EverMemo

• 大尺寸高精度测量
• 比如20 没法测 舰船没法测 小的部件可以
• 超宽带生成技术，测达目标，传统紧缩场，
• 测量区域 叫静区，静区的尺寸，决定了可测标的大小
• 一般紧缩场 只能做到10米，投了三个亿
• 常识 波打到平面 形成平面波 测平面波
• 平面波生成器 直接生成平面波，现在的技术
• 测隐身原理 发射平面波 打在飞机上 看会反射多少回来： 原来是馈源打出球面波，打在一个平面，反射形成平面波，再把平面波打在飞机上 看飞机返回多少波；
•  新的思路是取消平面 ，直接由多个馈源 直接生成平面波，不占地方 ，原来那个反射面要求很多；
