# 万里 (李万里):

万里 (李万里): 
瑕疵票一般指背书里有重复背书或者回头背书或者背书中有个体工商户这一类的，个体工商户法律规定是不允许做票据背书的，虽然他在网银技术上可以实现，但是有点违反规定，一般出现在建材类，因为好多供应商没有公司只是个门市部，重复背书上次咱那个情况就属于重复背书➕大回头，
第一种开票人是A，收款人是B，背书路径是ABB，B和B就重复了，
第二种回头背书指开票A，收款人B，第三首背书是c，第四首背书是D，背书路径是ABCDC，这叫小回头，
如果背书路径是ABCDB或者ABCDA这叫大回头，因为背书回到了票面的收款人或者开票人了，这样的大回头会影响更大，这是比较常见的瑕疵票，银票或者财务票有这类情况好处理，因为承兑主体实力强，可以走货款的方式付给其他供应商，但是商票有这几类情况就比较难处理了，付给其他供应商，人家不愿接受，银行更是不会接受这类贴现

万里 (李万里): 
所以价格上就会相应的有提高

万里 (李万里): 
因为流转或者去化渠道上受到了限制

万里 (李万里): 
还有一类，就是背书当中带有备注的，如果备注是货款或者备注某某工程什么的没关系，但是备注里备注借款一类的也是不可接受的，能不能高价收都不一定，因为到期兑付的时候可能会有扯皮

万里 (李万里): 
重复背书和回头背书也是这个原因，怕有扯皮可能存在导致承兑人不予兑付

万里 (李万里): 
因为背书流转一般来说是不同公司之间的，因为票据是基于贸易背景的，但是自己又背给自己了那在贸易环境中是不成立的，所以针对这种重复或者回头背书就属于瑕疵票
