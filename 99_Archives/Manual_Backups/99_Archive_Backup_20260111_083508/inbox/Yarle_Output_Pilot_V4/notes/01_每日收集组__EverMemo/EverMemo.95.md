# EverMemo

他在著作《小说课》里给出了一个“七步成故事法”——目标、阻碍、努力、结果、意外、转弯、结局。
很多成功的电影其实就是讲好了一个故事。我们可以用这七步法来分析一下。例如，根据真实事件改编的电影《摔跤吧，爸爸》。
● 目标：男主角自己想成为世界冠军。
● 阻碍：他把希望寄托在孩子身上，但是妻子生的都是女孩。
● 努力：一次偶然的机会，他发现了两个女儿的天赋。
● 结果：全村人无不打击、质疑他，包括他老婆，甚至包括两个孩子。
● 意外：报名参加摔跤比赛被拒绝后，正准备放弃，机会来了。
● 转弯：摔跤获得成功，一路走到全国冠军。
● 结局：中间又碰到了许多挫折，但最终获得了世界冠军。
