# 用未来视角太难了，而且人穷志短，越是困于当下生活琐事，越是不

用未来视角太难了，而且人穷志短，越是困于当下生活琐事，越是不敢看未来，担心自己期望太高，未来会失望。
这种自我保护而调低未来期望，调低未来期望后当下投入动力不足，就成为了恶性循环。

解，在哪儿？
我想应该是： 严格地逻辑推导未来，或者基于审慎地深度思考，未来趋势是怎样的，自己先深度相信自己的判断，而基于这个深度判断去生活和行动。

我目前对未来的预测和判断是依赖于老李的判断和规划，过去我对未来的预测和判断是依赖于老马的判断和规划；

我几乎没有自己的判断以及对未来大趋势的思考：
有一个好处是，我信任的人思考质量都不低的；
坏处是，因为对未来的判断和规划是他人的，自己遇到挑战困难时，则无法太坚定。 而且也会把责任甩给他们，觉得被他们“坑”了。

我自己的思考是什么？ 我相信自己的判断吗？

未来趋势，十年后不变的是什么？ 是我们都更老了些；
国内资本市场会更好吗？
我能怎样参与进去呢？
我可以投简历成为一个投资人吗？
我也好想成为一名投资经理，然后深度参与到资本市场。
但是目前的情况是： 我可以配合老李，帮助其把资源溢出增值，不用从零开始。

我和老李的搭配会否比我自己去打拼来得更高更快呢
