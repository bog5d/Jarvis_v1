# 无人机毁伤技术主要分为硬杀伤和软杀伤。硬杀伤是指通过防空武器

无人机毁伤技术主要分为硬杀伤和软杀伤。硬杀伤是指通过防空武器直接摧毁无人机，软杀伤是指通过特定手段使无人机非毁伤性失效难以完成任务。硬杀伤方式主要包括防空导弹、防空炮、激光武器。软杀伤方式主要包括电子干扰、电子诱捕、微波武器等。
（１）激光武器技术。反无人机激光武器系统，如１１年美战术激光器（ＡＴＬ）和战术高能激光器（ＴＨＥＬ）项目及其扩展项目，采用定向激光束高能流光毁伤无人机。其固体激光器具有体积较小、抗干扰能力强、命中精度高、目标转换灵活等优点，该系统功率为３０ｋＷ，可摧毁３．２km处速度４８２km／h的无人机，但其易受大气和大气衰减影响，毁伤无人机需要一定反应时间，跟踪瞄准系统易受到干扰。
（２）微波武器技术。高能微波武器作战成本低，杀伤范围广，受制天气敏感度低且对瞄准系统精度要求不高，能够快速反应，实现瞬间杀伤。美军认为，高功率微波技术是一种能够改变战争格局的颠覆性技术，自２０１５年始，美国致力于研究髙功率微波反无人机技术，２０１８年定向能峰会上，美大力支持高功率微波武器化意图明显。目前美雷锡恩公司研发战术高功率微波作战响应器（ＴＨＯＲ），经测试一次性击落无人机可达５０架以上。
（３）电子对抗技术。主要有电子干扰、电子诱骗、信号入侵３种方式进行电子对抗。美军反无人机电子对抗技术主要以电子干扰诱骗为主，如美国陆军的沉默弓箭手（Ｓｉｌｅｎｔ－Ａｒｃｈｅｒ）反无人机系统、无人机终结者（Ｄｒｏｎｅ－Ｂｕｓｔｅｒ）反无人机系统和美国空军的远征移动防空综合系统（Ｘ－ＭＡＤＩＳ），该技术所应用的系统通过干扰无人机的通信链路，使其返回基地或紧急迫降，在应对单个无人机和无人机蜂群时都有较好效果。但随着未来无人机抗干扰能力不断増强，无人机操控失效可能性降低。目前，美军正在建立多种技术融合的反无人机装备体系，如美军沉默弓箭手反无人机系统与反无人机激光武器结合，针对电子干扰对无机失效的情况，会将系统探测到的无人机目标信息传输给反无人机激光武器系统进行打击。
