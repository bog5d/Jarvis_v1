# 故事的套路

故事的套路

故事有没有套路？有，而且套路非常多，最典型的要数神话学家坎贝尔在《千面英雄》中提出的模型，坎贝尔分析了古往今来各种文明下的神话故事，发现他们都在讲同一个模型，好像是一个戴着各种面具的人，在做着同样的事情，演着同样的故事。
故事大概分为三幕。
第一幕讲的是英雄在新手村。他听到了某种召唤，这个召唤让他去冒险，但是此时英雄一定是犹豫的，他不太愿意离开自己的家乡，离开自己的舒适区，这种心态其实代表着大多数人对待挑战的第一反应。谁会不经过思考就去未知的地方探索呢，这种人估计都被进化的剪刀修理得差不多了。但是英雄终归是英雄，上路了才能有故事，有一天，他遇到了一件事，不得不开始自己的旅程，从一个熟悉的世界，来到一个陌生的世界。当英雄跳出舒适区的时候，故事正式开始。
第二幕讲的是历险的过程。这个过程当然会有挫折，会有迷茫，会有痛苦，没有这些故事就是不完整的，甚至称不上是故事。故事的过程就是遇到了一个挫折，克服它，然后发现又有一个更大的挫折，再克服它，终于遇到了最大的困难，英雄被挫折深深击败，但是最后总有绝地反击的那一刻，英雄凭借自己的力量，凤凰涅槃，冲出绝境，最终战胜了困难。
第三幕是英雄回归。战胜了终极困难之后，英雄要回到自己的家乡，开始和周围的人分享自己的故事，这时的他已经不再是出门前那个犹犹豫豫，对未知充满恐惧的少年了，他蜕变成为更加成熟、更加稳重的人。一段旅行告一段落，当然，生活永远是个连续剧，下一段英雄之旅即将开始。
我觉得这个模型非常精彩，但凡一个有决心，想要迈出自己的舒适圈，想要探索外部世界的年轻人，总会遵循这个框架。它最大的好处是给你描绘了一幅蓝图，在你遇到困难时给你希望，给你信心，因为你知道，英雄总是会战胜困难，脱离逆境的。
故事的来源

故事的来源无非有两种，一种是自己的故事，另一种是他人的故事。自己的故事与他人的故事相比，更能打动人。如果可以，请尽情讲述自己的经历、感悟。写作或是演讲的时候，只要自己的故事与主题有某种关联，即便联系得不够紧密，也会很容易打动人。即便故事看起来没有那么特别，也比生搬硬套讲一个听来的故事更具有说服力。
他人的故事也可以，但是讲好别人的故事要求要高得多，你总要提炼出一些道理、观点、规律，才好引用别人的故事，否则，就成了故事大会，别人听着过瘾，但没有收获。
我们要怎么讲故事

许荣哲在《小说课》里根据《千面英雄》的模型，总结出了一个故事公式，把故事浓缩成了七个具体的问题。想要开始一个故事的时候，问好这七个问题，一个故事的框架自然就出现了。
这七个问题是：目标、阻碍、努力、挫败、意外、转折、结局。就拿《肖申克的救赎》来说，主人公安迪蒙冤入狱，他的目标是越狱。越狱过程中当然会遇到阻碍，比如铜墙铁壁的环境、凶暴的狱友和警卫。一次意外让安迪发现墙面是可以挖通的，这让他燃起了越狱的希望，服刑期间安迪一直在努力，拿着他的小锤子，一点点地在墙壁上挖洞。经历过无数次的挫败后，在一个月黑风高电闪雷鸣的夜晚，安迪迎来了人生的转折，冒死爬进了下水管道，最后成功出逃，到自己梦想中的小岛安度晚年。
我认为这种高度凝练的规律几乎适用于所有故事，只要一个人有目标，他的故事就开始了。没有哪个人在达成目标的道路上会一帆风顺，没有哪条路途中没有阻碍，经过不懈努力之后，自然会轻松掌握套路和技巧，最后不管成功与否，其实都是完成了一次蜕变。一个个的故事，一次次的英雄之旅，都是一个人成长的台阶。
故事的本质并不仅仅是一种娱乐，它是人类给自己打造的第一把武器，漫长的进化历程，已经将人脑塑造成一台故事机。不管到什么时候，讲故事都是最符合人类心智的沟通方法。如果你想把你的理念传播出去，那最好的开头永远是：我给你讲个故事。
