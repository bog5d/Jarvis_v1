# - 踩的坑： 太多酒箱堆在车过道，既不美观，也很影响乘坐体验

\- 踩的坑： 太多酒箱堆在车过道，既不美观，也很影响乘坐体验，我全程扭着做，因为箱子在座位上；
\- 坑二： 错误预估行程需要花费的时间，我们用导航预测行程时间，导航默认的是你开轿车，轿车一般 120的速度，而实际大巴车速度 80～100 之间，所以去的路程很费时间； 跨城的话，能安排高铁就高铁，尤其是行程超过 3 小时的情况，直接找能否高铁，到地儿后再约个大巴车也不迟（不过这样会增加拎行李过程，也值得商榷）
\- 坑三： 话太多，其实话越少，越有内涵和气质；
\- 坑4， 成员订车目的地错，导致客人来迟。
