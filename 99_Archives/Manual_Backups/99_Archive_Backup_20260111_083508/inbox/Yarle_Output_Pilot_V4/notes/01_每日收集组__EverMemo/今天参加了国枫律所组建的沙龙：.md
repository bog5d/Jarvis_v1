# 今天参加了国枫律所组建的沙龙：

今天参加了国枫律所组建的沙龙：
\- 张立国 国枫首席合伙人（应该是创始人之一） 首先上台发言
\- 沈朝辉 清华大学法学院教授，分享新的公司法相关内容
    - 本轮公司法修改时间很长 2019-2023；
    - 什么算大改法律：即实施日期是否会变，若按修订日期后实施就是大改；反之则小改；
    - 修改公司法的指导思想： 很多人指责是为了改而改，其实是为了优化基础性法律制度，国外有的我们加入吸收进来。18年修改是为了注册资本制，此次修改好像没有太多抓点，所以说是基础
    - 这次修改比如：认缴制，股东实缴时间太长，找不到股东；
    - 为公司设立登记增加专章.
    - 股份被明确为两类股份：设立时发行的，和设立后增加的； 比如什么授权股份，这是个新概念；
    - 董事会授权股份，发行不超过50%的，可以不开股东会； 增加自主性。 但非现金支付需要股东会同意，之前深圳地铁用地块买股份，就是非现金支付，未来需要股东同意； 白衣骑士很难当了；
    - 毒丸计划，低价向行权股东发起。 目前的公司法只能解决第二步的法律障碍，第一步公司发行期权需要有权力，所以我国公司法并不算引入了毒丸反收购措施。 而且证监会监管也让新增发股发行很难； 公司法亟需反收购权证类的措施；
    - 单层制公司模式引入： 就是没有监事会或者监事的模式。 在董事会设置委员会就可以不设监事会或监事。
    - 单层制公司为降低公司治理成本； 大量上市公司不设监事会； 传统三角制衡的治理会被替代
    - 公司成立时就要求董事很多责任，比如股东不出资或出资瑕疵，董事要负责催交，责任也要董事承担；
    - 五年缴资期限；这是否意味着那些认缴的公司要大量注销呢？
    - 新公司法：可能会增加董事责任，使得董事和高管趋于保守，也许未来会有责任免除方面的机制，比如董事保险，或者承担责任为自己年收入六倍责任。
