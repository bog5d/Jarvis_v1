# 万姐，您好，我是王波之前参加过咱们万联供金峰会，启发很大。

万姐，您好，我是王波之前参加过咱们万联供金峰会，启发很大。
我们在军工行业（航空航天领域）做项目，一直对接航空航天的研究所和主机厂（核心企业），这些研究所和主机厂平均有数千家供应商，以合作较多的SF，一年80亿产值，供应商四千多家，跟供应链上下游打交道过程中，出现了很多供应链金融场景，比如票据融资，订单融资等。
目前我们帮助很多供应商解决了票据融资的需求，也给我们带来了一些收益。
在和供应商深度交流过程中，发现订单融资的需求极大：
军工订单激增，很多企业目前资金接不住；另一方面，军工订单采购方均为国央企，付款信用极好。
所以，想要请教，在做订单融资方面，有没有能做军工行业的服务供给方或者资金方；

从合作价值端来看，我们可做军工产业项目方，希望能跟好的资金方结合
