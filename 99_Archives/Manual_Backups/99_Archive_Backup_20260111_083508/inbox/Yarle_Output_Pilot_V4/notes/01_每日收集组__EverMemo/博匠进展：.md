# 博匠进展：

博匠进展：

1.公司这一年有什么里程碑事件和进展？
  - 推出了BMC芯片，实现完全自主可控，而且在传统的BMC芯片基础上，增加了军用需求的可信启动，毫秒供电等功能；启动时所有的软硬件是否被篡改过，这个会用加密算法验证是否有未授权后台行动； 而且已完成对飞腾龙芯cpu的适配；
  - 超小型操作系统研发完成，已在飞腾平台完成验证。 对于空类产品减重提速，以及配合BMC芯片，可实现极大的能量节省；启动时间4.5秒；
