# 商家你好，我是曾在贵店消费过的客户，我非常不满意我的消费体验

商家你好，我是曾在贵店消费过的客户，我非常不满意我的消费体验，想向大家分享我的遭遇。我认为贵店的消费陷阱、傲慢和推诿、威胁消费者的行为都非常令人不满。

首先，贵店设置了消费陷阱，把各种扣款条款写在收据的背面，没有提醒提示，就催着我们签字。

其次，贵店在协商中表现得极度傲慢和推诿，把自己当作很大的甲方，要求消费者顺从各种规则。
最后，我认为贵店威胁消费者的行为极其傲慢，店员电话强势答复，如果我们差评，他们会请律师告消费者诽谤。我希望贵店能尊重客户，提高服务质量，避免再次出现类似的问题。
