# 📱 Jarvis 移动端连接指南 (Mobile Guide)

> **前提条件**: 您的手机必须和电脑连接在**同一个 Wi-Fi** 网络下。

## 1. 获取访问地址
根据刚才的系统检测，您的 Jarvis 访问地址是：
> **http://192.168.31.49:8501**

*(如果未来电脑 IP 变了，请在电脑终端输入 `ipconfig` 查看新的 IPv4 地址)*

## 2. 连接步骤
1.  打开手机浏览器 (推荐 Safari 或 Chrome)。
2.  在地址栏输入上面的地址。
3.  如果加载成功，您将看到 Jarvis 的控制台界面。

## 3. 开启语音对话
1.  点击页面底部的 **"💬 顾问对话 (Chat)"** 标签。
2.  找到 **"🎤 语音输入 (Voice Input)"** 区域。
3.  点击麦克风图标。
    *   *首次使用时，浏览器会询问“是否允许使用麦克风”，请务必点击**【允许】**。*
4.  说完指令后，点击停止/完成，Jarvis 会自动识别并回复。

## 4. 像 App 一样使用 (添加到主屏幕)
为了方便以后一键启动，建议将网页添加到手机桌面：

*   **iPhone (Safari)**:
    1.  点击底部中间的 **分享按钮** (方框向上箭头)。
    2.  向下滑动，选择 **"添加到主屏幕"**。
    3.  命名为 "Jarvis"，点击添加。
*   **Android (Chrome)**:
    1.  点击右上角 **三个点** 菜单。
    2.  选择 **"添加到主屏幕"**。

## 5. 常见问题 (Troubleshooting)
*   **打不开网页 (连接超时)**:
    *   检查手机是否切到了 4G/5G？请切回 Wi-Fi。
    *   **防火墙拦截**: 在电脑上打开“控制面板” -> “Windows Defender 防火墙” -> “允许应用通过防火墙” -> 找到 `python.exe` -> 确保“专用”和“公用”都打钩。
*   **无法录音**:
    *   检查手机浏览器的权限设置，确保允许该网站使用麦克风。
    *   如果是 iPhone，尝试使用 Safari 浏览器（兼容性最好）。
